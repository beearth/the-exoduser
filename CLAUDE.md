# CLAUDE.md — 지옥의 길 (Hell Road) 프로젝트 규칙

## 프로젝트 개요

- **게임**: 지옥의 길 — 핵앤슬래시 로그라이크 액션 RPG
- **기반 파일**: `game.html` (단일 HTML, 메인 게임 코드, ~11000행)
- **Electron**: `Electron/main.js` + `Electron/preload.js` (데스크톱 래퍼)
- **레거시 파일**: `hell_souls3 (2).html` (구버전, 사용하지 않음)
- **기술**: Canvas 2D + Web Audio API + Supabase 인증
- **로컬 서버**: `http://localhost:3333/` (game.html은 index.html 로비에서 진입)
- **기획서**: `지옥의길_통합GDD.md` ← 모든 작업의 기준 문서

## ⚠️ 절대 규칙

### 건드리지 마라 (기존 시스템 보존)

아래 시스템은 이미 완성됐다. 수정/삭제/리팩토링 금지:

- **돌진 시스템** (관통, 체인킬, 피니셔, AoE, 잔상, 스피드라인) — `P.s==='charging'` 관련 전체
- **패링 시스템** (투사체 반사, 근접 카운터, 3초 스턴) — `P.s==='parryWin'` 관련 전체
- **마법 시스템** (E탭 투사체, E홀드 빔) — `magicCast`, `beaming` 관련 전체
- **포이즈/그로기** — `e.poise`, `e.stunned`, 그로기 3배 데미지
- **무기 7종** (WTYPES): sword/dagger/hammer/mace/club/axe/spear
- **활 5종** (BOWTYPES): shortbow/longbow/crossbow/mystic/composite
- **속성 5종** (EL): 물리/화염/빙결/암흑/뇌전 + 상성 배율
- **SFX 엔진 v3** — `SFX` 객체 전체 (Web Audio API)
- **저체력 경고** — 심박, 크랙, 비네팅
- **퀵슬롯 물약** — `QSLOTS`, `useQuickslot()`, `POT`
- **키 바인딩** — `BINDS`, 리바인딩 시스템
- **인벤토리** — `INV`, 필터, 비교, 가방 확장
- **데미지 계산** — `hurtE()`, `hurtP()` 의 공식 (DEF 감산, 50% 캡)
- **델타 타임** — `_dtSp` (dt*60), `sp` (slowMo*_dtSp), 모든 타이머/물리에 적용됨
- **FPS 캡** — `OPT.fpsCap`, `_FPS_STEPS=[30,60,120,144,244]`, Electron 기본 244
- **Electron 지원** — `IS_ELECTRON` 플래그, Tier A 강제, 로그 억제, 로컬모드
- **보스 AI** — Utility-based AI (`_bossAI`, `BOSS_MOVES`, `_bossScore`)
- **펫 시스템** — 까마귀(공중 수거) + 고양이(땅 수거+자동분해)
- **에너지 쉴드** — INT 100+ 해금, pDef/eDef 이원 방어
- **방패 시스템** — 방패막기(sBlock: 데미지감소만), 방패치기(sBash: 패링/반사 가능)

### 반드시 지켜라

- **무적판정 금지**: 캐릭터/몬스터 모두 무적(iframes) 없음. 장비 어픽스로도 무적 부여 금지. 특별히 지정된 경우(돌진, 패링윈, 블링크 등 기존 확정 시스템)만 예외.
- **흡혈/킬힐 금지**: lifeSteal, killHeal 등 HP 흡수류 장비 어픽스 금지. 장비로 체력 회복 불가.
- **단일 HTML 유지**: 모든 코드는 하나의 HTML 파일 안에. 외부 JS/CSS 분리 금지.
- **외부 라이브러리 금지**: npm, CDN, 프레임워크 사용 금지. 순수 JS + Canvas + Web Audio만.
- **한번에 하나만**: 한 작업에서 하나의 시스템만 추가/변경. 여러 시스템 동시 변경 금지.
- **기존 변수명 유지**: `P`, `G`, `INV`, `ens[]`, `projs[]`, `pProjs[]`, `worldItems[]`, `OPT`, `BINDS`, `STC` 등.
- **기존 함수 시그니처 유지**: `hurtE(e,dmg,ang)`, `hurtP(dmg)`, `addExp(v)`, `addTxt(x,y,t,c,l)`, `addParts(x,y,col,n)` 등.
- **테스트 후 커밋**: 변경 후 게임이 정상 실행되는지 브라우저에서 확인.
