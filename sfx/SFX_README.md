# 지옥의 길 — SFX 총 정리

## 폴더 구조

```
sfx/
├── beam/          — 빔/레이저 계열
├── chain/         — 체인(갈고리) 돌진
├── hit/           — 타격음
├── ice/           — 얼음 계열
├── magic/         — 마법/화염 마법
├── misc/          — 발소리, 함정 등
├── monster/       — 몬스터 이동/피격/사망 (기존)
├── q_parry/       — 방패 패링
├── swing/         — 무기 휘두르기
├── sword_parry/   — 검 패링/반격
├── voice/         — 플레이어 음성
├── weapon/        — 원거리 무기 (활/석궁)
├── _english_backup/ — 영어 음성 백업
└── (루트)         — 몬스터 사망 SFX (신규)
```

### 화염 사망 연출 (etype 30~39) — 3레이어
1. `hit_fire_01.mp3` — 즉시
2. `death_fire_01.mp3` — 즉시
3. `death_sizzle_01.mp3` — 100ms 딜레이

### 오디오 재생 규칙
- 게임루프 내 new Audio() 금지 → AudioPool 순환 재생
- volume: 0.6~0.8 (랜덤)
- playbackRate: 0.9~1.1 (랜덤 피치)
