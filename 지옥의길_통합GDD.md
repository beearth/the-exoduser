## 1. 게임 개요

### 현재 상태 (hell_souls3)

| 항목 | 현재 |
|------|------|
| 맵 | 방 기반 절차생성 (5+stage개 방, 80×60 타일) |
| 진행 | 8 스테이지 순차 클리어 |
| 적 | 방 안 고정 스폰, 클리어하면 끝 |
| 레벨 | 무제한이지만 실질 ~30 내외 |
| 멀티 | 없음 (싱글) |
| 보상 | 재료 → 대장간 제작 |
| 엔딩 | 시즌1 시네마틱 엔딩 |

### 변경 후 (지옥의 길)

| 항목 | 변경 |
|------|------|
| 맵 | **개미굴 미로 던전** (층별 거대 미로) |
| 진행 | 7층 미로 탈출 (같은 구조, 서버 레이스) |
| 적 | **소환굴에서 무한 리스폰** — 클리어 불가 |
| 레벨 | **1~1000** (3000 스탯포인트) |
| 멀티 | **서버 레이스** (64~128명 동시 진행, PvP 없음) |
| 보상 | 탈출 시 **랜덤 상자 1개** (전원 동일) |
| 엔딩 | 없음 — **영구 서버, 무한 플레이** |

## 2. 세계관 & 스토리

### 프롤로그 — 공통 시네마틱

> 전쟁에서 살아 돌아온 한 남자. 하지만 집은 이미 없다.
> 이웃이자 친구였던 **킬루**가 가문을 짓밟았다.
> 아내를 하녀로 끌고 가 몸종과 몹쓸일을 시키고,
> 아이들은 노예상에 팔았으며,
> 늙은 부모를 식량을 탐한다며 감옥에 가두어 굶겨 죽였다.
>
> 한밤중, 남자는 킬루 가문 31명을 몰살한다.
> 하인까지 포함해서 31명.
> 킬루의 손발을 먼저 잘랐으며, 불로 지져 지혈까지 해주었다.
>
> "기억해라."
>
> 그리고 지옥에 떨어진다.

### 전환 — "그것은 그자의 이야기다"

> *"...이것은 수천의 복수자 중 하나의 이야기일 뿐."*
> *"지옥의 미로에는 매일 새로운 영혼이 떨어진다."*
> *"분노로 가득 찬 자, 억울함에 미친 자, 사랑을 잃은 자..."*
>
> **"너는 왜 지옥에 왔느냐?"**

→ 캐릭터 생성으로 전환

### 핵심 서사 요소

- **복수의 신 네메시아**: 복수 자체의 순수함에 반한 신. 플레이어에게 힘을 빌려주며, 대장간을 운영한다.
- **보스 대사**: 각 층 보스가 플레이어에게 질문을 던진다 (스토리 깊이, 선택사항)
- **단 한 명만 올라간다**: 서버 레이스의 세계관적 근거

### 스테이지별 보스 서사

|---|---|---|
| 1. 얼음길 | 빙결의 군주 | "돌아갈 수 있다면, 또 그렇게 하겠는가?" |
| 2. 곤충길 | 군충의 여왕 | "킬루도 전쟁이 만든 괴물이 아니었는가?" |
| 3. 악마길 | 혈마 아스모데 | "네 칼에 죽은 하인의 이름을 아는가?" |

| 7. 고통의 심연 | 고통의 대마왕 | ??? (킬루가 지옥에서 기다림) |

## 3. 게임 플로우

```
[앱 시작]
    ↓
[시네마틱 인트로] — 공통 스토리 재생 (스킵 가능)
    ↓
[캐릭터 생성]
  ├─ "네 이름이 무엇이냐?" → 이름 입력
  ├─ "왜 지옥에 왔느냐?" → 복수 사연 선택 (빌드 보너스)
  ├─ 외형 커스텀 (갑옷 색상, 투구 형태, 초기 무기)
  └─ 네메시아가 낙인을 새긴다 → 게임 시작
    ↓
[로비 — 복수자의 안식처]
  ├─ 영구 창고 (재료 보관)
  ├─ 복수의 대장간 (제작/강화/리롤)
  ├─ 빌드 프리셋 저장
  ├─ 랭킹 보드
  └─ [서버 입장] 버튼
    ↓
[서버 레이스]
  ├─ 대기실 (5분 모집)
  ├─ 전원 동시 낙하 → 최하층
  ├─ 7층 미로 탈출 경쟁
  ├─ 탈출 or 시간종료 or 사망
  └─ 보상: 랜덤 상자 1개
    ↓
[로비 복귀] → 반복
```

### 캐릭터 생성 — 복수 사연 (빌드 보너스)

| 사연 | 이모지 | 보너스 | 설명 |
|------|--------|--------|------|
| 배신당한 전사 | 🔥 | 근력+화염 | 기본ATK +3, 화염 데미지 +15% |
| 가족을 잃은 학자 | 🧊 | 지능+빙결 | 기본MP +30, 빙결 데미지 +15% |
| 누명 쓴 기사 | ⚡ | 민첩+번개 | 이동속도 +0.3, 뇌전 데미지 +15% |
| 실험당한 죄수 | 🌑 | 체력+암흑 | 기본HP +50, 암흑 데미지 +15% |

## 4. 던전 구조 — 개미굴 미로 시스템

### 핵심 변경: 방 기반 → 미로 기반

**기존 `genMap()`**: 5+stage개의 직사각형 방을 배치하고 복도로 연결
**변경**: 거대한 미로 생성. 개미굴처럼 수직/수평으로 얽힌 터널. 방은 미로 안의 넓은 공간(광장)으로 존재.

### 층 구조 (아래→위)

```
☀️ 지상 — 탈출 (엔딩 없음, 로비 복귀)
━━━━━━━━━━━━━━━━━━━━━━━━━━
 ▲ 7층: 고통의 심연 (최종 보스)
 ▲ 6층: 용암굴
 ▲ 5층: 심해굴
 ▲ 4층: 괴물굴
 ▲ 3층: 악마굴
 ▲ 2층: 곤충굴
 ▲ 1층: 얼음굴
 ● 최하층: 낙하지점 (시작)
━━━━━━━━━━━━━━━━━━━━━━━━━━
🔥 나락 (죽으면 떨어짐)
```

### 미로 생성 알고리즘 (변경점)

```javascript

function genMaze(fi) {
  // 층이 올라갈수록 미로가 커지고 복잡해진다
  const mw = 120 + fi * 20;  // 120~260 타일 폭
  const mh = 90 + fi * 15;   // 90~195 타일 높이
  
  // 1. 미로 뼈대 생성 (Recursive Backtracking)
  // 2. 광장(방) 랜덤 배치 (파밍존, 대장간, 보스방)
  // 3. 소환굴 배치 (등급별)
  // 4. 위층 통로 1~3개 배치 (보스가 지키는 통로 1개 필수)
  // 5. 비밀 통로 배치 (벽 파괴로 발견)
}
```

## 5. 무한 리스폰 — 소환굴 시스템

### 핵심: 클리어 불가. 몬스터는 영원히 나온다.

**기존**: `checkRooms()` — 방 안 적 전멸 시 `room.cleared = true`
**변경**: `room.cleared` 개념 삭제. 소환굴에서 무한 리스폰.

### 소환굴 (Spawn Hole)

던전 벽 곳곳에 배치된 검은 구멍. 30초~1분 간격으로 몬스터 리스폰.

| 등급 | 색상 | 리스폰 주기 | 몬스터 레벨 오프셋 | 드랍 보너스 |
|------|------|------------|-------------------|-----------|
| 균열 | #888888 (회색) | 60초 | 층 기본 +0 | 일반 재료 |
| 구멍 | #cc3333 (붉은색) | 40초 | 층 기본 +20~50 | 희귀 재료 확률 ↑ |
| 심연 | #9933cc (보라색) | 25초 | 층 기본 +100~200 | 전설 재료 확률 ↑ |
| 나락의 입 | #111111 (검은색) | 15초 | 층 기본 +300+ | 보스급 드랍 |

```javascript
// 소환굴 데이터 구조
spawnHole = {
  x, y,           // 위치
  grade: 0~3,     // 균열/구멍/심연/나락의입
  timer: 0,       // 현재 타이머
  interval: 60~15, // 리스폰 주기 (초)
  lvOffset: 0~300, // 몬스터 레벨 오프셋
  active: true     // 파괴 불가 (항상 true)
}
```

### 파밍 루프

```
길을 찾는다 → 갈림길 → 잘못 간다 → 소환굴 지역
    ↑                                      ↓
  출구 발견 ← 강해진다 ← 레벨업 ← 끝없이 싸운다
```

### "잃어도 괜찮다" — 보상 장치

- **연속 킬 콤보 배율**: 100콤보/500콤보/1000콤보 경험치 폭발 (기존 `P.combo` 확장)
- **방랑자 버프**: 같은 구역 3분 이상 체류 시 네메시아 버프 (드랍률/경험치 +30%)
- **레어 몬스터**: 막다른 길에서만 출현하는 희귀 몬스터 (최상급 재료)
- **파밍존**: 소환굴이 밀집된 막다른 길 = 자연 파밍존

## 6. 레벨 1000 성장 시스템

### 기존 레벨 시스템

```javascript
// 현재: mkP() → lv:1, exp:0, maxExp:100
// 레벨업: P.maxExp = ~~(P.maxExp * 1.4)
// SP: 레벨당 1 SP
// 업그레이드: 6종, 각 max 3~5 (총 ~26 SP 소비)
```

### 변경: 1000레벨 시스템

```javascript
// 변경: 레벨 1~1000
// 레벨업 곡선: 1.15배 (1.4→1.15로 완화, 1000까지 도달 가능하게)
// SP: 레벨당 3 SP → 총 3000 SP
```

### 5종 스탯 (기존 6종 → 5종 통합)

| 스탯 | 효과 | SP당 증가 |
|------|------|-----------|
| **근력** (STR) | 근접 데미지, 벽 파괴력 | ATK +1 |
| **민첩** (DEX) | 이동속도, 패링 타이밍 여유 | Speed +0.02, 패링 윈도우 +1f |
| **지능** (INT) | 마법 데미지, 미니맵 시야 | 마법 DMG +1.5%, 시야 +0.5타일 |
| **체력** (VIT) | HP, 독/용암 저항 | HP +5, 저항 +0.5% |
| **운** (LCK) | 드랍률, 비밀통로 감지 | 드랍률 +0.3%, 감지 반경 +0.2타일 |

3000포인트 배분에 따라 완전히 다른 빌드:
- **근력 올인**: 탱커 근접
- **민첩 올인**: 패링 장인
- **지능 올인**: 마법 원거리
- **운 올인**: 파밍 특화
- **하이브리드**: 다양한 조합

## 7. 서버 레이스 시스템

### 핵심: 같은 던전, 같은 미로, 다른 실력

```
[서버 생성] 매 2~4시간마다 자동 오픈
     ↓
[모집] 5분 대기실 (인원 모집)
     ↓
[낙하] 전원 동시 시작 (같은 맵 구조)
     ↓
[레이스] 제한시간 내 7층 탈출 경쟁
     ↓
[종료] 시간 종료 or 전원 탈출/사망
     ↓
[보상] 전원 랜덤 상자 1개 + 던전 내 드랍
     ↓
[로비 복귀] 바로 다음 서버 입장 가능
```

### 서버 타입

| 타입 | 인원 | 제한시간 | 특징 |
|------|------|---------|------|
| **연옥** (입문) | 32명 | 없음 | 연습용, 보상 적음 |
| **아수라** (일반) | 64명 | 24시간 | 메인 경쟁 모드 |
| **나락** (하드코어) | 128명 | 12시간 | 죽으면 1층 추락, 고보상 |
| **심판** (엘리트) | 16명 | 6시간 | 랭커 전용, 최고 보상 |

### 멀티플레이어 요소 (PvP 없음)

- **유령 시스템**: 같은 서버의 다른 플레이어가 반투명하게 보임. 공격/상호작용 불가.
- **사망 표식**: 다른 플레이어가 죽은 자리에 해골 잔상. 위험 구간 경고.
- **층 클리어 알림**: 누군가 다음 층으로 올라가면 전체 서버에 알림.
  - *"복수자 [이름]이 얼음굴을 탈출했다 — 1등"*
- **탈출 시네마틱**: 1등 탈출 시 전체 서버에 짧은 연출.
- **실시간 랭킹 보드**: 현재 누가 몇 층인지 표시.

### 플레이어 캐릭터는 서버 간 영구

- 레벨, 장비, 재료 모두 **영구 보존** (서버 끝나도 유지)
- 서버에서 레벨업한 것도, 얻은 재료도 전부 로비로 가져감
- 서버는 "던전 입장"일 뿐, 캐릭터 리셋 없음

## 8. 보상 & 경제 — 영구 서버, 무한 소비

### 시즌 없음. 리셋 없음.

### 탈출 보상: 랜덤 상자 1개

```
「지옥의 잔해」 열기...

[일반] 60% — 지옥 철광석, 뼈 가루, 그을린 가죽 등 기본 재료
[희귀] 25% — 속성 원석, 저주받은 강철, 독충의 눈
[전설] 10% — 악마의 심장, 심판의 용해로, 태초의 원석
[신화]  4% — 네메시아의 눈물, 연옥의 거울(미러)
[???]   1% — 보스 전용 무기 설계도, 히든 레시피
```

전원 동일 확률. 1등이든 꼴등이든 상자는 같다.

### 빠른 클리어의 가치

```
느린 사람: 4시간에 1클리어 = 하루 3상자
빠른 사람: 1시간에 1클리어 = 하루 12상자
같은 확률, 4배 더 많은 시도 = 4배 더 많은 기회
```

### 재료 소비 구멍 (경제 순환)

시즌 리셋 없이도 경제가 돌아가려면 재료가 계속 소비되어야 한다.

| 소비처 | 메커니즘 | 소비량 |
|--------|---------|--------|
| **제작** | 장비 만들면 재료 소멸 | 중 |
| **강화** | 실패 시 재료만 소멸 (장비 유지) | 대 (고강화일수록 폭발적) |
| **리롤** | 장비 옵션 재설정 시 재료 소비 | 대 (완벽 옵션 추구 시 무한) |
| **내구도 수리** | 장비 수리에 재료 소비 | 소 (지속적) |
| **소모품 제작** | 횃불, 폭탄, 물약 (서버마다 소비) | 중 (지속적) |
| **분해** | 장비→재료 변환 (투입량 > 회수량) | 소 (자연 소멸) |

### 강화 시스템 (PoE2 참고)

```
+1~+5   성공률 80%  → 재료 소량
+6~+10  성공률 50%  → 재료 중량
+11~+15 성공률 20%  → 재료 대량
+16~+20 성공률 5%   → 재료 폭발적 소비
실패 시: 재료만 소멸 (장비는 유지)
```

### 크래프팅 오브 (PoE2식)

| 아이템 | 기능 | 획득 |
|--------|------|------|
| **지옥의 숫돌** | 장비에 랜덤 속성 1개 추가 | 서버 클리어 보상 |
| **망각의 물약** | 장비 속성 1개 랜덤 삭제 | 레어몹 드랍 |
| **심판의 천칭** | 속성 2개 교환 (A↔B) | 보스 드랍 |
| **네메시아의 눈물** | 장비 등급 1단계 승급 | 랜덤 상자 (4%) |
| **연옥의 거울(미러)** | 장비 복제 (1회용) | 랜덤 상자 (4%) |

### 미러도 소비된다

미러는 한 번 쓰면 사라진다. 최고급 장비 복제용. 미러가 쌓이는 건 안 쓰고 있다는 뜻이지 과잉이 아니다.

### 장기 목표 — 왜 끝이 없는가

레벨 1000 = 끝? **아니다.**
전설 장비 풀셋 = 끝? **아니다.**
+20 강화 = 끝? **아니다.**

**완벽한 4옵 장비**가 진짜 엔드게임. "근력+15, 화염DMG+20%, 패링시HP회복+5%, 크리티컬+12%" 이런 조합? 리롤 수천 번. 그 수천 번에 재료가 소비. 시즌 필요 없음.
