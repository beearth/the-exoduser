# 2-3. 돌진 + 패링 + 방패 시스템

## 돌진 시스템 (P.s==='charging')
- 관통, 체인킬, 피니셔, AoE
- 잔상 + 스피드라인 VFX
- **수정 금지** — 완성 시스템

## 패링 시스템 (P.s==='parryWin')
- 투사체 반사 + 근접 카운터
- 패링 윈도우: 20프레임(0.33초), 해제 후 12프레임(0.2초)
- **무지개탄(blackBean)은 절대 패링 불가**
- **수정 금지** — 완성 시스템

## 방패 시스템
| 모드 | 키 | 효과 |
|---|---|---|
| 견갑막기 (sBlock) | 우클 | 데미지 감소만 |
| 견갑치기 (sBash) | 우클 (타이밍) | 패링/반사 가능 |
- 보호막 쿨다운: 60프레임(1초)

## 포이즈/그로기
- `e.poise` — 몬스터 체간
- `e.stunned` — 그로기 상태
- 그로기 시 데미지 1.5배
- **수정 금지**
