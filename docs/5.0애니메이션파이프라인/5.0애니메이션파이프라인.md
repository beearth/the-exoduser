4. **절차적 렌더링** (`drawBossBody`) — 위 전부 없을 때 폴백


| 애니메이션 | 프레임 수 | 오프셋 | 용도 |
|-----------|----------|--------|------|
| idle | 4 | 0 | 대기 |
| walk | 8 | 4 | 이동 |
| atk | 16 | 12 | 근접공격 |
| whirl | 16 | 28 | 회전참 |
| throw | 16 | 44 | 작살 던지기 (3단계: 던지기0~3, 끌려가기4~9, 착지10~end) |
| chain | 16 | 60 | 사슬기동 체인공격 |
| dash | 16 | 76 | 사슬기동 이동 (3단계: 시작0~3, 도약4~11, 착지12~15) |

### dash 3단계 재생 로직
- **dash_start** (frame 0~3): 시작 모션, 1회 재생 후 fly로 전환
- **dash_fly** (frame 4~11): 공중 비행 루프
- **dash_land** (frame 12~15): 착지 모션, 1회 재생 후 idle로 복귀
- `_dashPhase` 변수로 상태 관리 (0=none, 1=start, 2=fly, 3=land)

## PixelLab 워크플로우

1. `boss_gen_queue.json`에서 미완성 보스 선택
2. PixelLab `create_character` → character ID 획득
3. `animate_character` → 각 애니메이션 타입별 생성
4. 결과를 `img/bosses/boss_NAME/` 폴더에 저장
5. `atlas_bosses.png/json` 리빌드
6. `boss_gen_queue.json`의 `done` 플래그 업데이트

### 필수 애니메이션 (보스)
- walking-4-frames (걷기)
- fight-stance-idle-8-frames (대기)
- cross-punch (근접공격)
- high-kick (강공격)
- taking-punch (피격)
- falling-back-death (사망) — 선택
- fireball (원거리) — 선택

### 방향 규격
- 4방향: south, east, north, west
- south 실패 시 east 미러링으로 대체
