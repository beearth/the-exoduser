## 보스 렌더링 우선순위

1. **외부 아틀라스** (`atlas_bosses.png/json`) — PixelLab 생성 개별 캐릭터
2. **보스 걷기 아틀라스** (`atlas_boss_walk.png`) — 통합 행 기반 시스템
3. **레거시 아틀라스** (`atlas_boss01.png`) — 구버전
4. **절차적 렌더링** (`drawBossBody`) — 위 전부 없을 때 폴백

## 보스 개별 애니메이션 파일 (img/bosses/)

### 폴더 구조
```
img/bosses/boss_NAME/
├── metadata.json          (PixelLab 생성 정보)
├── rotations/             (4방향 정지 프레임)
│   ├── south.png
│   ├── east.png
│   ├── north.png
│   └── west.png
├── animations/
│   ├── walking-4-frames/         (걷기 4프레임 × 4방향)
│   ├── fight-stance-idle-8-frames/ (대기 8프레임 × 4방향)
│   ├── cross-punch/              (공격1 6프레임 × 4방향)
│   ├── high-kick/                (공격2 7프레임 × 4방향)
│   ├── taking-punch/             (피격 6프레임 × 4방향)
│   ├── falling-back-death/       (사망, si0에만 존재)
│   └── fireball/                 (원거리공격, si0에만 존재)
└── dl.zip                 (전체 다운로드 압축)
```

각 애니메이션 하위: `south/`, `east/`, `north/`, `west/` 디렉토리에 `frame_000.png ~ frame_NNN.png`

| 애니메이션 | 프레임 수 | 오프셋 | 용도 |
|-----------|----------|--------|------|
| idle | 4 | 0 | 대기 |
| walk | 8 | 4 | 이동 |
| atk | 16 | 12 | 근접공격 |
| whirl | 16 | 28 | 회전참 |
| throw | 16 | 44 | 작살 던지기 (3단계: 던지기0~3, 끌려가기4~9, 착지10~end) |
| chain | 16 | 60 | 사슬기동 체인공격 |
| dash | 16 | 76 | 사슬기동 이동 (3단계: 시작0~3, 도약4~11, 착지12~15) |

### dash 3단계 재생 로직
- **dash_start** (frame 0~3): 시작 모션, 1회 재생 후 fly로 전환
- **dash_fly** (frame 4~11): 공중 비행 루프
- **dash_land** (frame 12~15): 착지 모션, 1회 재생 후 idle로 복귀
- `_dashPhase` 변수로 상태 관리 (0=none, 1=start, 2=fly, 3=land)

## PixelLab 워크플로우

1. `boss_gen_queue.json`에서 미완성 보스 선택
2. PixelLab `create_character` → character ID 획득
3. `animate_character` → 각 애니메이션 타입별 생성
4. 결과를 `img/bosses/boss_NAME/` 폴더에 저장
5. `atlas_bosses.png/json` 리빌드
6. `boss_gen_queue.json`의 `done` 플래그 업데이트

### 필수 애니메이션 (보스)
- walking-4-frames (걷기) → walk
- fight-stance-idle-8-frames (대기) → idle
- breathing-idle (숨고르기) → breath (없으면 idle 폴백)
- cross-punch (근접공격) → attack, windup (공유)
- high-kick (강공격) → rage_attack, rage_windup (HP≤50% 분노 모드 전용)
- taking-punch (피격) → stagger
- falling-back-death (사망) → death (없으면 idle 폴백) — 선택
- fireball (원거리) → windup (있으면 cross-punch 대신 사용) — 선택
