# 몬스터 스킨 시스템 (2026-04-07 정리)

## 아틀라스 구조

| 파일 | 크기 | 내용 |
|------|------|------|
| `img/atlas_ch1_8dir_{방향}.png` | 2048x1280 | 8방향 × 40종, 셀 256px, 8x5 레이아웃 |
| `img/atlas_ch1_8dir.json` | — | cell/cols/rows/mobs 메타데이터 |
| `img/atlas_ch1_8dir_walk_{방향}.png` | 8192x1280 | 걷기 애니 39종(skinless_hound 미완), 4프레임/종. idle 베이스 + walk 오버레이(이동 중만). walk 없으면 idle 폴백 |

### 방향 파일명
`south`, `south_east`, `east`, `north_east`, `north`, `north_west`, `west`, `south_west`

## 코드 상수

```
_MOB8_TOTAL = 40   // 아틀라스 내 몹 총 수
_MOB8_COLS  = 8    // 아틀라스 열 수 (행 = ceil(40/8) = 5)
```

## 스킨 배정 (스폰 시)

`mkEn()` → `_assignMobSkin(e)`:
```
seed = (etype × 7 + floor(homeX × 31)) & 0x7fffffff
idx  = seed % _MOB8_TOTAL
col  = idx % _MOB8_COLS
row  = floor(idx / _MOB8_COLS)
```
- 보스(`e.ib`)는 제외 — `atlas_bosses.png`로 별도 렌더
- meta/JSON 로드 불필요, col/row 즉시 확정
- etype + 스폰 좌표 기반 결정적 시드 → 같은 몹은 항상 같은 스킨

## 렌더 경로 (3곳)

### 1. 일반 렌더 (33193줄~)
```

방향 미로드 시 → south 폴백
아틀라스 미로드 시 → _usedSpr=true (투명, 동그라미 안 그림)
```

### 2. eLite 간소화 렌더 (32496줄~)
화면 밖/렌더 캡 초과 몹. 8dir south 고정으로 간소 렌더.

### 3. 구울 렌더 (32414줄~)
`e._isGhoul` 몹. 4방향 스프라이트 삭제 → 8dir 아틀라스 폴백.

### 4. 시체 잔해 (12245줄 `_addCorpse`)
사망 시 south 방향 스프라이트를 48px 캔버스에 캡처 → 래그돌 물리.
`_ch8Atlas[1].dirs['south']` → `e._mob8Col/Row` 기반 drawImage.

### 5. Gib 파편 (12280줄 `_addGibs`)
폭발 사망 시 south 스프라이트를 48px로 캡처 → 랜덤 잘라서 파편 생성.
시체와 동일하게 8dir 아틀라스 사용.
