# 몬스터 공격 시스템

## 핵심 원칙
- **모든 몬스터는 탄막으로만 공격** — 근접 비비기(접촉 데미지) 완전 제거
- 근접 몬스터도 가까이 와서 탄을 발사
- 플레이어와 겹치면 밀어내기 (데미지 없음)
- 탄막은 패링/회피 가능 — 리듬게임식 전투

## ATK 공식
```
일반몹: ATK = (49 + monLv×1) × dm × et.atkMul
보스:   ATK = (60 + monLv×1.5) × dm × et.atkMul
monLv = P.lv + 난이도오프셋
dm = 1 (고정)
```

## 탄 종류 (\_beanRoll)

### 근거리 제한
- **500px 이내**: 특수탄 금지, 일반탄만 발사
- 이유: 무지개탄/빨콩이 근접에서 발사되면 패링 불가능

### 탄 속도 보정
- 모든 적 탄막: 속도 ×1.3 (spawnProj에서 vx,vy ×1.3)

## 탄 발사 위치
- `_eMouthXY(e, ang)`: 몬스터 facing 방향 반지름×0.7 앞 (입 위치)
- `eProjAt(e, ang, spd, dmgMult, el, life, col, sz)`: 지정 방향 1발
- `radialProjs(e, count, spd, dmgMult, el, life, col, sz)`: 방사형 count×2발

## 근접 비비기 제거
- 접촉 데미지(contactCd) 코드 완전 제거
- `case 'attack'`: 보스 포함 모두 근거리 탄 발사로 대체
- 플레이어 겹침 시: 몬스터 밀어내기 (데미지 없음)
- 레어몹/특수AI의 직접 hurtP 호출 → eProjAt 탄 발사로 교체 (8종)

## 순간이동 후 공격
- 리치(etype7), M13 모드, 시간사도(etype95)
- 순간이동 직후 1발 탄막 발사 (뎀배율 ×0.8, 속도 3)
- 비비기 대신 패링 가능한 탄

## 몬스터 예비동작 (전조)
- 접촉 범위 진입 시 60f(1초) 예비동작
- 예비동작 중: 이동 정지 (\_aSpdM×0), 빨간 깜박임 + ❗ 표시
- 예비동작 완료 후 히트 판정 (현재 탄 발사로 대체됨)

## 한방캡
- 모든 데미지 경로는 `hurtP()` 통과 → 한방캡(HP 30%) 적용
- `P.hp -=` 직접 호출 금지 (한방캡 우회 방지)
- 한방캡 발동 시 1초 무적

## 기절 (pStun) 보호
- 기절 진입 시 무적 60f(1초)
- 기절 해제 시 무적 60f(1초)
- 탈출: Shift/Q/Ctrl 즉시 (무적 20~30f), 방향키 연타 15f 단축
