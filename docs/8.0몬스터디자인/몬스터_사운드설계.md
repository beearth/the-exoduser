# 몬스터 사운드 설계 — 현황 및 계획
> game.html 코드 기준 | 최종 갱신: 2026-03-27

---

## 1. 사운드 시스템 개요

몬스터 사운드는 **4가지 채널**로 구성된다.

| 채널 | 트리거 | 구현 상태 |
|------|--------|-----------|
| **사망(Death)** | `deathFX()` → `_deathSfxKey(et)` → `_pickDeathVar()` | etype별 8카테고리 + 랜덤변형 + 폴백 체인 (**일반몹 50% 확률**, 보스/구울 100%) |
| **피격(Hit)** | `hurtE()` → `_hitSfxForEtype(et)` | 보스 전용 + 일반 8카테고리 피치/키 분화 |
| **이동(Move)** | `updateE()` → `_ETYPE_SFX` / `_ARCH_SFX` | AI 행동별 10종 + crawl |
| **공격(Attack)** | `_bossStartPattern()` 56종 전체 SFX 매핑 | 보스 패턴 완료, 일반몹 미구현 |

## 4. 피격(Hit) 사운드 매핑

### 로직 (`hurtE` → `_hitSfxForEtype(et)`)

```
if(보스)        → boss_hit (vol 0.3, pitch 1.0±0.2) + 15% monster_hurt
else(일반몹)    → _hitSfxForEtype(etype) → {key, pitch} (vol 0.3, pitch ±0.15 랜덤)
```

- **예외**: bladeShard / maliceHunt / iceBlade 옵션 시 피격음 생략 (별도 스킬 사운드)
- **DOT 피격**: 사운드 로직 변경 없음 (noSound 플래그 무관)
- **회전참(_wwHitSfx)**: 별도 로직, 변경 없음

### 4-1. 카테고리별 피격 사운드 매핑 (`_hitSfxForEtype`)

| 카테고리 | Set | 사운드 키 | 기본 피치 | 특성 |
|----------|-----|-----------|-----------|------|
| HUMANOID | `_HUMANOID_ET` | `monster_hit` / `monster_hit1` (50:50) | 1.0 | 기본 인간형 |
| INSECT | `_INSECT_ET` | `monster_hit1` | 1.3 | 높은 톤 (벌레 껍질) |
| GHOST | `_GHOST_ET` | `monster_hit` | 0.7 | 저음 에코감 |
| FLESH | `_FLESH_ET` | `monster_hit_flesh1/2` | ±0.1 랜덤 | 살덩이 전용 피격 (검+살점) |
| BONE | `_BONE_ET` | `monster_hit1` | 1.15 | 뼈 부딪힘 |
| BEAST | `_BEAST_ET` | `monster_hit` | 0.9 | 낮은 짐승 톤 |
| MAGIC | `_MAGIC_ET` | `monster_hit1` | 1.2 | 마법 잔향 |
| ARMOR | `_ARMOR_ET` | `flat_hit` | 1.0 | 금속 타격음 |
| 폴백 | 미매핑 etype | `monster_hit` / `monster_hit1` (50:50) | 1.0 | HUMANOID와 동일 |

> **피치 랜덤 편차**: 모든 카테고리에 `±0.15` 랜덤 적용 (`_r(pitch, 0.15)`)
> **볼륨**: 전 카테고리 동일 0.3

### 4-2. 체크 순서 (코드 내 if 분기 순서)

1. INSECT → 2. GHOST → 3. FLESH → 4. BONE → 5. BEAST → 6. MAGIC → 7. ARMOR → 8. 폴백(HUMANOID)

> 사망 사운드와 동일한 8개 카테고리 Set 재사용. etype 50(기형 보행자)은 HUMANOID_ET와 FLESH_ET 양쪽에 있지만, 피격 함수에서는 FLESH_ET가 먼저 체크되므로 `monster_hit pitch 0.85` 재생. (사망 사운드에서는 HUMANOID_ET 우선)

## 9. 장별 음향 특성 — 원소별 고유음 방향

### 1장 썩은 숲 + 얼음길 (etype 0~29)
- **기본 톤**: 축축한 숲 + 얼음 깨지는 소리
- **사망**: 인간형(비명) + 뼈/얼음 파쇄
- **이동**: 발소리 + 얼음 위 걷기 + 눈 밟는 소리
- **방향**: 빙결 사운드(결정 깨짐, 서리 바람) 강화, 반향 있는 동굴 리버브

### 2장 야생악마길 (etype 30~39)
- **기본 톤**: 짐승 울음 + 숲속 웅크림
- **사망**: 짐승 비명 + 육중한 쓰러짐
- **이동**: 네발 달리기 + 갈퀴 긁는 소리 + 낙엽 밟힘
- **방향**: 포효/울부짖음 추가, 야생 동물 사운드 레이어링

### 3장 악마길 (etype 40~49)
- **기본 톤**: 악마적 에코 + 비명 + 어두운 울림
- **사망**: 유령 소멸(에코 감쇄) + 살덩이 터짐
- **이동**: 부유 웅웅 + 사슬 딸깍 + 발톱 긁기
- **방향**: 깊은 리버브, 저주 속삭임, 악마 속성 강조

### 4장 괴물길 (etype 50~59)
- **기본 톤**: 축축한 기어감 + 뼈 갈리는 소리
- **사망**: 살덩이 터짐(습한) + 뼈 흩어짐
- **이동**: monster_crawl 전용 — 축축한 기어감, 점액 소리
- **방향**: 젖은 표면 사운드, 기형적 신음, ASMR적 불쾌감

### 5장 해저길 (etype 60~69)
- **기본 톤**: 수중 울림 + 거품 + 산호 긁힘
- **사망**: 금속(갑각) 깨짐 + 물거품 + 짐승 비명
- **이동**: 물속 걷기 + 촉수 미끄러짐 + 거품
- **방향**: 수중 필터(로우패스), 거품 레이어, 고래 울음 같은 배경

### 6장 용암길 (etype 70~79)
- **기본 톤**: 화염 쉭쉭 + 용암 부글 + 암석 균열
- **사망**: death_demon(악마 비명) + 화염 소멸 + 용암 튀김
- **이동**: 뜨거운 바닥 밟기 + 불꽃 튀김 + 암석 굴러감
- **방향**: 화염 크래클링, 용암 버블, 열기 왜곡 필터

### 7장 고통의길 (etype 80~89)
- **기본 톤**: 비명 + 사슬 + 금속 마찰 + 고통의 신음
- **사망**: 갑옷 쓰러짐 + 뼈 부서짐 + 유령 소멸
- **이동**: **현재 전체 무음** → 사슬 끌기, 금속 삐걱, 공포의 속삭임 추가 필요
- **방향**: 인더스트리얼 사운드(금속, 사슬, 톱니), 비명 에코, 가장 공포감 높은 사운드
