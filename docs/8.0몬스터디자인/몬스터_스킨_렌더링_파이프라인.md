> 2026-04-07 | 8방향 베르세르크 스킨 시스템

## 스킨 할당 흐름

```
몹 스폰 → _assignMobSkin(e)
  ├─ 보스(e.ib) → _assignBossSkin(e, si) → 보스 전용 아틀라스
  └─ 일반몹
      ├─ 1) _ch8Atlas[chN] 체크 (8방향 베르세르크)
      │   ├─ mobList 있으면 → e._mob8dir=true, 랜덤 스킨 할당
      │   └─ 없으면 → 건너뜀
      └─ 2) _chAtlas[chN] 체크 (4방향 기존)
          ├─ mobList 있으면 → e._mobSkin 할당 (폴백용)
          └─ 없으면 → e._mobSkinPending=true (대기)
```

## 렌더링 우선순위 (33088번 블록)

```
적 렌더 루프 (매 프레임)
  ├─ 1) WebGL instancing (_ensGL) — e._mobCh 있으면 건너뜀
  ├─ 2) 몬스터 스킨 아틀라스 블록 (e._mobCh 필요)
  │   ├─ 2a) e._mob8dir=true → _ch8Atlas[chN] 8방향 렌더링 ★최우선★
  │   │   └─ _mobFacingDir8(e.facing) → 8방향 중 정확한 방향 아틀라스 선택
  │   └─ 2b) 폴백 → _chAtlas[chN] 4방향 렌더링
  │       └─ _mobDirKey(e.facing) → S/E/N/W 중 선택 + west→east flip
  ├─ 3) _atlasE (신규 적 아틀라스) — e._mob8dir면 건너뜀
  ├─ 4) _atlas (레거시 적 아틀라스) — e._mob8dir면 건너뜀
  └─ 5) drawEnBody (최종 폴백)
```

```javascript
// 8방향 (atan2 → 0~2PI 정규화 → 45도 단위 스냅)
```

## 아틀라스 빌드 방법

```bash
# 1. PixelLab에서 8방향 캐릭터 생성 (ai_freedom=999, detailed shading, high detail)
# 2. 8방향 PNG 다운로드 → img/ch1_8dir/{이름}/{방향}.png
# 3. 아틀라스 빌드:
cd G:/hell && node -e "
const fs=require('fs');const {createCanvas,loadImage}=require('canvas');
const CELL=48,DIRS=['south','south-east','east','north-east','north','north-west','west','south-west'];
const mobDir='img/ch1_8dir';
const mobs=fs.readdirSync(mobDir).filter(d=>fs.statSync(mobDir+'/'+d).isDirectory()).sort();
const cols=Math.ceil(Math.sqrt(mobs.length)),rows=Math.ceil(mobs.length/cols);
async function b(){for(const dir of DIRS){const c=createCanvas(cols*CELL,rows*CELL);const ctx=c.getContext('2d');
for(let i=0;i<mobs.length;i++){const p=mobDir+'/'+mobs[i]+'/'+dir+'.png';
if(fs.existsSync(p)){const img=await loadImage(p);ctx.drawImage(img,0,0,img.width,img.height,(i%cols)*CELL,Math.floor(i/cols)*CELL,CELL,CELL)}}
fs.writeFileSync('img/atlas_ch1_8dir_'+dir.replace(/-/g,'_')+'.png',c.toBuffer('image/png'))}
const meta={cell:CELL,cols,rows,total:mobs.length,directions:DIRS,mobs:{}};
mobs.forEach((m,i)=>{meta.mobs[m]={idx:i,col:i%cols,row:Math.floor(i/cols)}});
fs.writeFileSync('img/atlas_ch1_8dir.json',JSON.stringify(meta,null,2));
console.log('Done:',mobs.length)}b();
"
```

## idle 배회 시스템

- 어그로 안 걸린 몹(거리 1800+) → 30% 속도로 랜덤 방향 배회
- 1~3초마다 방향 전환, 벽 충돌 시 즉시 전환
- `e._isMoving=true` → 걷기 스프라이트 활성화
- `e.facing=e._wanderA` → 8방향 방향 반영

## PixelLab 프롬프트 규칙

- **필수 키워드**: `Berserk`, `Kentaro Miura`, `dark fantasy`
- **ai_freedom**: 999 (최대)
- **shading**: detailed shading
- **detail**: high detail
- **body_type**: humanoid만 (quadruped은 API 버그)
- **비인간형 불가**: 슬라임, 구체, 4발 동물, 벌레 등은 PixelLab으로 못 만듦
