## 합체 흡수 규칙

합체 시 하위 스킬은 **흡수**되어 독립 사용 불가. 상위 합체는 하위의 흡수도 포함.

| 합체 | 흡수 스킬 |
|---|---|
| sixFuse (뇌전) | omniBeam |
| elemFuse (추적뇌전) | fanShot, **omniBeam**, elemMissile |
| stormBeam (뇌전나선) | detonate, giantSlam, fanShot, omniBeam, elemMissile |
| dimBreach | magicBlink |
| dimSlam | magicBlink, chainAssault |
| dimThunder | magicBlink, chainAssault, chainSlam |
| whirlDet | detonate |
| slamStorm | detonate, giantSlam |
| bladeFuse | guardian |
| plagueFuse | guardian, plagueBurst |
| iceMortar | iceOrb |
| boneStorm | boneWall |
| elecRepent | hellRay, boneWall |
| shieldFuse | shieldThrow |
| holyFuse | holyPrison |
| holyIce | holyPrison, iceStorm |

흡수 처리 위치 (코드):
- `_getAbsorbed()` — 흡수 목록 정의
- `_fKick` — 슬롯에서 제거
- `_fusedOut` — 스킬패널 숨김
- `_fuseKick` — 습득 시 자동 정리
- `_popFusedOut` — 슬롯 팝업 숨김
- E슬롯 전환 팝업 — `_popAbsorbed` 필터

## 레이저(빔) 스킬 MP 소비량

회전참(ST)과 동일 구조: `최대MP × 틱% × 레벨배율 × pMagicCost()`

| 스킬 | 틱% | 레벨배율 공식 | 1렙 실소비 | 10렙 실소비 | 비고 |
|---|---|---|---|---|---|
| 멸살광선 (단일) | 0.0015 | 0.5+(lv-1)×0.078 | mmp×0.075%/f | mmp×0.18%/f | 회전참과 동일 |
| 뇌전 (sixFuse) | 0.002 | 0.5+(lv-1)×0.078 | mmp×0.1%/f | mmp×0.24%/f | 6가닥이라 33% 증가 |
| 추적뇌전 (elemFuse) | 0.002 | ×0.9 적용 | mmp×0.09%/f | mmp×0.22%/f | 유도라 약간 할인 |
| 뇌전나선 (stormBeam) | 0.001 | 없음 (고정) | mmp×0.1%/f | mmp×0.1%/f | 회전참이 메인이므로 66% 할인 |

**참고**: 회전참 ST 소비 = `mst × 0.0015 × (0.5+(lv-1)×0.078)`

### 빔 관통

빔은 투사체 관통 패시브(`pPierce`) + 장비 `bowPierce`에 의해 관통 확률이 결정됨.
- 패시브/장비 없으면 **0% 관통** → 첫 적에서 끊김
- 패시브 1렙 = 5%, 10렙 = 50%
- 캡 95%

## 합체 비용 공식

- **합체 수행**: 성급 × 3 SP
- **합체 강화**: `_fuseUpSpCost(curLv, gemCnt)` — 기본테이블 + (성급-1)
- **스킬 습득**: 모든 스킬 10 SP 통일
