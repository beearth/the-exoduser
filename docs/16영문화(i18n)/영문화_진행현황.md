# 영문화(i18n) 진행 현황

## 시스템 구조

### 번역 함수: `_T(s)` (game.html:3007)
- `OPT.lang!=='en'`이면 한글 그대로 반환
- `_EN[s]` 딕셔너리에 있으면 영문 반환
- 장비명은 `_EN_PFX` + `_EN_BASE` 조합 번역 (접두사+기본형)

### 자동 번역 함수들
- `showPH(t,c)` — 내부에서 `_T(t)` 자동 적용 (line 26145)
- `addTxt(x,y,t,c,l)` — 내부에서 `_T(t)` 자동 적용 (line 26360)
- `notify(msg)` — 내부에서 `_T(msg)` 자동 적용 (line 26146)

### 배열 상수 헬퍼 함수
| 함수 | 원본 배열 | 영문 배열 | 비고 |
|------|-----------|-----------|------|
| `_diffName(i)` | `DIFF_NAME` | `DIFF_NAME_EN` | 난이도 이름 |
| `_rarName(i)` | `RARITY_N` | `RARITY_N_EN` | 등급 이름 |
| `_slotName(i)` | `SLOT_KR` | `SLOT_EN` | 장비 슬롯명 |
| `_rarLabel(i)` | `RAR_LABEL` | `RAR_LABEL_EN` | 줍기 등급 라벨 |

### 번역 딕셔너리: `_EN` (line 2778~3020)
- 약 270+ 엔트리
- 카테고리: 스테이지, 튜토리얼, 시스템메시지, 스킬명, HUD, 메뉴, 속성, 패시브, 무기, 어픽스, 설정UI, 캔버스UI, 인벤, 대장간, 결정, 적 이름 등
- `_EN_PFX`: 장비 접두사 30+종
- `_EN_BASE`: 장비 기본형 15+종

### 언어 전환 UI
- 설정 패널 `#optLang` select (line 696)
- `_applyLang()`: HUD 슬롯 라벨, 패널 헤더, 설정 UI 전체, BGM옵션, 화면모드옵션, 버튼, 도움말 오버레이, 그래픽 프리셋, 스킬 슬롯 갱신, 슬롯 title 속성 8건

## 작업 규칙

1. **_EN 딕셔너리 우선 등록** → 사용처에서 `_T()` 감싸기
2. **배열 상수**는 영문 배열 별도 + 헬퍼 함수 (`_diffName`, `_rarName`, `_slotName`, `_rarLabel`)
3. **OPT.lang==='en' 삼항**은 이미 처리된 것으로 간주
4. **showPH / addTxt / notify**는 함수 내부 자동 _T → 호출처 수정 불요
5. **정적 HTML**은 `_applyLang()`에서 동적 교체
6. **Electron/game.html**은 game.html 완료 후 복사
7. 매 Phase 완료마다 이 문서 업데이트
