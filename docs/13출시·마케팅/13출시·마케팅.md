# 13. 출시 · 마케팅

## BIC 2026 데모 빌드 (EXODUSER_prototype)

### 데모 제한 사항
| 항목 | 제한 | 비고 |
|------|------|------|
| **레벨 캡** | **Lv.200** | 200 도달 시 경험치 획득 중단 + "DEMO MAX LEVEL" 표시 |
| 챕터 | 1-1(썩은숲 1층)만 | ch1_1만 포함, 1-2 이후 및 ch2~7 미포함 |
| 캐릭터 외형 | 엑소듀서 전사만 | 나머지 4종 스프라이트 미포함 |
| 세이브 | 로컬 파일(saves/) | Supabase 미연결, 서버 종료 시 유지 |
| 워터마크 | 우상단 "DEMO BUILD - BIC 2026" | 투명도 18% |

### 빌드 구성
```
EXODUSER_prototype/
├── EXODUSER.bat        ← 더블클릭 실행 (Node.js 필요)
├── server.js           ← 정적 파일 서버 + API 스텁 (포트 3000)
├── index.html          ← 시네마틱 프롤로그 + 로비 + 캐릭터 선택
├── game.html           ← 게임 본체 (maps_data.js 인라인, DEMO 워터마크)
├── README.txt
├── img/                ← UI, 스킬아이콘, 펫, 시네마틱 이미지 등
├── assets/             ← VFX, ch1 맵 타일/오브젝트
├── video/intro.mp4     ← 인트로 영상
├── bgm/                ← 1장 + 공통 BGM
├── sfx/                ← 효과음 전체
├── sprites/            ← 스킬 VFX 스프라이트
└── insect_tileset_min_pack/
```

### 실행 흐름
1. `EXODUSER.bat` → `node server.js` → http://localhost:3000
2. `index.html`: 시네마틱 프롤로그 → 로비 → 캐릭터 생성/선택
3. `game.html`: 1장 플레이 (Lv.200 캡)
