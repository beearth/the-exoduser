## 생성 도구 추천

| 도구 | 적합도 | 비고 |
|------|--------|------|
| **GPT-4o (DALL-E 3)** | ★★★★ | 텍스트 렌더링 가장 정확, 레이아웃 지시 잘 따름 |
| **Midjourney v6** | ★★★ | 스타일 좋지만 텍스트 부정확, 후가공 필요 |
| **Figma/Canva** | ★★★★★ | 직접 제작이 가장 정확, 프롬프트 기반 아님 |
| **ludo-ai** | ★★★ | 게임 에셋 특화지만 UI 가이드는 약함 |

### 권장 워크플로우
1. DALL-E 3으로 **배경/프레임/컨트롤러 실루엣** 생성
2. Figma/포토샵에서 **텍스트 + 라벨 + 아이콘** 수동 배치 (AI 텍스트 렌더링 부정확)
3. 게임 내 `img/guide/` 폴더에 저장
4. 설정 화면 또는 첫 패드 연결 시 표시

## 인게임 표시 위치

| 트리거 | 표시 이미지 | 방식 |
|--------|-------------|------|
| 첫 패드 연결 | 이미지 1 (전투) + 이미지 2 (모디파이어) | 풀스크린 오버레이, A로 닫기 |
| 설정 > 조작법 | 이미지 1 + 2 + 3 | 탭 전환 |
| 인벤토리 첫 오픈 (패드) | 이미지 3 (UI) | 반투명 오버레이, 3초 후 자동 숨김 |
| 일시정지 (Start) | 이미지 1 축소판 | 하단 힌트 바 |
