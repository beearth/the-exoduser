# 14. 밸런스 + 수치 테이블

## 스탯 5종 (STATS)
| 스탯 | 효과 |
|---|---|
| STR | 물리 공격력 배율 (`statStr()`) |
| DEX | 공격속도/회피 배율 (`statDex()`) |
| INT | 마법 공격력 배율 (`statInt()`), INT 100+에서 에너지쉴드 해금 |
| VIT | 최대 HP/방어력 |
| LCK | 크리티컬/드롭률 |

## 데미지 공식
- `hurtE(e, dmg, ang)` — 적에게 데미지
- `hurtP(dmg)` — 플레이어 피격
- DEF 감산, 50% 캡
- **수정 금지**

## 강화 시스템
- `enhMul()`, `enhCost()` — +1~+20
- 강화 확률: 99 × 0.99^n %
- **수정 금지**

## 레벨
- `P.lv` 상한 1000
- `addExp(v)` 경험치 추가

## 장비 내구도
- DUR_MAX 레어리티별
- 파손 시 50% 성능 감소
- 대장간 수리 탭

## 장비 제작
- CRAFT_COST 100악의 → 영웅 이상 랜덤 장비
