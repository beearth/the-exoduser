# 14. 밸런스 + 수치 테이블

## 스탯 5종 (STATS)
| 스탯 | 효과 |
|---|---|
| STR | 물리 공격력 배율 (`statStr()`) |
| DEX | 공격속도/회피 배율 (`statDex()`) |
| INT | 마법 공격력 배율 (`statInt()`), INT 100+에서 에너지쉴드 해금 |
| VIT | MAX HP +2/pt, 방어력 +0.2, MAX ST +1, HP리젠 +0.003 |
| LCK | 크리티컬/드롭률 |

## 데미지 공식
- `hurtE(e, dmg, ang)` — 적에게 데미지
- `hurtP(dmg)` — 플레이어 피격
- DEF 감산, 50% 캡
- **최소 데미지 1 보장** — DR 중첩 시 `~~()` 절삭으로 0뎀 방지. STR 배율 적용 후 `if(dmg<1)dmg=1`
- **엘리트/챔피언 시스템 비활성화** — `rollElite()` 항상 0 반환. 모든 몹 일반 등급. 모디파이어(M01~M28)/오라(함성/치유/신속/수호) 미적용
- **수정 금지**

## 강화 시스템
- `enhMul()`, `enhCost()` — +1~+20
- 강화 비용: `ceil((1 + n×0.15) × 1.5)` × 등급배율 (50% 상향 적용됨)
- 등급배율: 일반 0.5, 마법 0.7, 희귀 1.0, 영웅 1.3, 전설 1.8
- 강화 확률: 99 × 0.99^n %

## 레벨 & SP 수급
- `P.lv` 상한 없음 (1000+ 초월 시스템)
- `addExp(v)` 경험치 추가
- `_calcMaxExp(lv)`: lv<1000 → `15+lv×2.5+lv²×0.05`, lv≥1000 → `5000000+(lv-999)²×10000`

### SP/AP 획득
| 조건 | 보상 |
|---|---|
| 레벨업 | SP +3 |
| 5의 배수 레벨 | SP +2 추가 (합계 +5) |
| 10의 배수 레벨 | AP +2 |
| 일반 보스 처치 | AP +5 |
| 장보스(챕터 최종) 처치 | AP +10 |

### SP/AP 저장·복원
- 로드 시 저장된 `P.sp`, `P.ap` 값을 **그대로 복원** (재계산 없음)
- SP 변동은 레벨업·스킬습득·합체·스탯투자·보스클리어 등 이벤트마다 실시간 반영 + 자동저장
- 기본 스킬 (kiSlash, fireball): **완전 무료 보너스** — 습득비·강화비 0, 해제해도 SP 환불 없음

### 초월 레벨 (Lv.1000+)
| 항목 | 값 |
|---|---|
| 경험치 공식 | `5,000,000 + (lv-999)² × 10,000` |
| 경험치 고정값 | 일반몹 1, 엘리트 2, 보스 10 (기존 공식 무시) |
| 일일 레벨업 제한 | **1회/일** (`P._transLvDate`, `P._transLvCount`) |
| 보상 | SP +3/렙, 5렙마다 SP+2, 10렙마다 AP+2 (일반과 동일) |
| HUD 표시 | 경험치바 보라색, 텍스트 "⚡ T.{lv}" |

예시 필요 경험치 (몹 1XP 기준 ≈ 필요 킬 수):
| 레벨 | maxExp | 필요 맵 (~1000킬/맵) |
|---|---|---|
| 1000→1001 | 5,010,000 | ~5,010판 |
| 1005→1006 | 5,360,000 | ~5,360판 |
| 1010→1011 | 6,210,000 | ~6,210판 |
| 1050→1051 | 31,010,000 | ~31,010판 |
| 1100→1101 | 107,010,000 | ~107,010판 |

## 몬스터 스탯 성장
| 항목 | 공식 | 비고 |
|---|---|---|
| HP | `(보스?1500:150) × 1.014^monLv × dm × et.hpMul` | 1.4% 복리 성장 |
| ATK | `(보스?50:30) + monLv×(보스?4:3) × dm × et.atkMul` | 레벨당 선형 증가 |
| monLv | `P.lv + DIFF_LV_OFF[난이도]` | 플레이어 레벨 + 난이도 오프셋 |
| 피격쿨다운 | 없음 (핵슬이므로 동시피격 제한 없음) | 투사체마다 개별 데미지 적용 |
| eShield | `HP와 동일` | 이원 방어: 물리→쉴드10%/HP100%, 마법→쉴드100%/HP10% (기본 90% 패널티, 패시브로 관통) |
| 포이즈 | `HP × 50%` | |
| 물리관통(pPhysPen) | `min(50%, 10%+Lv×4%)` | 패시브: 물리→쉴드 패널티 관통. 기본10%, 10lv=50% |
| 마법관통(pMagPen) | `min(50%, 10%+Lv×4%)` | 패시브: 마법→HP 패널티 관통. 기본10%, 10lv=50% |

## 장비 ATK 시스템 (디아블로식)

### base ATK 생성
```
_atkMax = 200 + tier × 200    (tier0=200, tier4=1000)
_atkFloor = [0, 0.05, 0.15, 0.25, 0.40][rarity]   (전설=40% 최소보장)
_atkRoll = _atkMax × (floor + (1-floor) × random²)  (skew: 고뎀 극악)
```

| tier | _atkMax | 전설(40%보장) 범위 | 일반(0%보장) 범위 |
|---|---|---|---|
| 0 | 200 | 80~200 | 0~200 |
| 1 | 400 | 160~400 | 0~400 |
| 2 | 600 | 240~600 | 0~600 |
| 3 | 800 | 320~800 | 0~800 |
| 4 | 1000 | 400~1000 | 0~1000 |

- 무기/석궁/투구 동일 공식, wtype/btype의 atkMul 추가 곱셈
- 어픽스 3종(예리한/맹렬한/파멸의) 각 50~330 추가 (skewRoll:3)
- 풀옵 이론 최대: base 1000 + 어픽스 990 = **~1990**

### 공격력 기반 함수
| 함수 | 기반 장비 | 공식 |
|---|---|---|
| `meleeRef()` | 무기(wp) | `(wp.atk × (1+강화배율) + flatAtk어픽스 + 15 + baseAtk) × 봉인` |
| `bowRef()` | 석궁(bw) | `(bw.atk × (1+강화배율) + flatAtk어픽스 + 15 + baseAtk) × 봉인` |
| `magicRef()` | 투구(hm) | `(hm.atk × (1+강화배율) + flatAtk어픽스 + 15 + baseAtk) × 봉인` |
