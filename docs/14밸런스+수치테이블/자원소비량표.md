# 자원 소비량 총정리

> game.html 2026-04-14 기준. 할인 패시브 0레벨 기준

### 악의 소비 전역 할인
- 악의를 쓰는 모든 체크/차감 경로는 최종적으로 `_malCost(v)=max(1,ceil(v×0.5))` 적용.
- 본 표의 악의 수치는 `기준 원가`이며, 실제 플레이 소모는 반값(50%) 기준으로 계산.

---

## 비용 공식

```
단발성: min( COST_BASE(50) × (1 + (skLv-1) × DPS스케일) × 할인계수, P.mst × 0.90 )
채널링: 기본(30)/초 × (1 + (Lv-1) × DPS스케일)
기본공격: 고정 5~25 (연타용, 별도 공식)

핵심: 비용 성장률 = DPS 성장률 → 레벨 올려도 효율 동일
최대ST 90% 캡: ST 풀이면 어떤 스킬이든 항상 발동 가능 (저렙 보호)
```

## 변경 이력

```
기존: base(10) × skLv          → Lv1=10, Lv10=100 (×10)
1차: base(250) × (1+(Lv-1)×DPS) → Lv1=250, Lv10=362~587 → 저렙 사용 불가
2차: base(50) × (1+(Lv-1)×DPS) → Lv1=50, Lv10=117~252 (고쿨=고비용)
     + P.mst×90% 캡 (풀ST면 항상 발동 가능)
채널링: 10/초 → 30/초 기본, DPS비례 성장
업화선: 10/발 → 50/발, 마력연사: 10/발 → 12/발
```
