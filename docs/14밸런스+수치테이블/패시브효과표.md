## DPS 배율 함수 상세 (2026-04-18 합연산 변경)

### 통합 배율 함수 — `_passDmgSum(type)`

> **핵심 변경**: 모든 damage multiplier를 **합연산(additive)**으로 통합. 개별 함수들이 이 함수를 반환하며, 반환값이 곧 최종 multiplier

```javascript
function _passDmgSum(type){
  let s=0;
  s += PASSIVES.pAtk × 0.15       // 맹공: +15%/lv (항상)
  s += (P._altAtk||0)             // 제단 맹공 보너스 
  if(type==='melee')   s += PASSIVES.pMelee × 0.15        // 광전사: +15%/lv (근접전용)
  if(type==='magic')   s += PASSIVES.pMagic × 0.15 + 마법어픽스 + 마법내재  // 마력폭주: +15%/lv (마법전용)
  if(type==='bow')     s += PASSIVES.pBow × 0.15          // 정밀사격: +15%/lv (석궁전용)
  if(type==='xbow')    s += PASSIVES.pXbow × 0.05         // 쇠뇌력: +5%/lv (쇠뇌전용)
  if(type==='dot')     s += PASSIVES.pDot × 0.15          // 침식: +15%/lv (DOT전용)
  if(P.hp >= P.mhp × 0.7) s += PASSIVES.pAbund × 0.20   // 풍요: +20%/lv (내HP≥70%)
  return 1 + s
}
```

**예시 (최대 레벨 기준 pAtk=10, pMelee=10, pAbund=10, HP≥70%)**:
- 이전 (곱연산): `2.0 × 2.5 × 3.0 = 15.0×`
- 현재 (합연산): `1 + 1.0 + 1.5 + 2.0 = 4.5×`

### 별도 적용 함수

| 함수 | 조건 | 적용 방식 |
|---|---|---|
| `_predBonus(e)` | 적HP≤50% | `dmg *= (1 + _predBonus(e))` — hitEnemy에서 별도 적용 |

### hitEnemy 최종 데미지 계산

```javascript
dmg = ref() × stat() × passive_multiplier()  // 기본 데미지
dmg *= (1 + _predBonus(e))  // 약자포식 추가 (적HP≤50%)
dmg *= (1 + 조건부_어픽스)   // 어픽스 (abundDmg, predDmg, etc)
```

**약자포식 + 어픽스 시너지:**
- 약자포식 패시브 _predBonus(e): 적HP≤50% 시 합연산 추가 (최대 +200%)
- 약자포식 어픽스 (predDmg): 적HP≤50% 시 별도 곱연산 (1회만)

| 함수 | 연동 패시브 | 조건 | 레벨당 | 10렙 |

> 풍요(HP≥70%)와 약자포식(HP≤50%)은 조건이 상호배타 → 회복력 동시 발동 불가
