# 스킬 데미지 공식 총정리

> game.html 2026-04-11 기준 (DPS 균등화 + 50% 곡선 재밸런스)

## 기본 공식 (2026-04-11 변경)

```
데미지 = ref() × stat() × passive() × _skMul('skillId') × _fuseMul('skillId')

_skMul(skId) = (base + (skLv-1) × growth × 0.5) × 1.006^(P.lv-1)
  → 스킬레벨 효과 절반 (10단계→20단계 쪼개기)
  → 캐릭터 레벨 복리 보너스: 1.006^(lv-1) (몬스터 HP v2 이차→선형 커브의 약 50% 따라가기)

캐릭터 레벨 보너스: Lv1=×1, Lv20=×1.12, Lv100=×1.81, Lv500=×20, Lv1000=×400
나머지 50%는 장비/강화/세팅으로 채움 (장비 의존 게임)

스킬레벨 캡: Lv20 (50렙당 1렙 해금), Lv1→Lv20 = 약 ×4.8 성장

무기ATK = ref 함수가 결정:
  meleeRef() = 주무기ATK + enhMul + slotFlat + baseAtk
  bowRef()   = 석궁ATK + enhMul + slotFlat + baseAtk
  magicRef() = 마법ATK + enhMul + slotFlat + baseAtk

baseAtk = 15 + (stage-1)×2
```

## 통일 관통 시스템 (`_pierceRate`)

모든 투사체는 `_pierceRate` 베이스 값을 가지고, 어픽스/패시브로 추가됨.

### 공식
```
총 관통률 = 베이스 + bowPierce 어픽스 × 1000 + pPierce 패시브 × 100
보장 관통수 = floor(관통률 × 91 / 1000)  ← 1000=91, 2000=182
보장 초과 시: 1회=90%, 2회=80%, ..., 9회=10% (최대 +9회 추가)
최종 최대 = 보장수 + 9
```

### 스킬별 베이스 관통률

| 스킬 | 베이스 _pierceRate | 보장 (Lv1) | 비고 |
|---|---|---|---|
| **기본값 (모든 투사체)** | **50** | **4회** | `_resetPProj`에서 자동 부여 |
| 푸른비 (blueShot) | 200 | 18 | + 어픽스/패시브 |
| 악의 사냥 (maliceHunt) | 800 + Lv×20 + bowPierce×300 + pPierce×30 | 약 74 + 장비/패시브 가산 | 별도 로직 (`_mhMaxPierce`) |

## ref 함수 구조

```
meleeRef() = (주무기ATK + enhMul(강화) + slotFlatAtk + baseAtk) × 봉인배율
bowRef()   = (석궁ATK   + enhMul(강화) + slotFlatAtk + baseAtk) × 봉인배율
magicRef() = (마법ATK   + enhMul(강화) + slotFlatAtk + baseAtk) × 봉인배율

baseAtk = 15 + (stage-1)×2
봉인배율 = weaponSeal 중이면 ×0.3, 아니면 ×1
```

## 스탯 함수

```
statStr() = 1 + P.stats.str × 0.01   (STR 100 = ×2)
statInt() = 1 + P.stats.int × 0.01   (INT 100 = ×2)
statDex() = 1 + P.stats.dex × 0.01   (DEX 100 = ×2)
```

## 패시브 배율

```
pAtkMul()   = 물리 공격력 패시브 (pMelee 레벨 기반)
pBowMul()   = 석궁 공격력 패시브 (pBow 레벨 기반)
pMagicMul() = 마법 공격력 패시브 (pMagic 레벨 기반)
pShieldMul()= 방패 배율 (방패 장착 시)
```
