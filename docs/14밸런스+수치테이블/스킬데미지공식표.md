## 기본 공식

```
데미지 = ref() × stat() × passive() × _skMul('skillId') × _fuseMul('skillId')

_skMul(skId) = (base + (skLv-1) × growth) × (1 + P.lv × 0.01)
  → 스킬별 개별 배율 (_SK_MUL 테이블) × 캐릭터 레벨 보너스 +1%/Lv

캐릭터 레벨 보너스 (+0.2%/Lv): Lv1=×1.002, Lv100=×1.2, Lv500=×2, Lv1000=×3

무기ATK = ref 함수가 결정:
  meleeRef() = 주무기ATK + enhMul + slotFlat + 15 + baseAtk
  bowRef()   = 석궁ATK + enhMul + slotFlat + 15 + baseAtk
  magicRef() = 마법ATK + enhMul + slotFlat + 15 + baseAtk

baseAtk = 15 + (stage-1)×2
```

## ref 함수 구조

```
meleeRef() = (주무기ATK + enhMul(강화) + slotFlatAtk + 15 + baseAtk) × 봉인배율
bowRef()   = (석궁ATK   + enhMul(강화) + slotFlatAtk + 15 + baseAtk) × 봉인배율
magicRef() = (마법ATK   + enhMul(강화) + slotFlatAtk + 15 + baseAtk) × 봉인배율

baseAtk = 15 + (stage-1)×2
봉인배율 = weaponSeal 중이면 ×0.3, 아니면 ×1
```

## 스탯 함수

```
statStr() = 1 + P.stats.str × 0.01   (STR 100 = ×2)
statInt() = 1 + P.stats.int × 0.01   (INT 100 = ×2)
statDex() = 1 + P.stats.dex × 0.01   (DEX 100 = ×2)
```

## 패시브 배율

```
pAtkMul()   = 물리 공격력 패시브 (pMelee 레벨 기반)
pBowMul()   = 석궁 공격력 패시브 (pBow 레벨 기반)
pMagicMul() = 마법 공격력 패시브 (pMagic 레벨 기반)
pShieldMul()= 방패 배율 (방패 장착 시)
```
