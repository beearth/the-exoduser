# 덧셈 밸런스 설계서 — 정직한 성장

## 핵심 철학
- **모든 수치는 덧셈**. 곱연산 없음.
- **무기 데미지 1~1000** 이 세상의 최대 단일 수치.
- 스킬은 무기뎀 + 플랫 보너스.
- 누가 봐도 계산 가능. "내 데미지 = 무기 + 스탯 + 스킬 + 패시브"

---

## 1. 무기 데미지

| 등급 | ATK 범위 | 비고 |
|---|---|---|
| 일반(흰) | 1~50 | 초반 |
| 고급(녹) | 30~150 | 1지옥 중후반 |
| 희귀(파) | 100~350 | 2~3지옥 |
| 영웅(보라) | 250~600 | 4~5지옥 |
| 전설(금) | 450~1000 | 6~7지옥 |

강화: +1당 **+5 ATK** (고정). +20강 = +100 ATK.

---

## 2. 스탯 보너스 (덧셈)

SP 1포인트 = 스탯 1 투자. 레벨당 SP 2개.

| 스탯 | 효과 | 1000포인트 |
|---|---|---|
| **STR** | 물리 데미지 **+1** per pt | +1000 |
| **INT** | 마법 데미지 **+1** per pt | +1000 |
| **DEX** | 크리 확률 **+0.01%** per pt, 공속 **+0.05%** per pt | +10% 크리, +50% 공속 |
| **VIT** | HP **+5** per pt, DEF **+0.5** per pt | +5000 HP, +500 DEF |
| **LCK** | 크리 뎀 **+0.1%** per pt, 드롭 **+0.1%** per pt | +100% 크리뎀, +100% 드롭 |

---

## 3. 데미지 공식

### 물리 공격
```
최종 데미지 = 무기ATK + STR보너스 + 스킬보너스 + 패시브보너스
```

### 마법 공격
```
최종 데미지 = 무기ATK + INT보너스 + 스킬보너스 + 패시브보너스
```

### 석궁 공격
```
최종 데미지 = 활ATK + DEX보너스×0.5 + 스킬보너스 + 패시브보너스
```

### 크리티컬
```
크리뎀 = 최종데미지 × (1.3 + LCK보너스)
```
크리만 유일한 곱연산 (피해량 증폭이므로).

## 6. 몬스터 스케일링

### HP 공식
```
몬스터HP = 몬스터레벨 × 150
보스HP = 몬스터레벨 × 1500 (일반의 10배)
```

| 레벨 | 일반몹 HP | 보스 HP |
|---|---|---|
| 1 | 150 | 1,500 |
| 10 | 1,500 | 15,000 |
| 100 | 15,000 | 150,000 |
| 300 | 45,000 | 450,000 |
| 500 | 75,000 | 750,000 |
| 900 | 135,000 | 1,350,000 |

### ATK 공식
```
몬스터ATK = 몬스터레벨 × 2
보스ATK = 몬스터레벨 × 5
```

| 레벨 | 일반 ATK | 보스 ATK |
|---|---|---|
| 1 | 2 | 5 |
| 100 | 200 | 500 |
| 500 | 1,000 | 2,500 |
| 900 | 1,800 | 4,500 |

## 7. 플레이어 DPS 예시

### Lv1 (무기ATK 10, STR 0, 패시브0)

```
기검참 = 10 + 0 + 20(스킬1렙) + 0 = 30/타
초당 3회 = 90 DPS
```

vs 몬스터 HP 150 → **약 2초**에 처치

### Lv100 (무기ATK 150, STR 200, 패시브3렙)

```
기검참 = 150 + 200 + 20(스킬1렙) + 60(패시브) = 430/타
초당 3회 = 1,290 DPS
```

vs 몬스터 HP 15,000 → **약 12초**에 처치 (스킬 조합으로 5~7초)

### Lv500 (무기ATK 600, STR 1000, 패시브7렙)

```
기검참 = 600 + 1000 + 200(스킬10렙) + 140(패시브) = 1,940/타
초당 3회 = 5,820 DPS
```

vs 몬스터 HP 75,000 → **약 13초**에 처치 (스킬 조합으로 6~8초)

### Lv900 (무기ATK 1000, STR 1800, 패시브10렙)

```
기검참 = 1000 + 1800 + 200(스킬10렙) + 200(패시브) = 3,200/타
초당 3회 = 9,600 DPS
```

vs 몬스터 HP 135,000 → **약 14초**에 처치 (스킬 조합으로 7~9초)

## 8. 밸런스 핵심

- **모든 레벨에서 일반몹 처치 시간 ≈ 5~10초**
- **보스 ≈ 30~60초**
- 성장은 느끼되, 적도 같이 강해져서 긴장감 유지
- 스킬 선택 = DPS 차이가 아니라 **플레이 스타일 차이**
- 크리만 유일한 곱연산 → "운 좋으면 한방" 재미

---

## 9. DEF 공식

```
피해감소 = DEF / (DEF + 500)
최종피해 = 받는뎀 × (1 - 피해감소)
```

| DEF | 감소율 |
|---|---|
| 50 | 9% |
| 100 | 17% |
| 250 | 33% |
| 500 | 50% |
| 1000 | 67% |
| 2000 | 80% |

캡 없음. 하지만 수확체감(diminishing returns)이라 무한 쌓아도 100% 불가.
