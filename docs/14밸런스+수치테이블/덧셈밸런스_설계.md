# 덧셈 밸런스 설계서 — 정직한 성장

## 핵심 철학
- **기본은 덧셈**, 스탯·패시브·강화는 **곱셈 배율**.
- **무기 데미지 1~1000** 이 세상의 최대 단일 수치.
- 스킬은 무기뎀 기반 배율 + 레벨 보너스.
- 스탯/패시브/강화가 곱셈으로 스케일링되어 후반 성장 체감 확보.

---

## 1. 무기 데미지

| 등급 | ATK 범위 | 비고 |
|---|---|---|
| 일반(흰) | 1~50 | 초반 |
| 고급(녹) | 30~150 | 1지옥 중후반 |
| 희귀(파) | 100~350 | 2~3지옥 |
| 영웅(보라) | 250~600 | 4~5지옥 |
| 전설(금) | 450~1000 | 6~7지옥 |

강화: **복리 누적 곱셈 +1%/강** (step×1.01). +20강 ≈ ×1.22 ATK.

---

## 2. 스탯 보너스 (덧셈)

SP 1포인트 = 스탯 1 투자. 레벨당 SP 2개.

| 스탯 | 효과 | 1000포인트 |
|---|---|---|
| **STR** | **물리(대검) 데미지** 곱셈 ×(1 + pt×0.001), 포인트당 +0.1% | 1000pt → ×2.0 |
| **INT** | **마법 데미지** 곱셈 ×(1 + pt×0.001), 포인트당 +0.1% | 1000pt → ×2.0 |
| **DEX** | **석궁 데미지 + 전체 공속** +0.05%/pt, 크리 확률 +0.015%/pt | +50% 공속/석궁뎀, +15% 크리 |
| **VIT** | HP **+5** per pt, DEF **+0.5** per pt | +5000 HP, +500 DEF |
| **LCK** | 크리 뎀 **+0.2%** per pt, 드롭 **+0.15%** per pt | +200% 크리뎀, +150% 드롭 |

## 3. 데미지 공식

### 물리 공격 (대검 — STR 스케일)
```
최종 데미지 = meleeRef() × statStr() × 스킬배율 × 패시브배율
```

### 마법 공격 (INT 스케일)
```
최종 데미지 = meleeRef() × statInt() × pMagicMul() × DPS_BAL.magic × 스킬배율
```

### 석궁 공격 (DEX 스케일)
```
최종 데미지 = meleeRef() × statDex() × pBowMul() × DPS_BAL.bow × 스킬배율
```

### 크리티컬
```
크리뎀 = 최종데미지 × (1.3 + LCK보너스)
```
크리만 유일한 곱연산 (피해량 증폭이므로).

### 상태이상 DOT (틱당, 60f=초당1틱)
| 상태이상 | 공식 | 스탯 스케일 |
|---|---|---|
| **출혈** 🩸 | 적 최대HP × 0.7% × **statStr()** | STR (물리) |
| **독** 🟢 | 적 최대HP × 0.7% × **statDex()** | DEX (석궁/민첩) |
| **화상** 🔥 | 적 최대HP × 0.7% × **statInt()** | INT (마법) |
| **감전** ⚡ | 데미지 없음 (CC 효과만) | — |
| **빙결DOT** ❄ | 적 최대HP × 0.8% + 어픽스 보정 | 없음 (고정) |

## 6. 몬스터 스케일링

### HP 공식
```
몬스터HP = 150 × 1.012^레벨 (1.2% 복리)
보스HP = 몬스터HP × 10
```

| 레벨 | 일반몹 HP | 보스 HP |
|---|---|---|
| 1 | 152 | 1,520 |
| 10 | 171 | 1,710 |
| 100 | 550 | 5,500 |
| 300 | 7,260 | 72,600 |
| 500 | 95,850 | 958,500 |
| 900 | 16,710,000 | 167,100,000 |

### ATK 공식
```
몬스터ATK = 20 + 레벨×2
보스ATK = 40 + 레벨×3
```

| 레벨 | 일반 ATK | 보스 ATK |
|---|---|---|
| 1 | 22 | 43 |
| 100 | 220 | 340 |
| 500 | 1,020 | 1,540 |
| 900 | 1,820 | 2,740 |

## 7. 플레이어 DPS 예시

### Lv1 (무기ATK 10, STR 0, 패시브0)

```
기검참 = 10 + 0 + 20(스킬1렙) + 0 = 30/타
초당 3회 = 90 DPS
```

vs 몬스터 HP 150 → **약 2초**에 처치

### Lv100 (무기ATK 150, STR 200, 패시브3렙)

```
기검참 = 150 + 200 + 20(스킬1렙) + 60(패시브) = 430/타
초당 3회 = 1,290 DPS
```

vs 몬스터 HP 15,000 → **약 12초**에 처치 (스킬 조합으로 5~7초)

### Lv500 (무기ATK 600, STR 1000, 패시브7렙)

```
기검참 = 600 + 1000 + 200(스킬10렙) + 140(패시브) = 1,940/타
초당 3회 = 5,820 DPS
```

vs 몬스터 HP 75,000 → **약 13초**에 처치 (스킬 조합으로 6~8초)

### Lv900 (무기ATK 1000, STR 1800, 패시브10렙)

```
기검참 = 1000 + 1800 + 200(스킬10렙) + 200(패시브) = 3,200/타
초당 3회 = 9,600 DPS
```

vs 몬스터 HP 135,000 → **약 14초**에 처치 (스킬 조합으로 7~9초)

## 8. 밸런스 핵심

- **모든 레벨에서 일반몹 처치 시간 ≈ 5~10초**
- **보스 ≈ 30~60초**
- 성장은 느끼되, 적도 같이 강해져서 긴장감 유지
- 스킬 선택 = DPS 차이가 아니라 **플레이 스타일 차이**
- 스탯/패시브/강화가 곱셈 배율 → 후반 성장 체감 확보

---

## 9. DEF 공식

```
피해감소 = DEF / (DEF + 50)
최종피해 = 받는뎀 × (1 - 피해감소)
```

| DEF | 감소율 |
|---|---|
| 10 | 17% |
| 25 | 33% |
| 50 | 50% |
| 100 | 67% |
| 200 | 80% |
| 500 | 91% |

캡 없음. 하지만 수확체감(diminishing returns)이라 무한 쌓아도 100% 불가.

## 10. 적용 순서 (접목 체크리스트)
