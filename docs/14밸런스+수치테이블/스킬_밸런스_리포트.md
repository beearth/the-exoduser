## 밸런스 분석

### A. 배율 비교 (같은 투자 대비)

**1배율 = ref × stat × passiveMul 기준**

```
최고 배율 (Lv10):
  1위 chainSlam     46.5배 (거리×6 최대)  ← 조건부(사슬기동 중)
  2위 chainAssault  36.6배 (거리×4 최대)  ← 조건부(사슬기동 중)
  3위 maliceMortar  42배                  ← 11초 쿨
  4위 holyBlast     28.5배               ← 90초 쿨

  6위 fireAura      12.48배 (범위틱)      ← 12초 쿨
  7위 burstLoop     11.75배 (3단만충)     ← 차지 필요

 10위 darkPillar    9.4배 (9기둥)         ← 15초 쿨

### B. DPS 효율 (배율 ÷ 쿨타임)

```
DPS 효율 순위 (배율/초, Lv10 기준):
  1위 bladeShot     24.7 / 0.5초  = 49.4/초  ← ST 소모 큼
  2위 fireBeam      10.4 / ~1초   = ~10.4/초  ← MP 10/발, 가성비 최고
  3위 energyShot    5.7 × 4발     = 22.8/초  ← MP 20/초

  6위 kiSlash       3.6 × 3단     = ~10.8/초  ← 기본공격

### C. 밸런스 문제점

#### 1. 붉은꽃(bladeShot) 배율 이상 — 기본배율 13
- 다른 석궁이 1.2~1.8인데 **혼자 13배율**
- 0.5초 쿨에 자동발사 → 초당 49.4 DPS 효율
- **만화방창(1.8)의 7배**, needleShot(1.2)의 **11배**
- 파편 데미지까지 있어 실질 DPS 더 높음
- **평가: 명백히 OP. 배율 조정 필요**

#### 2. 폭풍소환(maliceMortar) 배율 이상 — 기본배율 15, Lv스케일 +20%
- 다른 마법 스킬이 0.8~6.0인데 **혼자 15배율 + 20%/Lv**
- Lv10에서 42배율 → 쿨 11초 = **3.8배율/초**
- 같은 쿨 계열 fireAura(6.0, 12초쿨)의 약 **3배**
- **Lv 스케일링도 20%로 최고치** (다른 마법 12~15%)
- **평가: 배율 or Lv스케일 하향 필요**

#### 3. 기동파괴(chainSlam) 거리배율 과다 — ×6 캡
- 기동불꽃이 ×4인데 기동파괴는 **×6**
- 기본배율도 3.3 × 6 = 실질 19.8 (Lv1)
- 보스 체간 삭감 + 즉기절 5초까지 붙어있음
- 같은 이동 중 스킬 chainAssault(4.4 × 4 = 17.6)�� 비교해도 높음
- **평가: 적정 수준이나 거리배율 ×5 정도가 균형적**

#### 4. 악의구(fireball) 배율 저조 — 기본배율 0.8
- 기본 E스킬인데 배율이 **가장 낮음**
- 업��선(5.0)의 **16%**, 마력연사(3.0)의 **27%**
- Lv 스케일 +15%가 높아도 Lv10에서 1.88 → 여전히 낮음
- E 마법 중 업화선(10.4)의 **18%** 수준
- **평가: 의도적 약세? 범위폭발+넉백 유틸리티 고려 시 납득 가능하나, 기본 스킬치고 너무 약함**

#### 5. 멸살광선(omniBeam) 틱당 0.15 — 단독 시 약함
- 채널링이라 연속 히트지만, 틱당 0.15는 낮음
- **소비량 보너스(drainBonus)가 핵심** — MP 풀 클수록 뎀 증가
- 합체(만화광선, 추적암전) 시 대폭 강화
- 단독 사용 시 DPS 부족
- **평가: 합체 전제 설계로 단독은 약한 게 의도적**

#### 6. 칼등날개(shieldThrow) 배율 극저 — 0.4
- STR × 0.8 × 0.5 = **0.4배율**
- 쿨 5초에 비용도 ST 30~50
- 다른 물리 스킬(kiSlash 1.2~2.0)의 **20~33%**
- **평가: 합체 보너스 효과(돌진+충격파)가 주 목적이라면 OK, 아니면 상향 필요**

#### 7. 업화선(fireBeam) 가성비 최고
- 배율 5.0, MP 10, 쿨 DEX 스케일(약 1초)
- 100% 관통 + 유도 + 화상
- 초당 ~10.4 DPS에 MP 10/초 = **1MP당 1.04 DPS**
- 마력연사(3.0 × 4 = 12/초, MP 20/초) = 1MP당 0.6 DPS
- **평가: 약간 효율 높지만 단일타깃이라 적정**

#### 8. 궁극기 간 밸런스
- **신성폭발**: 28.5배 + 넉백+스턴 2초 → 범용 최강
- **용암소환**: 4배 초기 + HP% 틱 20회 → 보스 특화 (HP 높을수록 유리)
- **블랙**: 데미지 0 (흡인) → 순수 유틸
- **처형**: HP 30% 순수뎀 → 그로기 보스 전용
- **평가: 역할 분담 잘 됨. 신성폭발이 범용성 최고이나, 용암의 보스 특화가 보완**
