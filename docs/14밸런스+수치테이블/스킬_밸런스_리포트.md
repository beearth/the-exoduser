### A. 배율 비교 (같은 투자 대비)

**1배율 = ref × stat × passiveMul 기준**

  1위 chainAssault  36.6배 (거리×4 최대)  ← 조건부(사슬기동 중)
  2위 holyBlast     28.5배               ← 90초 쿨
  3위 chainSlam     27.2배 (거리×3.5 최대) ← 조건부(사슬기동 중)

### C. 밸런스 문제점

#### 1. 붉은꽃(bladeShot) 배율 조정 완료 — 기본배율 13→6
- 석궁 평균(1.5)의 4배로 조정 (기존 8.7배)
- 0.5초 쿨에 자동발사 → 초당 22.8 DPS 효율 (기존 49.4)
- ST도 1+Lv÷2로 하향 (1렙1, 10렙6)
- **평가: 조정 완료. 파편 포함 시 여전히 강하나 적정 수준**

#### 2. 폭풍소환(maliceMortar) 배율 조정 완료 — 기본배율 15→10, Lv스케일 +20%→+15%
- 마법 평균(4)의 2.5배로 조정 (기존 3.75배)
- Lv10에서 23.5배율 → 쿨 11초 = **2.14배율/초** (기존 3.8)
- 같은 쿨 계열 fireAura(6.0, 12초쿨)의 약 **2배** (기존 3배)
- **Lv 스케일링도 15%로 하향** (마법 평균과 동일)
- **평가: 조정 완료. 범위 스킬로서 적정 수준**

#### 3. 기동파괴(chainSlam) 거리배율 조정 완료 — ×6→×3.5 캡 (distMul max 3→2.5)
- 기동불꽃(×4)보다 낮게 조정
- 기본배율 3.3 × 3.5 = 실질 11.55 (Lv1), 기존 19.8
- 보스 체간 삭감 + 즉기절 5초 유지
- Lv10: 7.76 x 3.5 = 27.2 (기존 46.5)
- **평가: 조정 완료. chainAssault(36.6)보다 낮아 균형적**

#### 4. 악의구(fireball) 배율 상향 완료 — 기본배율 0.8→1.3
- 기본 E스킬 배율 상향으로 체감 개선
- 업화선(5.0)의 26%, 마력연사(3.0)의 43%
- Lv10에서 3.055 → 초당 ~4.4 DPS
- 범위폭발+넉백 유틸리티 포함 시 적정
- **평가: 조정 완료. 기본 E스킬로서 적정 수준**

#### 5. 멸살광선(omniBeam) 틱당 0.15→0.22 — 단독 체감 개선
- 채널링 틱 0.22로 상향 (기존 0.15)
- 채널링 고정값/초로 전환: 고급 70 MP/초 (MP1000=14초, MP2000=29초)
- **소비량 보너스(drainBonus)가 핵심** — MP 풀 클수록 뎀 증가
- 합체(만화광선, 추적암전) 시 대폭 강화
- **평가: 틱뎀 상향+지속시간 증가로 단독 운용성 개선**

#### 6. 칼등날개(shieldThrow) 배율 상향 완료 — 0.4→0.6
- STR × 1.2 × 0.5 = **0.6배율** (기존 0.8×0.5=0.4)
- 쿨 5초에 비용 ST 30~50
- Lv10: 1.03배율 (기존 0.69)
- **평가: 조정 완료. 합체 보너스+본체 뎀 모두 개선**

#### 7. 업화선(fireBeam) 가성비 최고
- 배율 5.0, MP 10, 쿨 DEX 스케일(약 1초)
- 100% 관통 + 유도 + 화상
- 초당 ~10.4 DPS에 MP 10/초 = **1MP당 1.04 DPS**
- 마력연사(3.0 × 4 = 12/초, MP 20/초) = 1MP당 0.6 DPS
- **평가: 약간 효율 높지만 단일타깃이라 적정**

#### 8. 궁극기 간 밸런스
- **신성폭발**: 28.5배 + 넉백+스턴 2초 → 범용 최강
- **용암소환**: 4배 초기 + HP% 틱 20회 → 보스 특화 (HP 높을수록 유리)
- **블랙**: 데미지 0 (흡인) → 순수 유틸
- **처형**: HP 30% 순수뎀 → 그로기 보스 전용
- **평가: 역할 분담 잘 됨. 신성폭발이 범용성 최고이나, 용암의 보스 특화가 보완**
