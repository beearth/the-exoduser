# 2. 게임디자인 / 레벨디자인

## 콤보 시스템

| 항목 | 값 |
|---|---|
| 타이머 | 30초 (1800틱) |
| 리셋 조건 | 30초간 킬 없으면 콤보 0으로 리셋 |
| 킬 시 | `G.combo++`, 타이머 1800으로 리셋 |
| 최대 기록 | `G.comboMax` (세션 내 최고 콤보) |
| HUD | 우측 상단, 2콤보 이상 시 표시 |
| 표시 색상 | 2~9: 노란, 10~19: 주황, 20~49: 진주황, 50+: 빨강 |
| 마일스톤 텍스트 | 5콤보마다 `🔥 N COMBO!` 팝업 |
| 종료 텍스트 | 10콤보 이상에서 종료 시 `콤보 종료: N` 팝업 |

## 예정 항목
- 레벨 디자인 원칙
- 난이도 커브
- 진행 구조 (스테이지별 밀도/템포)
- 전투 밸런스 공식
