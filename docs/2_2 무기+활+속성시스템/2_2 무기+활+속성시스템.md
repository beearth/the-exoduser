# 2-2. 무기 + 활 + 속성 시스템

## 무기 7종 (WTYPES)
| 키 | 이름 | 특징 |
|---|---|---|
| sword | 검 | 기본, 넓은 호 스윙 |
| dagger | 단검 | 빠른 찌르기 |
| hammer | 해머 | 내려치기, 충격파 |
| mace | 메이스 | 사선 휘두르기 |
| club | 곤봉 | — |
| axe | 도끼 | 대각선 내려치기 |
| spear | 창 | 빠른 찌르기, 돌진 관통 |

## 활 3종 (BOWTYPES) — 🔲 미구현 (활 캐릭터 추가 시 구현 예정)
| 키 | 이름 | 특징 |
|---|---|---|
| crossbow | 석궁 | 중간 속도 |
| repeater | 연사석궁 | 빠른 연사 |
| arcane | 마력석궁 | 마법 투사체 |

> 현재 석궁은 자동발사(T키) 보조무기로만 사용. 활 전용 캐릭터/빌드 추가 시 본격 구현 예정.

## 속성 7종 (EL) + 상성
| 속성 | 키 | 상성 강점 | 상성 약점 |
|---|---|---|---|
| 물리 | P | — | — |
| 화염 | F | 빙결에 강 | 암전에 약 |
| 빙결 | I | 암전에 강 | 화염에 약 |
| 암흑 | D | 신성에 강 | 신성에 약 |
| 암전 | L | 화염에 강 | 빙결에 약, 대지에 약 |
| 신성 | H | 암흑에 강 | 암흑에 약 |
| 대지 | E | 암전에 강 | 암전에 약 |

## 수정 금지
- WTYPES, BOWTYPES, EL 객체 구조
- `ELC[]` (속성 색상), `ETYPE_COL[]` (적 타입 색상)
