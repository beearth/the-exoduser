|---|---|---|
| sword | 검 | 기본, 넓은 호 스윙 |

## 활 3종 (BOWTYPES) — 🔲 미구현 (활 캐릭터 추가 시 구현 예정)
| 키 | 이름 | 특징 |
|---|---|---|
| crossbow | 석궁 | range:800, atkMul:1.0, spdMul:1.0 |

## 속성 7종 (EL) + 상성
| 속성 | 키 | 상성 강점 | 상성 약점 |
|---|---|---|---|
| 물리 | P | — | — |
| 화염 | F | 빙결에 강 | 암전에 약 |
| 빙결 | I | 암전에 강 | 화염에 약 |
| 암흑 | D | 신성에 강 | 신성에 약 |
| 암전 | L | 화염에 강 | 빙결에 약, 대지에 약 |
| 신성 | H | 암흑에 강 | 암흑에 약 |
| 대지 | E | 암전에 강 | 암전에 약 |

## 속성별 상태이상

### 감전 (Shock) — 번개 속성 (EL.L) (2026-05-01)
PoE2 스타일. 번개 속성 피격 시 자동 부여.

| 항목 | 값 |
|---|---|
| 트리거 | 번개(EL.L) 속성 비DOT 피격 |
| 지속시간 | 5초 (300f), 피격마다 갱신 |
| 최대 스택 | 10 |
| 받는 뎀 증가 | 스택당 +5% (최대 +50%) |
| 마비(스턴) | 3스택 이상: 0.3초(18f) 경직, 보스 제외 |
| 타이머 | `e._shockT` (만료 시 스택 리셋) |
| HUD 표시 | 보스바 상태이상 `⚡ 감전N Xs` |
| 적용 위치 | `hurtE()` 내부, 뎀 증가는 STR 적용 전 |

### 기존 상태이상 (참고)
- **출혈**: 물리/출혈 어픽스, D4식 풀 축적, 5초
- **중독**: 독 어픽스, D4식 풀 축적, 5초
- **화상**: 화염 속성, D4식 풀 축적, 3초
- **저주**: 암흑 어픽스(`darkCurse`), 받는 뎀 +25%, 3초

- `ELC[]` (속성 색상), `ETYPE_COL[]` (적 타입 색상)
