## 속성 시너지 — 전류강화 (물 × 전기) (구현 완료, 2026-05-01)

뇌전창(thunderStake) 전기 아크선이 **물 속성 존**(iceStorm / holyDome)과 겹칠 때 자동 발동.
합체(fuse)가 아닌 **선분-원 교차 판정**으로 자동 발동되며, 스킬 습득만 하면 조건 충족.

### 발동 조건
| 조건 | 설명 |
|---|---|
| 뇌전창 아크 | thunderStake 2개 이상 설치, 1000px 이내 연결 |
| 물 속성 존 | iceStorm 또는 holyDome 타입 fireZone 활성 |
| 교차 판정 | 아크 **선분** 위 최근접점이 존 원 내부 (선분-원 교차) |
| 조합 예시 | 물의 영역(holyIce 3성) + 뇌전창 → 자동 발동 |

### 효과 1: 정전기 흡인 (Electrostatic Pull)
| 항목 | 값 |
|---|---|
| 흡인 대상 | 적 (alive, 30px 이상 거리) |
| 흡인 목표 | 아크 **선분 위 최근접점** (중점 아님, 가장 가까운 아크 지점) |
| 흡인 범위 | 강화 아크 연결 수에 비례 |
| 흡인 속도 | `5*(1 - 거리/범위*0.3)` px/frame (블랙급, 매 프레임) |
| 벽 체크 | `isW()` — 벽이면 끌리지 않음 |

#### 연결 수별 흡인 범위
| 강화 아크 연결 수 | 흡인 범위 | 비고 |
|---|---|---|
| 1연결 | 350px | 뇌전창 2개, 1쌍 아크가 존 교차 |
| 2연결 | 500px | 3개 이상 창, 해당 창에 2개 강화 아크 연결 |
| 3연결 | 650px | 4개 이상 창, 3개 강화 아크 연결 |
| N연결 | 350+(N-1)×150px | +150px/연결 |

- **연결 수 계산**: 각 창이 가진 강화 아크(존 교차) 개수를 세고, 쌍의 양쪽 중 최대값 사용
- **`_tsIceLinks[i]`**: 창 i의 강화 아크 연결 수 (Uint8Array)

### 효과 2: 아크 데미지 강화
| 항목 | 값 |
|---|---|
| 아크 틱 데미지 | 기본 `(창A뎀+창B뎀)×0.15` → **×2** (100% 증가) |
| 적용 코드 | `_arcMul=_arcInIce?2:1` → `hurtE(e, _arcDmg*_arcMul, ...)` |
| 틱 간격 | 20프레임 (0.33초) |
| 존 밖 아크 | 기본 데미지 유지 (`_arcMul=1`) |

### 효과 3: 아크 근접 감전 틱
| 항목 | 값 |
|---|---|
| 판정 범위 | 아크 선분 최근접점 35+e.r px 이내 |
| 데미지 | `(창A뎀+창B뎀)×0.05` per frame |
| 속성 | EL.L (전기) |
| 넉백 | 없음 (`noKb:true`) |
| 실행 빈도 | 매 프레임 (흡인 루프 내) |
| 용도 | 흡인으로 아크에 달라붙은 적 지속 데미지 |

### 효과 4: 비주얼 피드백
| 항목 | 기본 아크 | 강화 아크 |
|---|---|---|
| 메인 색상 | `#44ddff` | `#ff6644` (붉은색) |
| 서브 색상 | `#aaeeff` | `#ffaa88` |
| 선 두께 | 3px / 1px | 4px / 2px |
| 알파 | 0.35 | 0.50 |
| 파티클 | `#ffffff` | `#ffccaa` |
| 판정 | 렌더러 내 선분-원 교차 재판정 (프레임마다) |

### 코드 위치
| 위치 | 설명 |
|---|---|
| game.html ~21310 | 뇌전창 수명 + 아크 데미지 + 전류강화 ×2 (히트스톱 밖) |
| game.html ~21355 | 전류강화 흡인 매 프레임 루프 (연결 수 계산 → 흡인) |
| game.html ~40380 | 아크 렌더 — 강화 아크 붉은색 표시 |

### 밸런스 메모
- 물의 영역은 holyFuse(2성) → holyIce(3성) 선행 필요 → 고성급 시너지
- 뇌전창은 1성 스킬, 전격의창(thunderGhost 2성) 합체 시 전격이동 착지 자동 배치로 편의성↑
- 흡인(블랙급) + 감전 틱 + 데미지 ×2 → 존 안에 몰아넣고 아크로 녹이는 콤보
- 흡인 속도 5px/f, 범위 350~650+px, 데미지 ×2 — 실 테스트 후 조정
