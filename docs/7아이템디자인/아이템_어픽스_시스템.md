### 조건부 어픽스 합연산 규칙

**어픽스만** 같은 카테고리 조건부 뎀업은 **전부 합연산** 후 1회 곱연산:
```
dmg *= (1 + closeDmg + stunDmgUp + bossSlayer + abundDmg + ...)
```

**패시브와 어픽스 적용 순서 (2026-04-18 변경):**
1. 기본 데미지 계산: `dmg = ref × stat × _passDmgSum(type)` (패시브는 이미 합연산)
2. 약자포식 추가: `dmg *= (1 + _predBonus(e))` (패시브, 적HP≤50%)
3. 어픽스 적용: `dmg *= (1 + 조건부어픽스 합)` (abundDmg/predDmg 포함, 별도 곱연산)

### 티어 결정 규칙
- baseTier: 전설=2, 영웅=1, 나머지=0
- lvBonus: `floor(PLv / 200)` (200렙마다 +1)
- maxTier: `min(어픽스별 티어수-1, baseTier + lvBonus)`
- tierW 가중치가 있으면 가중 랜덤, 없으면 lo~hi 연속 롤
- skewRoll이 있으면 `Math.random()^skewRoll` (높을수록 하위 편향)
- 같은 group 중복 불가
