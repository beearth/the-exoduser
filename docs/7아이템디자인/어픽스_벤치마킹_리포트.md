# 어픽스 시스템 벤치마킹 — PoE2 / D4 / Torchlight / Last Epoch vs EXODUSER

> 2026-04-15 기준

---

## 1. 경쟁작 어픽스 구조 비교

| 항목 | PoE2 | Diablo 4 | Torchlight Infinite | Last Epoch |
|---|---|---|---|---|
| 프리/서픽 구분 | O (3+3) | X (구분 없음) | O (3+3) | O (4+4) |
| 아이템당 최대 | 6개 | 3~4 + Aspect 1 | 6개 | 8개 |
| 티어 수 | 12~13 (T1=최고) | 연속 수치 범위 | 7 (T7=최고) | 7 (T7=최고) |
| 총 어픽스 수 | ~1,500~2,000 | ~400~500 | ~600~800 | ~300~400 |
| 조건부 어픽스 | 매우 다양 | 체계적 (4축) | 보통 | 적음 |

### PoE2 — 가장 깊은 어픽스 시스템
- 프리 3 + 서픽 3, 영향력(Influence) 모드로 전용 풀 추가
- 조건부: Full Life, Low Life, On Kill, On Hit, Nearby Enemies, Culling Strike(적HP 10%↓ 즉사)
- 크래프팅 커런시 20종+ → 어픽스 잠금/재굴림/추가/제거
- **시사점:** 티어 깊이(12단계)가 파밍 동기 유지의 핵심

### Diablo 4 — 가장 직관적인 조건 분류
- 프리/서픽 구분 없음, 단순 슬롯 3~4개
- **Greater Affix:** 일반의 1.5배 값 (2.5% 확률, 금테) → 엔드게임 목표
- 조건부 4축 체계:
  - **While Healthy** (HP≥80%)
  - **While Injured** (HP≤35%)
  - **Close Enemies** (~10m)
  - **Distant Enemies** (>10m)
  - While Fortified, While Berserking 등 버프 조건
- **Tempering:** 카테고리 지정 어픽스 1~2개 추가 (횟수 제한)
- **Masterworking:** 12단계 강화, 4/8/12에서 랜덤 1어픽스 +25%
- **시사점:** 조건부 어픽스가 빌드 다양성의 핵심 축

### Torchlight Infinite — 영웅 특성으로 차별화
- PoE식 프리3+서픽3, T1~T7
- 조건부: 최근 N초 적 처치, 이동 중, 보스 대상
- **영웅 특성(Hero Trait):** 캐릭터 고유 모디파이어 트리 (장비 외 별도 성장축)
- **시사점:** 장비 어픽스 + 캐릭터 고유 시스템 이중 레이어

### Last Epoch — 크래프팅 리스크 관리
- 프리4+서픽4 (가장 관대), T1~T7
- **단조 잠재력(Forging Potential):** 크래프팅 "수명" → 0되면 수정 불가
- **Legendary Potential:** 유니크에 1~4LP → Exalted와 합체
- **시사점:** 크래프팅 자체가 콘텐츠 (리스크/리워드 긴장감)

## 2. EXODUSER 현재 상태

| 항목 | EXODUSER |
|---|---|
| 프리/서픽 | O (균형 51:49) |
| 총 어픽스 | 69개 엔트리 (67 고유 ID) |
| 티어 | 3-티어(74%) + 5-티어(17%) |
| 아이템당 | 프리1+서픽1 (최대 2개) |
| 슬롯 분류 | 3파트 (무기/방어구/장신구) |
| 조건부 | 9종 |
| Implicit | 슬롯별 1개 고정 |

### 조건부 어픽스 현황 (9종)

| 어픽스 | 조건 | 효과 |
|---|---|---|
| abundDmg | 내HP≥70% | 뎀+ |
| predDmg | 적HP≤50% | 뎀+ |
| crisisBoost | 내HP≤30% | 뎀+ |
| counterHit | 피격 시 | 확률 반격 |
| elemConvert | 물리 공격 시 | 속성 전환 |
| parryExplosion | 패링 성공 시 | AoE |
| staggerExplosion | 그로기 시 | 범위폭발 |
| lastStand | 사망 시 | 1회 생존 |
| reviveOnce | 사망 시 | 1회 부활 |

### 강점
- 3파트 슬롯 분리가 명확 (무기=공격, 방어구=생존, 장신구=저항/자원)
- weight+tierW+skewRoll 3중 희귀도 시스템
- 팔찌 brType(악마/생명) 특화 선택
- Implicit으로 슬롯 정체성 강화

### 약점
- 아이템당 어픽스 2개(프리1+서픽1)로 빌드 표현력 제한
- 슬롯별 어픽스 수 불균형 (무기 26 vs 글러브 4)
- 조건부 9종은 D4(10종+)/PoE2(20종+) 대비 부족
- 티어 깊이 3~5단계 → PoE2(12단계) 대비 파밍 목표 부족

## 3. 벤치마킹 개선 제안

### A. 조건부 어픽스 확충 (D4식 4축 참고)

현재 HP 기반 3종(abundant/predator/crisis)만 있음. D4식 확장:

| 축 | 현재 | 추가 제안 |
|---|---|---|
| **HP 상태** | ≥70%, ≤50%, ≤30% | ✅ 충분 |
| **거리** | 없음 | `closeDmg` (근접적 뎀+), `farDmg` (원거리적 뎀+) |
| **상태 조건** | 패링/그로기 | `onDodge` (회피 시), `onCrit` (크리 시 버프) |
| **환경** | 없음 | `bossSlayer` (보스 대상 뎀+), `mobSlayer` (일반몹 뎀+) |

### B. 슬롯 어픽스 수 균형화

| 슬롯 | 현재 | 제안 |
|---|---|---|
| 글러브 | 4종 | +2~3종 (공격계: 공속, 크리, 스킬쿨) |
| 부츠 | 7종 | 적정 |
| 투구 | 10종 (무기 공유) | 적정 |
| 갑옷 | 12종 | 가장 풍부, 적정 |

### C. Greater Affix (D4 참고)

- 드롭 시 2~3% 확률로 **강화 어픽스** (수치 1.5배, 특수 테두리)
- 엔드게임 파밍 목표로 작용
- 구현 난이도 낮음: 롤 시 `isGreater` 플래그 + 값 ×1.5

### D. 티어 확장 (장기)

- 현재 3-티어 → 최소 5-티어 통일 (이미 일부 5-티어 존재)
- PoE2처럼 12단계까지는 과하나, 7-티어(Last Epoch식)가 적절
- 아이템 레벨 / 스테이지에 따라 상위 티어 해금

## 4. 설계 철학 포지셔닝

```
복잡도   PoE2 ━━━━━━━━ TL:I ━━━ D4 ━━━ LE ━━━ EXODUSER
         (최고)                                    (간결)
         
조건부   PoE2 ━━━━ D4 ━━━━━━ TL:I ━━━ EXODUSER ━━━ LE
         (최다)                                      (최소)

파밍깊이  PoE2 ━━━━━━━ D4 ━━━━ TL:I ━━━ LE ━━━ EXODUSER
         (티어12)    (Greater)              (3~5티어)
```

**EXODUSER 포지션:** D4~LE 사이의 "간결하지만 의미있는" 어픽스 시스템.
PoE2급 복잡도는 오버킬, D4식 직관적 조건부 + Greater Affix가 가장 ROI 높은 개선 방향.
