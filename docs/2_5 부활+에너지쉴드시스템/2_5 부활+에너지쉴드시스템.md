# 2-5. 부활 + 에너지 쉴드 시스템

## 부활 시스템

### 보스 (e.ib) — 확률+포인트 병행 시스템
- 본모습으로 풀피(100% HP) 부활 (1차 즉시 부활)
- **확률+포인트 병행**: 포인트 고갈 OR 확률 실패 → 둘 중 하나만 충족되면 최종 사망
- **부활력 포인트** (`_revPts`): 초기값 `150 + round(si × 4.412)` (si0=150, si34=300)
  - 1차 즉시 부활 소모: 기본 10 + antiRevive(+2~5) + holyPrison존(+3) + 완벽처치(+2) = 최대 ~15
  - 2차 폴백 부활 소모: 기본 10 고정
- **확률 공식**: `max(0, 150% + 스테이지×5% - (부활횟수-1)×15% - 신성력)`
- **2차 폴백 (3초 카운트다운)**: `max(0, 150% + 스테이지×5% - 부활횟수×25% - 신성력)`, 포인트 0이면 자동 실패, 성공 시 HP 50%/eShield 50% 부활
- 신성력(antiRevive) + 구속의 영역(15%~30%)으로 억제 가능
- **무기 3종 기본 신성력**: weapon/shield/bow에 antiRevive 티어별 밴드 롤 (T0=0~3%, T1=3~6%, T2=6~9%, T3=9~12%, T4=12~15%). 3개 T4 맥스 합산 = 45%
- 부활마다: ATK ×1.5, SPD ×1.25, 포이즈 ×1.5, 패턴쿨 -30f
- 부활력 UI: 플로팅 텍스트로 소모량/잔여/최대 표시, 고갈 시 빨간색 경고

### 일반몹 (!e.ib)
- 1회만 부활, **구울로 변환** (본모습 아님)
- 부활력: 기본 50% + 100Lv당 +10% (엘리트2+ 기본 60%)
  - Lv100: 60%, Lv500: 100%, Lv1000: 150%(확정+2회부활 가능성)
- 구울 3종: 소형(r≤10), 중형(11~18), 대형(r≥19)
- **벽 끼임 구출 실패**: 플레이어 위치로 강제 이동 (kb 초기화)

### 신성력 (억제 + 언데드 데미지 보너스)
- 목걸이: T1~T3 10~30%
- 무기: 10~30%
- 반지: 0~20%
- 합산 최대 100%
- 구속의 영역 fireZone: 15%~30% (Lv1=15%, Lv10=30%)
- **언데드 데미지 보너스** (2026-04-11 신규): 신성력 1%당 언데드 적에게 데미지 +1% (`hurtE`에서 `dmg×(1+_eqAffix('antiRevive'))` 적용). 대상 etype: `_UNDEAD_ET`(4,10,12,14,23,40,47,82,83,85) + `e._isGhoul`. 보스 제외

### 악마화 부활 (플레이어) — 🔲 부분 구현

#### 핵심 컨셉
- 레벨이 오를수록 악마의 힘이 깨어남
- 사망 시 "악마화 vs 인간 유지" 선택지 (영구 분기, 1회 선택, 되돌릴 수 없음)

#### 부활 규칙 — 악마화 부활 패시브 (`demonRevive`)
- **방어 탭 패시브 스킬** — 부활 강화 (버튼 없음, 자동 발동)
- 미습득이어도 자연부활력+장비로 부활 가능 (스킬은 강화용)

#### 부활력 공식
`부활력% = min(100, max(0, 자연부활력 + 스킬Lv×2 + 장비보너스 - 인간성Lv×10))`

| 요소 | 부활력 | 비고 |
|---|---|---|
| 자연부활력 | `floor(P.lv×0.03)` | Lv100=3%, Lv500=15%, Lv1000=30% |
| 스킬 (`demonRevive`) | Lv당 부활확률+2%, HP+5%, 쿨-30초 | 부활 강화 스킬 (미습득이어도 부활 가능) |
| 장비 어픽스 | revivePow + _iRevPow | 악마의 팔찌 등 |
| 부활의 영역 | +10% + Lv당 1% | 스킬 영역 안에서 |
| **인간성 (`humanity`)** | **Lv당 부활력 -10** | **부활력 감소 패시브 (트레이드오프)** |

#### 쿨다운
`10분(36000f) - Lv당 30초(1800f). 최소 100초(6000f)`

| Lv | 쿨다운 |
|---|---|
| 1 | 9분 30초 |
| 5 | 7분 30초 |
| 10 | 5분 |
| 17+ | 1분 40초 (최소) |

- HP/MP/ST 풀 회복, 2초 무적
- 습득비: SP 15, 악의 150. 강화비: SP 10/Lv

#### 현재 구현 상태
- ✅ 쓰러짐 연출: 사망 시 'fallen' 상태 진입 → 5초(300f) 카운트다운 → 부활 판정 + 쓰러짐 회전/페이드 연출 + fallen/dead 중 모든 행동 차단
- ✅ 쿨다운제 (10분, P._demonRevCd)
- ✅ 카운트다운 중 무적, 피 파티클, 비네팅, 부활력% 표시
- 🔲 선택지 UI 미구현
- 🔲 인간 HP+30% 미구현
- 🔲 세이브 영구 저장 미연동

### 인간성 패시브 (`humanity`) — ✅ 구현

#### 핵심 컨셉
- 악마화를 거부하고 인간의 의지로 강해지는 트레이드오프 패시브
- 부활력을 포기하는 대신 전투력+생존력 강화

#### 효과 (Lv당)
| 효과 | 수치 | 비고 |
|---|---|---|
| 부활력 | **-10/Lv** | 악마화 부활 확률 감소 |
| 전체 데미지 | **+5%/Lv** | _passDmgSum 합연산 |
| HP | **+100/Lv** | mhp에 가산 |
| MP | **+100/Lv** | mmp에 가산 |
| ST | **+100/Lv** | mst에 가산 |

#### 습득/강화 비용
- 습득: SP 10, 악의 100
- 강화: SP 5/Lv

#### 카테고리
- 방어 탭 (`cat:'def'`), 패시브 (액티브 발동 없음)

#### 향후 확장 — 부활 패시브 스킬화
- **방어 탭에 부활 패시브 배치** — 부활을 스킬로 습득/강화하는 구조
- 부활확률 = 부활력(revivePow)으로 환산
  - 기본 부활력: 스킬 레벨 기반
  - 추가 부활력: 장비 어픽스(악마의 팔찌 등) + 부활의 영역 스킬
- 쿨다운: 스킬 레벨로 감소 가능 (기본 10분)
- 악마화: 외형 변화, 악마 전용 스킬 트리
- 인간: 신성력 스킬 트리 해금
- 엔딩 분기: 악마 vs 인간

## 에너지 쉴드 (2026-04-17 레벨 성장 변경, bonusShield 슬롯 분리)
- **Lv1부터 기본 제공** (공식: `100 + INT×5 + bonusShield + shieldFlat어픽스`)
  - **NO Lv×1 직접 보너스** — INT 스탯만으로 증가
  - INT = STATS.int + eqInt + floor(Lv×0.5)로 자동 성장
  - **bonusShield**: 장신구 6슬롯 (belt, necklace, ring1, ring2, bracelet, headband)에서만 제공 = `tier*100 + random(20~100)` [T0: 20~100, T1: 120~200, T2: 220~300, T3: 320~400, T4: 420~500]
  - **shieldFlat 어픽스**: 장신구 슬롯 전용
- pDef/eDef 이원 방어: 물리→쉴드10%/HP100%, 마법→쉴드100%/HP10% (기본 90% 패널티)
- 패시브 물리관통(pPhysPen): +5%/lv → 물리→쉴드 10%→60%
- 패시브 마법관통(pMagPen): +5%/lv → 마법→HP 10%→60%

### 쉴드 자동 리젠
- 피격 시 3초(180f) 쿨다운 (`shieldRegenT=180`), 쿨다운 소진 후 리젠 시작
- **레벨업 시 쉴드 회복 없음** (HP/ST/MP만 20% 회복)
- 쿨다운 감소 속도: `pShieldCdMul()` = `1 - min(0.3, pMRegen×0.03)` (마력회복 패시브 -3%/lv, 최대 -30%)
- 리젠량/f: `(0.3 + INT×0.003) × pMRegenMul()`
- `pMRegenMul()` = `1 + pMRegen×0.03` (+3%/lv, MP/쉴드 리젠 배율)
