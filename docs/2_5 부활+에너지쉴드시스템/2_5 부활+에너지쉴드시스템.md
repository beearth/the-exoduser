# 2-5. 부활 + 에너지 쉴드 시스템

## 부활 시스템

### 보스 (e.ib)
- 본모습으로 부활 (HP 2배, 강화)
- **부활력 공식**: `100% + 장×10% - 사망횟수×10% - 신성력`
  - 1장: 100%, 2장: 110%, ... 7장: 160%
  - 사망 1회: -10%, 2회: -20%, ...
- 1~7단 보장부활 (메인) + 이후 3초 카운트다운 → 확률 판정 (2차)
- 3초 카운트다운 중: 숫자 표시 + 부활력% + 게이지바
- 신성력(antiRevive) + 신성한 감옥(30%+Lv당2%)으로 억제 가능
- TODO: 보스 부활/사망 확정 전용 SFX

### 일반몹 (!e.ib)
- 1회만 부활, **구울로 변환** (본모습 아님)
- 부활력 0.5 (엘리트 0.5~0.6)
- 구울 3종: 소형(r≤10), 중형(11~18), 대형(r≥19)

### 신성력 (억제)
- 목걸이: T1~T3 10~30%
- 무기: 10~30%
- 반지: 0~20%
- 합산 최대 100%
- 신성한 감옥 fireZone: 30% + Lv당 2% (Lv1=30%, Lv10=48%)

### 악마화 부활 (플레이어) — 🔲 부분 구현

#### 핵심 컨셉
- 레벨이 오를수록 악마의 힘이 깨어남
- 사망 시 "악마화 vs 인간 유지" 선택지 (영구 분기, 1회 선택, 되돌릴 수 없음)

#### 부활력 공식
- `자연부활력 = min(100, floor(Lv / 100) * 5)`
- Lv100=5%, Lv500=25%, Lv1000=50%, Lv2000=100%(캡)
- 부족분은 아이템 + 스킬로 보충:
  - **악마의 팔찌** 암묵 `_iRevPow` (5~12%) + 어픽스 `revivePow` (10~28%) + `reviveCd` (쿨감)
  - **부활의 영역** (weakRev) 스킬: +10% (+1%/Lv, 최대 19%)
- 생명의 팔찌는 부활력 대신 HP/리젠/회복력 보너스 (생존 특화)
- 악마화 선택 시에만 부활 확률로 작동

#### 선택지 (첫 사망, Lv100+)
| 선택 | 효과 |
|---|---|
| 악마화 | 사망 시 부활력% 확률 즉시 부활(풀HP), 게임당 1회 |
| 인간 유지 | 부활 불가, 최대 HP +30% 영구 |

#### 현재 구현 상태
- ✅ 부활력 공식 적용 (`자연 5%/100Lv + 팔찌 revivePow 어픽스 + _iRevPow 임플리싯 + 부활의 영역 스킬`)
- ✅ 쓰러짐 연출: 사망 시 'fallen' 상태 진입 → 1.5초(90f) 카운트다운 → 부활 판정 + 쓰러짐 회전/페이드 연출
- ✅ 쿨다운제 (기본 3분, 팔찌 reviveCd로 감소, P._demonRevCd)
- ✅ 카운트다운 중 무적, 피 파티클, 비네팅, 부활력% 표시
- 🔲 선택지 UI 미구현
- 🔲 인간 HP+30% 미구현
- 🔲 세이브 영구 저장 미연동

#### 향후 확장
- 악마화: 외형 변화, 악마 전용 스킬 트리
- 인간: 신성력 스킬 트리 해금
- 엔딩 분기: 악마 vs 인간
