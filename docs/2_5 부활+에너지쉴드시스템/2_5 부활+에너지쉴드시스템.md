# 2-5. 부활 + 에너지 쉴드 시스템

## 부활 시스템

### 보스 (e.ib)
- 본모습으로 부활 (HP 2배, 강화)
- **부활력 공식**: `125% + 스테이지×5% - 부활횟수×25% - 신성력`
  - 스테이지0: 125%, 스테이지10: 175%, 스테이지34: 295%
  - 부활 1회: -25%, 2회: -50%, 5회: -125%
- 1~7단 보장부활 (메인) + 이후 3초 카운트다운 → 확률 판정 (2차)
- 3초 카운트다운 중: 숫자 표시 + 부활력% + 게이지바
- 신성력(antiRevive) + 신성한 감옥(30%+Lv당2%)으로 억제 가능
- TODO: 보스 부활/사망 확정 전용 SFX

### 일반몹 (!e.ib)
- 1회만 부활, **구울로 변환** (본모습 아님)
- 부활력 0.5 (엘리트 0.5~0.6)
- 구울 3종: 소형(r≤10), 중형(11~18), 대형(r≥19)

### 신성력 (억제)
- 목걸이: T1~T3 10~30%
- 무기: 10~30%
- 반지: 0~20%
- 합산 최대 100%
- 신성한 감옥 fireZone: 30% + Lv당 2% (Lv1=30%, Lv10=48%)

### 악마화 부활 (플레이어) — 🔲 부분 구현

#### 핵심 컨셉
- 레벨이 오를수록 악마의 힘이 깨어남
- 사망 시 "악마화 vs 인간 유지" 선택지 (영구 분기, 1회 선택, 되돌릴 수 없음)

#### 부활 규칙 — 악마화 부활 패시브 (`demonRevive`)
- **필살기 탭 패시브 스킬** — 부활 강화 (버튼 없음, 자동 발동)
- 미습득이어도 자연부활력+장비로 부활 가능 (스킬은 강화용)

#### 부활력 공식
`부활력% = min(100, 자연부활력 + 스킬Lv×2 + 장비보너스)`

| 요소 | 부활력 | 비고 |
|---|---|---|
| 자연부활력 | `floor(P.lv/100)×5` | Lv100=5%, Lv500=25%, Lv1000=50%, Lv2000=100% |
| 스킬 (`demonRevive`) | Lv당 부활확률+2%, HP+5%, 쿨-30초 | 부활 강화 스킬 (미습득이어도 부활 가능) |
| 장비 어픽스 | revivePow + _iRevPow | 악마의 팔찌 등 |
| 부활의 영역 | +10% + Lv당 1% | 스킬 영역 안에서 |

#### 쿨다운
`10분(36000f) - Lv당 30초(1800f). 최소 100초(6000f)`

| Lv | 쿨다운 |
|---|---|
| 1 | 9분 30초 |
| 5 | 7분 30초 |
| 10 | 5분 |
| 17+ | 1분 40초 (최소) |

- HP/MP/ST 풀 회복, 2초 무적
- 습득비: SP 15, 악의 150. 강화비: SP 10/Lv

#### 현재 구현 상태
- ✅ 쓰러짐 연출: 사망 시 'fallen' 상태 진입 → 5초(300f) 카운트다운 → 부활 판정 + 쓰러짐 회전/페이드 연출 + fallen/dead 중 모든 행동 차단
- ✅ 쿨다운제 (10분, P._demonRevCd)
- ✅ 카운트다운 중 무적, 피 파티클, 비네팅, 부활력% 표시
- 🔲 선택지 UI 미구현
- 🔲 인간 HP+30% 미구현
- 🔲 세이브 영구 저장 미연동

#### 향후 확장 — 부활 패시브 스킬화
- **필살기 탭에 부활 패시브 추가** — 부활을 스킬로 습득/강화하는 구조
- 부활확률 = 부활력(revivePow)으로 환산
  - 기본 부활력: 스킬 레벨 기반
  - 추가 부활력: 장비 어픽스(악마의 팔찌 등) + 부활의 영역 스킬
- 쿨다운: 스킬 레벨로 감소 가능 (기본 10분)
- 악마화: 외형 변화, 악마 전용 스킬 트리
- 인간: 신성력 스킬 트리 해금
- 엔딩 분기: 악마 vs 인간
