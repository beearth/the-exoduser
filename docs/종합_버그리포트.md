## 2026-04-03

### [치명] 보스 게이트 나무 미표시 + 투명 충돌체 차단
- **증상**: 보스 게이트 나무 이미지 안 보임, 투명 충돌체가 플레이어 이동 차단, 포탈 진입 불가
- **원인**: tileRLE 맵에서 맵 캐시 Phase 2 미실행 → 나무 drawImage 누락. exit/충돌/렌더 좌표 불일치
- **수정**: 나무를 실시간 렌더로 이전, 충돌체를 실시간 렌더 나무 중심 `(bossCx+0.5, gateY+0.5)*T`로 3곳 통일
- **상세**: `docs/4.1맵디자인+설정/보스게이트_버그리포트.md`

### [중] MAP_OBJS 투명 오브젝트 벽 현상
- **증상**: `boss_gate_col` 등 스프라이트 없는 충돌 오브젝트가 보이지 않는 벽으로 작용
- **원인**: MAP_OBJS 정의에 `col:1` (충돌 활성)이지만 렌더 스프라이트가 null → 화면에 안 보이지만 충돌 판정 활성
- **영향**: 플레이어가 빈 공간에서 막히는 현상
- **대응**: 충돌 오브젝트 위치를 실시간 렌더 나무 중심 `(bossCx+0.5, gateY+0.5)*T`와 일치시켜 의도된 봉인으로만 작동하게 함
- **향후**: 스프라이트 없는 col 오브젝트에 디버그 모드 시 빨간 윤곽 표시 권장

### [중] 사운드 시스템 — 300마리 동시타격 시 전체 사운드 멈춤
- **증상**: 대량 적 동시타격 시 BGM 포함 모든 사운드 정지
- **원인**: `playTone`/`playNoise`/`playFM`/`playSub`/`playDist` 5개 프로시저럴 사운드 함수에 AudioNode 수 제한 없음 → 수백 개 노드 생성 → AudioContext 스레드 포화
- **수정**: 
  - 5개 함수 전부 `_activeNodeCnt >= _MAX_ACTIVE_NODES` 체크 추가
  - `onended` + `setTimeout` 이중 정리 (onended 미발화 대비)
  - 사운드 우선순위 시스템 (`_SFX_PRI`) 도입: SKILL(10) > VOICE(9) > BOSS(5) > HIT(1) > AMB(0)
  - 채널 포화 시 최저 우선순위 노드 stop() 후 교체
  - HIT 동시 최대 4개 하드캡
  - 30ms 중복 재생 방지

### [경] BGM 루트 폴더 중복 파일
- **증상**: `bgm/` 루트에 장 폴더 안과 동일한 mp3 파일 17개 중복
- **원인**: 파일 정리 안 된 레거시
- **수정**: 바이트 비교 후 루트 중복 삭제, `lobby.mp3`/`intro_voice.mp3` 경로를 `bgm/공통/`으로 수정
