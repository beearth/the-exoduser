## 참고: 장비명 조합 번역 구조

장비명 = `접두사` + `기본형` 조합이므로 각각 번역하면 자동 조합됨.
- 예: "녹슨 대검" → _EN_PFX["녹슨"]="Rusty" + _EN_BASE["대검"]="Greatsword" → "Rusty Greatsword"
- 다국어 확장 시 `_LANG_PFX`, `_LANG_BASE` 분리 필요
