## [한 줄 정의]

한 전사가 가족의 복수를 위해 지옥에 강하한다. 그는 자신의 가족 살해도, 친구의 배신도, 자신의 강함도, 탈출 의지조차 한 사람의 농사임을 모른다.

## [악의 4법칙]

1. 악의는 생물만이 만든다.
2. 씨앗 = 탐욕의 도구 = 폭력 + 구속.
3. 뿌리 = 균형 붕괴 (과잉 권력, 과잉 욕망).
4. 운동 = 블랙홀. 팽창 후 무한 흡수.

## [악의 정의]

악의 = 균형을 깨는 모든 힘.
운동 형태 = 더 모음 / 가짐 / 약탈 / 죽임 / 가져감 / 파괴.
모두 한 방향 = 외부에서 자기로의 흡수.

"악" = 결과/평가
"악의" = 의지 + 운동성 + 에너지

지옥은 명사가 아니라 동사로 작동하는 세계. 악의는 항상 작동 중인 것. 멈추면 악의가 아님.

## [단일 에너지 법칙]

지옥에는 다른 에너지가 없다. 빛, 열, 영양, 마나, 신성력 - 전부 없다. 오직 악의 하나로 모든 것이 작동한다.

생성 = 악의의 응결
파괴 = 악의의 분산
무기 = 악의의 집약체
음식 = 악의의 섭취 가능 형태
회복 = 악의의 보충
이동/공격/방어 = 악의의 운용

## [네메시아 = 무의 의인화]

표면 정체: 복수의 여신
중간 정체: 영혼 수집자
진짜 정체: 무를 원하는 자. 천계 파멸이 1차 목적, 모든 것의 소멸이 최종 목적.

그리피스를 초월하는 빌런 카테고리:
- 그리피스 = 자아 팽창 (자기를 신으로)
- 네메시아 = 자아 소멸 (모든 것을 무로)

네메시아는 즉위를 원하지 않을 수 있다. 즉위는 수단. 천계 신의 권능이 있어야 천계를 무너뜨릴 수 있으므로 통과 지점일 뿐. 최종 단계에서 자신마저 소멸시킨다.

도진님 4법칙의 완전한 화신 = 흡수만 하는 블랙홀이 결국 자기 자신까지 흡수.

그리피스형 5요소 (캐릭터 디자인/연출에 박힘):
1. 가장 사랑한 자들을 제물로 바칠 수 있는 자
2. 그 행위에 죄책감이 아니라 정당성을 느끼는 자
3. 자신의 야심을 운명/사명으로 믿는 자
4. 아름다운 외형 안에 절대적 공허
5. 배신이 아니라 "당신은 처음부터 도구였다"고 통보하는 자

7장 폭로 시 차분하게, 거의 다정하게 말한다 - "너는 이 자리를 위해 만들어졌다."

그리피스를 초월하는 메커니즘:
그리피스는 사람을 죽였다.

## [챕터 7단계 + 보스 분배]

챕터 1 - 추락 (5보스): 평범한 자들이 충에 잠식된 사례. 농부, 어부, 어머니, 노인, 아이.

챕터 7 - 진실 (5보스): 지옥마왕 + 호위 + 진짜 최종전 (네메시아).

## [진행 원동력 - 떡밥 페이싱]

전제: 학살 쾌감을 절대 깨지 않는다. 스토리는 학살의 양념이지 본편이 아님.

떡밥 박는 위치 우선순위:
1. 환경 오브젝트 (멈출 필요 없음, 지나가며 본다) - 50%
2. 적/보스 디자인 (실루엣, 비대칭, 충의 잔재) - 20%
3. 보스 죽기 직전 한 줄 대사 - 15%
4. 거점 짧은 대화 - 10%
5. 챕터 끝 컷신 (30초 이내) - 5%

읽을 텍스트가 없어도 게임은 끝까지 진행 가능.
추적자는 환경을 보며 알아간다.
무시하는 자는 학살에 집중한다.
둘 다 7장에서 동일한 충격을 받는다.

페이싱 곡선:
1~3장 = 쾌감 우선. 도덕적 단서 거의 없음. 영웅 서사 몰입.
4~5장 = 첫 균열. 죽인 적이 마지막 순간 인간으로 돌아오는 연출 1~2회.
6장 = 의심 누적. 동료 행동의 미세한 이상.
7장 = 모든 것이 한 번에 무너짐. 폭로 연쇄.

쾌감과 무게가 시간차로 도착해야 한다. 동시 도착하면 둘 다 죽는다.

## [시스템 코어 일치 검증]

기존 시스템 무엇도 안 깨짐. 모두 코어와 정합.

HP = 보유 악의량. 0 = 악의 고갈 = 형체 유지 불가.
공격 = 악의의 일방향 방출. 적의 악의를 빼앗으려는 행위.
피격 = 악의를 빼앗김.
회복 = 외부 악의 흡수.
강화 = 악의의 농축.
패링 = 악의의 역류.
오패링 폭발 풀뎀+넉백+슬로우모 = 농축된 악의의 즉시 분산.

BULLET LANGUAGE = 악의의 언어:
탄환 = 충이 발사하는 농축 악의
색 = 충의 종류 (6원소)
패링 = 충의 흐름을 정확히 인식하고 역류
오패링 = 충의 도화선을 잘못 끊음

CHAIN DRIVE = 천계가 지옥에 박은 채굴 라인. 주인공이 알지 못하고 동력으로 써왔다. 그가 강해질수록 채굴이 활성화된다. 도구가 도구를 사용하는 자를 도구로 만든다.

6원소 = 충의 6가지 변종 = 6가지 흡수 양식:
화 = 태우며 흡수 (소진)
빙 = 멈추며 흡수 (고립)
뇌 = 꿰뚫으며 흡수 (관통)
지 = 으깨며 흡수 (압살)
암 = 빼앗으며 흡수 (회복 차단)
신 = 정화하며 흡수 (부활/언데드)

원소 상태이상 x 공격 타입 상성:
화상(불) = 베기 약점
빙결 = 둔기 약점
감전 = 투사체 약점
대지 = 명중률 감소 디버프
암흑 = 회복력 감소 디버프
신성 = 부활력 + 언데드 강함

펫 시스템 = 길들여진 충. 부산 악의의 자동 흡수.
대장간 = 악의의 정련소.
강화 확률 공식 = 악의 농축의 통계적 한계 (천계가 만든 시스템).

## [비주얼 랭귀지]

악의 = 일방향 운동에서 자동 도출.

빛: 한쪽으로 흘러간다. 광원이 아니라 광종착지가 존재한다.
소리: 메아리 없음. 잔향 0. 발생하면 어딘가로 흡수돼 사라진다.
공기: 항상 한 방향 흐름. 흡인. 캐릭터 망토/머리카락이 보스 쪽으로 끌린다.
지형: 모든 것이 중심부로 기울어 있다. 수직선 없음. 모든 선이 어딘가를 향해 휘어있다.
좌우 대칭 금지: 대칭 = 균형 = 악의 부재. 지옥에 좌우 대칭 존재 불가.
생물 형상: 입/구멍/손 비대. 먹는/잡는/가져가는 부위만 발달. 주는 부위 없거나 흡수기관으로 변형.
색: 채도 낮은 바탕 + 한 점의 강한 색 (피, 불, 영혼광). 강한 색 = 흡수당하는 중인 것. 곧 사라질 것.
캐릭터 비대칭: 흡수한 쪽 비대, 빼앗긴 쪽 결손.
보스: 가장 많이 흡수한 자 = 가장 비대 + 가장 일그러진 자. 약점 = 흡수기관 자체.

네메시아 예외:
다른 모든 것은 일그러져 있는데 네메시아만 정제된 형상. 정제된 아름다움. 이 대비가 그의 정체를 무의식적으로 폭로함.
네메시아 등장 시 주변 환경이 일그러지지 않음 (다른 보스와 반대).

## [BIC 피치 한 문단]

한 전사가 가족의 복수를 위해 지옥에 강하한다.
그는 모르고 있다 - 자신의 가족이 살해당한 것도, 친구가 그 손에 이용당한 것도, 자신이 지옥에 떨어진 것도, 그 안에서 강해지는 것도, 모두 한 사람의 농사라는 것을.
악의는 생물이 만들고, 누군가는 그것을 수확해왔다.
이 게임은 핵앤슬래쉬의 쾌감 위에, 자신이 무엇을 위해 싸우는지 모르는 자의 35회 학살을 얹는다.
모든 분노, 모든 절망, 모든 학살의 잔향이 어딘가의 창고로 흘러들고 있다.
플레이어가 그 사실을 깨닫는 순간 게임은 끝나지만, 이야기는 그제서야 시작된다.
