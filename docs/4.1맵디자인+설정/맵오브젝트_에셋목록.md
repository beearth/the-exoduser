## 코드 위치
- game.html: `_OBJ_SPR_MAP` (이미지 경로), `_OBJ_META` (크기 메타)
- editor.html: `OBJ_DEFS` (에디터 오브젝트 정의)
- 렌더링: `MAP_OBJS` switch default case
