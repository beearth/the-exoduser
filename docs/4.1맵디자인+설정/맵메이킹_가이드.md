## 1. 전체 구조

### 맵 생성 파이프라인
```
1) FIXED_MAPS (방/복도 레이아웃)  →  genFromTemplate()  →  좁은 통로 자동 확장 (폭5미만→양쪽+1)  →  tileRLE (바닥/벽 이진맵)
2) 타일 텍스처 에디터            →  PNG 내보내기        →  게임 렌더링에서 바닥 위에 덮어쓰기
3) MAP_OBJS (오브젝트 배치)      →  에디터 or 자동배치  →  게임 렌더링
```

### 게임 타일 렌더링 레이어 (아래→위)
| 순서 | 레이어 | 설명 |
|------|--------|------|
| 1 | 바닥색 | `_MTHEME[hell].f` 단색 |
| 2 | 타일 텍스처 | 인섹트 팩 등 256x256 스탬프 (5×5타일) |
| 3 | 벽 엣지 | 벽/바닥 경계에 엣지 텍스처 |
| 4 | 데칼 | 알둥지, 고치, 점액 등 |
| 5 | 글로우 | 발광 포자, 룬 문양 |
| 6 | 오버레이 | 바닥 노이즈, 벽 때 |
| 7 | MAP_OBJS | 나무, 뼈, 버섯, 거미줄 등 |

## 5. 에디터 사용법

### 기본 워크플로우
1. **맵 선택**: 왼쪽 장탭 → 스테이지 클릭
2. **타일 선택**: 팔레트에서 텍스처 클릭 (1~9 단축키)
3. **스탬프 찍기**: 캔버스에 클릭/드래그
4. **회전/반전**: R(90°), F(좌우), V(상하), 화살표키
5. **저장**: 💾현재맵 저장 or 💾전체저장 (.json)
6. **내보내기**: 📤 PNG내보내기 (게임 해상도 40px/tile)

### 단축키
| 키 | 기능 |
|----|------|
| S | 스탬프 도구 |
| E | 지우개 도구 |
| Space(홀드) | 이동 도구 |
| R | 90° 회전 |
| F | 좌우 반전 |
| V | 상하 반전 |
| [ / ] | 스탬프 크기 ±1타일 |
| 1~9 | 팔레트 선택 |
| Ctrl+Z | 되돌리기 |
| 마우스휠 | 줌 |
| 마우스 가운데 | 이동 |

### 레이어 시스템
- 배경(바닥), 엣지, 데칼 등 레이어 분리 가능
- 레이어 추가/삭제/병합/가시성
- 불투명도 조절 (스탬프 블렌딩)

### 맵 가이드
- 🗺️맵가이드 버튼: map_tiles.json의 tileRLE 데이터를 오버레이
- 벽/바닥 경계선 표시 → 텍스처 칠할 영역 확인

```
1. 바닥 4종 — "top-down seamless floor texture, [장 테마], dark fantasy, 256x256"
2. 엣지 4종 — "top-down wall edge texture, [장 테마], seamless horizontal, 256x256"
3. 데칼 4종 — "top-down decal overlay, [장 테마], transparent background, 256x256"
4. 글로우 2종 — "top-down glow effect, [장 테마], transparent, 256x256"
```
