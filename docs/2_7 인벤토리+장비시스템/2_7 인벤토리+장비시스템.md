# 2-7. 인벤토리 + 장비 시스템

## INV 구조
```
INV = {
  bag: [],          // 가방 아이템 배열
  equipped: {       // 장착 슬롯
    weapon, shield, boots, armor, helmet,
    bow, gloves, pants, belt,
    necklace, ring1, ring2, cape
  }
}
```

## 기능
- 필터 (슬롯별, 레어리티별)
- 아이템 비교 (장착 vs 가방)
- 가방 확장
- 장착/탈착
- 장비 분해 (`salvageVal()`, 악의 변환, 일괄분해)
- 쓰레기 자동분해 (고양이 펫)

## 장비 슬롯 (SLOT_NAMES)
weapon, shield, boots, armor, helmet, bow, gloves, pants, belt, necklace, ring1, ring2, cape, bracelet, headband

### 신규 슬롯
- **bracelet (팔찌)**: 부활력(revivePow), 부활쿨감(reviveCd), 부활억제(antiRevive) 특화
- **headband (머리띠)**: 마법데미지(magicDmg), MP소비감소(mpCostRed), MP%(maxMPPct) 특화

## 레어리티 5단계
| 등급 | 이름 | 어픽스 수 |
|---|---|---|
| 0 | 일반 | 0 |
| 1 | 고급 | 1 |
| 2 | 희귀 | 2 |
| 3 | 영웅 | 3 |
| 4 | 전설 | 4 + 전설특수 |

## 수정 금지
- INV 객체 구조
- 기존 equipItem/unequipItem 시그니처
