# 2-7. 인벤토리 + 장비 시스템

## INV 구조
```
INV = {
  bag: [],          // 가방 아이템 배열
  equipped: {       // 장착 슬롯
    weapon, shield, boots, armor, helmet,
    bow, gloves, pants, belt,
    necklace, ring1, ring2, cape
  }
}
```

## 기능
- 필터 (슬롯별, 레어리티별)
- 아이템 비교 (장착 vs 가방)
- 가방 확장
- 장착/탈착
- 장비 분해 (`salvageVal()`, 악의 변환, 일괄분해)
- 쓰레기 자동분해 (까마귀=아이템 수거, 고양이=악의 수거. **전설(4) 이상은 바닥에 남김 — 자동분해 대상 제외**)

### 정렬 (`↕ 정렬` 버튼 / `sortInvBag()` / `_invAutoPlace()`)
정렬 우선순위:
1. **즐겨찾기(fav)** → 최상단
2. **쓰레기(junk)** → 최하단
3. **종합지수(itemPower) 내림차순**

### 종합지수 (`itemPower`) 산출 공식
| 카테고리 | 스탯 | 가중치 |
|---|---|---|
| DPS | 무기: atk×spd×2, 활: atk×bowSpd×2, 기타: atk×2 | — |
| 방어 | def×3, eDef×2, spd×40 | — |
| 전투 | wStagger×1.5, wRange×3, bonusRange×8, bowRange×2, bowPierce×200 | — |
| 전투2 | stagger×1.5, blockRate×2, charge×1, chargeDist×1, chargeW×0.5, chargeAoE×2, bonusChargeDist×8, chargeCharge×60 | — |
| HP/ST | bonusSt×1.5, bonusHp×2 | — |
| 크리/속도 | critRate×30, critDmg×200, atkSpd×300, potCd×100 | — |
| 마법/빔 | beamDmg×20, splashR×4, splashMul×80, magicPen×30, 원소dmg×0.8 | — |
| 리젠 | stRegen×200, mpRegen×500 | — |
| 스탯보너스 | bStr×3, bDex×3, bInt×3 | — |
| 임플리싯 | >1이면 val×2, ≤1이면 val×100 | — |
| 어픽스 | |value|×10 + tier×5 | — |
| 등급/강화 | enh×15, rarity×20, itemLv×2 | — |
| 전설특수 | +80 고정 | — |

### 분해 보상 공식 (`salvageVal`)
```
기본값 = [1000, 2000, 4000, 8000, 16000][rarity]   // 등급당 2배
강화값 = enh < 100 ? enh × 3 : 300 + (enh-100) × 1  (최대 1000)
티어값 = tier × 2
총합   = 기본값 + 강화값 + 티어값
```
| 등급 | 기본 악의 |
|------|-----------|
| 일반(0) | 1,000 |
| 고급(1) | 2,000 |
| 희귀(2) | 4,000 |
| 영웅(3) | 8,000 |
| 전설(4) | 16,000 |

## 아이템 레벨 체계
- **itemLv** = `floor(캐릭터레벨 / 10) × 10` (캡 900)
- **reqLv** = `max(0, itemLv - 10)` (착용 요구 레벨)
- 10레벨 단위로 아이템 등급 상승 (0, 10, 20, ... 900)
- 캐릭터 레벨 기반 — 맵/장 무관

## 장비 슬롯 (SLOT_NAMES)
weapon, shield, boots, armor, helmet, bow, gloves, pants, belt, necklace, ring1, ring2, cape, bracelet, headband

### 신규 슬롯
- **bracelet (팔찌)**: 서브타입 2종 (드롭 시 50:50 랜덤)
  - **악마의 팔찌** (`brType:'demon'`): 임플리싯 부활력+5~12%, 전용 어픽스 revivePow/reviveCd
  - **생명의 팔찌** (`brType:'life'`): 임플리싯 최대HP+5~12%, 전용 어픽스 lifeMaxHP/lifeRegen/lifeHealPow
  - 공통 어픽스: antiRevive (부활억제)
- **headband (머리띠)**: 마법데미지(magicDmg), MP소비감소(mpCostRed), MP%(maxMPPct) 특화

## 레어리티 5단계
| 등급 | 이름 | 어픽스 수 |
|---|---|---|
| 0 | 일반 | 0 |
| 1 | 고급 | 1 |
| 2 | 희귀 | 2 |
| 3 | 영웅 | 3 |
| 4 | 전설 | 4 + 전설특수 |

## 강화 효과 (슬롯별)
| 슬롯 분류 | 강화 효과 |
|---|---|
| 무기류 (weapon, bow, helmet) | ATK × (1 + enhMul(enh)) |
| 방어구 (shield, armor, boots, gloves, pants, belt, necklace, ring1, ring2, cape, bracelet, headband) | DEF × enhMul(enh) + **HP +1/강화레벨** |

- 방어구 12슬롯 각각의 enh 합산 → 총 HP 보너스 (예: 갑옷+100, 바지+50 → HP +150)
- 강화 UI 툴팁에 HP 보너스 표시됨
