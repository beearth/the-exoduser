# 3파트 아이템 분류 (무기 / 방어구 / 장신구)
> 2026-04-10 제정 | 15개 장비 슬롯을 3개 파트로 그룹화
> 이 문서는 단일 진실 공급원(SoT): 결정 슬롯 제한, 장착창 레이아웃, 스탯 배분, 어픽스 풀 전부 여기서 파생

---

## 개요

| 파트 | 슬롯 수 | 역할 | 결정 카테고리 | 스탯 보너스 풀 |
|------|---------|------|---------------|----------------|
| **무기** | 3 | 공격 / 크리 / 원소 데미지 | 공격결정 8종 (`CR_ATK_SLOTS`) | **스탯 없음** (순수 크리/뎀) |
| **방어구** | 6 | HP / DEF / 이속 / 드롭률 | 방어결정 5종 (`CR_ARMOR_SLOTS`) | STR / DEX / INT |
| **장신구** | 6 | ST / MP / 리젠 / 속성방어 / 유틸 | 자원결정 4종 (`CR_ACC_SLOTS`) | **LCK / 근성(Grit)** |

- 총 15슬롯 = 3 + 6 + 6
- **결정 파트 제한**: 각 파트는 전용 결정 카테고리만 장착 가능 (`attachCrystal` 체크 + `openCrystalPicker` 필터)
- **스탯 보너스 제한**: 장신구는 LCK/근성만, 방어구는 STR/DEX/INT, **무기는 스탯 안 붙음** (`_statPool` 분기, game.html 라인 ~6844)

## 🔒 파트별 어픽스 룰 (절대 규칙)

> 이 섹션은 어픽스 풀 생성/검증 시 강제 준수. AFFIX_POOL 편집 시 반드시 체크할 것.

### 무기 파트 (weapon / bow / helmet)
✅ **허용**:
- 순수 크리 어픽스: 크리확률(`critRate`), 크리배율(`critDmg`)
- 공격 관련: 공격력, 공격속도, 원소 데미지, 관통, 스태거, 사거리
- 전설 특수 효과 (`LEGENDARY_SPECIAL` wtype별)

❌ **금지**:
- STR / DEX / INT / LCK / 근성 등 **모든 스탯 보너스** (`bStr`, `bDex`, `bInt`, `bLck`, `bGrit`)
- 방어력(flat DEF), 방어율(%), HP flat
- 속성 저항(원소 저항)

**근거**: 무기는 순수 공격 DPS 파트로 정체성 유지. 스탯 증폭은 방어구/장신구가 담당.

### 방어구 파트 (shield / armor / gloves / pants / boots / cape)
✅ **허용**:
- **방어 어픽스**: flat DEF +X (**최대 +50/티어, 5티어 = +250**), 방어율 %, 물리 피해감소
- HP flat/%, HP 리젠, 피해감소
- 이동속도, 회피, 방어 관련 유틸
- 스탯 보너스: STR / DEX / INT (`bStr`, `bDex`, `bInt`)

❌ **금지**:
- 크리확률/크리배율 (무기 전용)
- ST/MP flat (장신구 전용)
- 속성 저항(원소 저항) — **장신구 전용**
- LCK / 근성 (장신구 전용)

**방어 어픽스 5티어 테이블**:
| 티어 | flat DEF |
|------|----------|
| T1 | +10 |
| T2 | +20 |
| T3 | +30 |
| T4 | +40 |
| T5 | **+50** (최대) |

### 장신구 파트 (belt / bracelet / ring1 / ring2 / necklace / earring)
✅ **허용**:
- **속성 방어**: 화염/냉기/암흑/번개/신성/독 저항 (`fireRes`, `iceRes`, `darkRes`, `lightRes`, `holyRes`, `poisonRes`)
- 원소 저항 종합 (`_iElemRes` 임플리싯)
- ST/MP flat/%, ST/MP 리젠, 쿨다운 감소
- 드롭률, 경험치 증가, 물약 효율
- 스탯 보너스: **LCK / 근성만** (`bLck`, `bGrit`)
- 팔찌 서브타입: 부활력(demon), HP%(life)

❌ **금지**:
- **flat DEF (방어력)** — 장신구는 일반 방어력 절대 안 붙음
- 크리확률/크리배율 (무기 전용)
- STR / DEX / INT (방어구/무기 아님)
- 방어율 % (방어구 전용)

**근거**: 장신구는 **속성 저항 전담 파트**. 일반 방어력은 방어구 파트에서 얻고, 속성 저항은 장신구에서만 얻는 이원화 구조.

## 1. 무기 파트 (3슬롯)

### 슬롯 구성
| slot key | 한글명 | 영문명 | 임플리싯 고정 스탯 |
|----------|--------|--------|-------------------|
| `weapon` | 무기 | Weapon | 공격력 +X% (`_iAtkPct`, 5~15%) |
| `bow` | 석궁 | Crossbow | 관통 확률 +X% (`_iPierce`, 5~15%) |
| `helmet` | 왕관 | Crown | 최대 HP +X (`_iMaxHP`, 10~30) |

### 결정 (공격결정 8종)
| id | 이름 | 1성 기본값 | 효과 |
|----|------|-----------|------|
| cr_atk | ATK결정 | +2 | 공격력 |
| cr_crit | 크리결정 | +0.23% | 크리확률 |
| cr_cdmg | 치명결정 | +0.1% | 크리배율 |
| cr_fire | 화염결정 | +1.2 | 화염 추가뎀 |
| cr_ice | 냉기결정 | +1 | 냉기 추가뎀 + 감속 |
| cr_dark | 암흑결정 | +1.2 | 암흑 추가뎀 |
| cr_light | 번개결정 | +1.5 | 번개 추가뎀 |
| cr_holy | 신성결정 | +1 | 신성 추가뎀 + 부활억제 |

### 특징
- **공격 DPS 파트**: 기본 공격력, 크리티컬, 원소 데미지 중심
- 전설 특수 효과(`LEGENDARY_SPECIAL`)의 대부분이 무기에 집중 (wtype별 고유 효과)
- weapon 슬롯은 `wtype` (sword/dagger/hammer/mace/club/axe/spear)에 따라 세부 스탯 분기
- bow 슬롯은 `btype` (bow/crossbow)에 따라 사정거리/속도/관통 분기
- helmet은 무기 파트지만 결정만 공격계, 어픽스는 방어적 성격 겸비 (HP 임플리싯)

### 장착창 위치 (EQ_POS Row0)
```
[weapon] [helmet] [bow]
   L        M       R
```

## 2. 방어구 파트 (6슬롯)

### 슬롯 구성
| slot key | 한글명 | 영문명 | 임플리싯 고정 스탯 |
|----------|--------|--------|-------------------|
| `shield` | 견갑 | Shield | 방어율 +X% (`_iBlockPct`, 3~8%) |
| `armor` | 갑옷 | Armor | 물리 피해감소 +X% (`_iPhysDR`, 3~8%) |
| `gloves` | 장갑 | Gloves | 공격속도 +X% (`_iAtkSpd`, 3~8%) |
| `pants` | 바지 | Pants | 상태이상 저항 +X% (`_iStsRes`, 5~12%) |
| `boots` | 부츠 | Boots | 이동속도 +X% (`_iMovSpd`, 3~8%) |
| `cape` | 망토 | Cape | 회피 쿨다운 -X% (`_iDashCd`, 5~12%) |

### 결정 (방어결정 5종)
| id | 이름 | 1성 기본값 | 효과 |
|----|------|-----------|------|
| cr_hp | HP결정 | +8 | 최대HP (제작 가능) |
| cr_def | DEF결정 | +1 | 방어력 |
| cr_spd | 속도결정 | +0.08% | 이동속도 |
| cr_hpReg | 생명결정 | +0.15 | HP리젠/초 |
| cr_drop | 행운결정 | +0.15% | 드롭률 |

### 특징
- **생존/기동 파트**: HP, 방어력, 이속, 회피, 상태이상 저항
- 어픽스 풀: HP/방어 관련 스탯이 집중
- shield의 `_iBlockPct`는 우클릭 방패 시스템과 직결
- cape의 `_iDashCd`는 돌진 스킬 쿨다운에 직접 적용
- 대부분의 임플리싯이 %형이라 고티어로 갈수록 기하급수 성장

### 장착창 위치 (EQ_POS Row3~4)
```
Row3: [gloves] [armor]  [shield]
Row4: [pants]  [boots]  [cape]
```

## 3. 장신구 파트 (6슬롯)

### 슬롯 구성
| slot key | 한글명 | 영문명 | 임플리싯 고정 스탯 |
|----------|--------|--------|-------------------|
| `belt` | 벨트 | Belt | 물약 쿨다운 -X%, **데미지 +10~50%** (`dmgBonus:10+tier×10`) |
| `bracelet` | 팔찌 | Bracelet | **서브타입별 분기**, **데미지 +10~50%** (`dmgBonus:10+tier×10`) |
| `ring1` | 반지1 | Ring1 | 원소 저항 +X%, **크리율 +5%**, **데미지 +10~50%** (`dmgBonus:10+tier×10`) |
| `ring2` | 반지2 | Ring2 | 원소 저항 +X%, **크리율 +5%**, **데미지 +10~50%** (`dmgBonus:10+tier×10`) |
| `necklace` | 목걸이 | Necklace | 최대 HP +X%, **크리율 +10%**, **데미지 +20~100%** (`dmgBonus:20+tier×20`) |
| `headband` (표시명: 귀걸이) | 귀걸이 | Earring | 마법 데미지 +X%, **데미지 +10~50%** (`dmgBonus:10+tier×10`) |

> 내부 slot key는 `headband` 유지 (세이브/어픽스 슬롯 호환), 표시명만 귀걸이로 변경 (2026-04-10)

### 팔찌 서브타입 (bracelet)
| brType | 한글 | 전용 임플리싯 | 전용 어픽스 |
|--------|------|---------------|------------|
| demon | 악마의 팔찌 | 부활력 +X% (`_iRevPow`, 5~12%) | 부활억제, 부활력+, 부활쿨- |
| life | 생명의 팔찌 | 최대 HP +X% (`_iLifeHPPct`, 5~12%) | HP+, HP리젠+, 회복력+ |

### 결정 (자원결정 4종)
| id | 이름 | 1성 기본값 | 효과 |
|----|------|-----------|------|
| cr_st | ST결정 | +5 | 최대ST (제작 가능) |
| cr_mp | MP결정 | +5 | 최대MP (제작 가능) |
| cr_stReg | 기력결정 | +0.08 | ST리젠/초 |
| cr_mpReg | 마력결정 | +0.08 | MP리젠/초 |

### 특징
- **자원/유틸 파트**: ST/MP 풀, 리젠, 쿨다운 감소, 원소 저항
- **스탯 보너스 풀 차별**: 장신구만 `bLck`(LCK) / `bGrit`(근성) 롤, 다른 파트는 STR/DEX/INT
  - LCK: 드롭률/행운 스탯 증폭
  - 근성: 피격 시 생존 보정
- belt의 `_iBeltPot`은 물약 쿨다운 직결 (포션 연타 빌드 핵심)
- necklace의 `_iMaxHPPct`는 %형이라 armor/helmet의 flat HP와 승수 시너지
- 귀걸이(headband)의 `_iMagDmg`는 마법 DPS 빌드 핵심, `pMagicMul()`에 직접 반영

### 장착창 위치 (EQ_POS Row1~2)
```
Row1: [bracelet] [necklace] [headband=귀걸이]
Row2: [ring1]    [ring2]    [belt]
```

## 🐞 수정된 버그

### AFFIX_DEFS ReferenceError (2026-04-11 수정)
**증상**: 반지(ring1, ring2) / 목걸이(necklace) 드롭 안 됨

**원인**: `mkItem()` 반지/목걸이 추가 어픽스 생성 블록(~6871, ~6883)에서 `AFFIX_DEFS.find(...)` 호출.
해당 상수는 존재하지 않음 (실제 이름 `AFFIX_POOL`) → `ReferenceError` 발생 → 아이템 생성 예외로 드롭 실패.

**수정**: `AFFIX_DEFS` → `AFFIX_POOL` 4곳 일괄 치환 (game.html 라인 ~6871/6872/6883/6884).

**교훈**: 새 어픽스 추가/이름 변경 시 해당 블록 참조도 함께 검증 필요.

- **2026-04-10 (AFFIX_POOL 적용)**:
  - `_AFSLOT.belt` 버그 수정: `'armor'` → `'belt'`
  - `mkItem` 스탯 보너스 분기: `_isWeaponPart` 추가 (무기 파트 스탯 보너스 스킵), `_isAccess`에 belt/headband 포함
