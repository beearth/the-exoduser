### 사망 사운드 재생 규칙
- 보스: `death_boss` (100%)
- 구울: 크기별 전용 사운드 (100%)
- 일반몹: **항상 재생** (카테고리별 사운드), 볼륨 0.35~0.7
- 대형(r≥40): 추가 25%로 `monster_die1` 중첩 (vol 0.5)
- 전체: 추가 15%로 `death_grunt` 신음 레이어 (vol 0.3)

### 사망 사운드 변형 목록 (_DEATH_VARIANTS)

|-----------|----------|

| death_flesh | death_flesh, death_flesh2 |
| death_heavy | death_heavy, death_heavy3, die_normal |
| death_insect | death_insect, death_insect2 |
| death_bone | death_bone, death_skull |

### 범위 폴백 (카테고리 미분류 시)
- et≥50: death_demon
- et≥40: death_heavy (랜덤 변형)
- et≥20: death_ice
- et≥10: death_slime
- 기본: monster_die1

### 특수 오버라이드 (_ETYPE_SFX)
개별 etype에 아키타입과 다른 이동 사운드 매핑 (예: etype 5=자폭형→m_suicide)

## 몬스터 피격 사운드
- `_hitSfxForEtype(et)` — 카테고리별 키+피치 반환
- hurtE에서 50% 확률로 재생 (과다 방지)
- 보스: `boss_hit` 전용 + 15% `monster_hurt`

## 예정 항목
- BGM 목록 (지옥별 14곡 + 타이틀/엔딩)
- ElevenLabs 보이스 목록 (voice_* 키 매핑)
- SFX 파일 포맷 규칙 (mp3, wav 교체 대상)
- 사운드 우선순위/동시 재생 제한
