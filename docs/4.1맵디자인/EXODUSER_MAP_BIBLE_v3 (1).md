# 세계관 흐름

지옥에 떨어진 PLR이 7개 층을 돌파해 지상으로 탈출한다.

```
썩은 숲 → 벌레굴 → 얼음굴 → 화염지대 → 지옥 군단 → 사도의 마굴 → 지옥성 → 탈출
```

## STAGE 1 — 썩은 숲 (Rotten Forest)

### 개요
| 항목 | 내용 |
|------|------|
| 에리어 수 | 4 |
| 원소 | 독/물리 |
| 분위기 | 썩어가는 숲, 안개, 뿌리와 덩굴, 눈 달린 버섯 |
| 사운드 | 나뭇가지 부러지는 소리, 벌레 소리, 물방울 떨어지는 소리 |
| 보스 | 숲의 기생수 |
| 난이도 | ★☆☆☆☆ (입문) |

### 에리어 구성
| # | 에리어 | 크기 | 맵 타입 | 특징 |
|---|--------|------|---------|------|
| 1 | 숲 입구 | 200×200 | vert | 시작맵, 튜토리얼. 썩은 나무 사이로 길이 보인다 |
| 2 | 뒤틀린 숲길 | 300×300 | horiz | 나무가 점점 기괴해짐. 눈 달린 버섯 첫 등장 |
| 3 | 눈달린 버섯 군락 | 350×350 | vert | 거대 버섯 사이 미로. 벌레떼 본격 등장 |

### 환경 요소
- 썩은 나무: 쓰러진 거대 나무가 길을 막음 → 돌아가야 함

- 눈 달린 버섯: 배경 장식이면서 일부는 몬스터
- 독 웅덩이: 밟으면 지속 독 데미지
- 안개: 시야 제한 (숲 깊이 갈수록 심해짐)

| 사운드 | 점액 물방울, 벌레 기어가는 소리, 습한 공기 |

| # | 에리어 | 크기 | 맵 타입 | 특징 |
|---|--------|------|---------|------|
| 1 | 벌레 터널 입구 | 250×250 | vert | 숲에서 굴로 전환. 벽이 점점 살색으로 |
| 2 | 점액 동굴 | 300×300 | horiz | 점액 웅덩이(감속). 넓은 동굴 |
| 3 | 기생충 서식지 | 300×300 | vert | 벽에서 기생충이 튀어나옴 |

### 환경 요소
- 점액 웅덩이: 밟으면 이동속도 50% 감소
- 알주머니: 공격하면 터지며 구더기 2~3마리 스폰
- 살점 벽: 배경 장식. 가끔 맥박처럼 움직임
- 벌레 터널: 좁은 통로 (벌레가 줄줄이 나옴)
- 기생충 구멍: 벽 구멍에서 기생충이 튀어나오는 트랩

### 몬스터
| 이름 | 디자인 | 공격 |
|------|--------|------|
| 구더기 | 하얀 통통한 유충, 이빨 원형 | 느린 추적, 자폭(독) |
| 기생충 | 가늘고 긴 벌레, 벽에서 튀어나옴 | 잠복→물기, 감속 |
| 벌레 군단 | 20마리가 하나로 움직임 | 군체 돌진, 분산→재집결 |
| 점액 괴물 | 투명 젤리 안에 눈알 | 느린 추적, 밟으면 감속장판 |
| 알낳는 등살 | 등에서 알을 낳으며 도망 | 도주+소환(구더기) |
| 거대 지네 | 2m 지네, 갑각+다리 수십 개 | 빠른 돌진, 감기(잡기) |
| 살벽 입 | 벽에 달린 거대 입 | 고정형, 지나가면 물기 |
| 기생 인형 | 인간형인데 안에 벌레가 조종 | 근접, 죽으면 벌레 2마리 |

- 고드름 트랩: 천장에서 떨어지는 고드름 (범위 데미지)
- 냉기 안개: 시야 제한 + 지속 빙결 축적
