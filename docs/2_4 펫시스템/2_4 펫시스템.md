## 유령불꽃 시스템
- **크기**: 1~100 조절 가능 (ESC 설정 → 슬라이더, 기본값 20)
- **악의 소모**: 1분(3600프레임)마다 `크기 × 1` 악의 자동 차감
- **악의 부족 시**: 크기 절반으로 자동 축소 (최소 1), "악의 부족! 불꽃 축소" 경고 표시
- **조명 효과**:
  - 동적 광원: 반경 120+(sz/100×380)px, 밝기 0.4+(sz/100×0.6)
  - 보조 광원: 메인의 50% 반경, 흰보라(230,200,255)
  - 토치 오버레이: 반경 80+(sz/100×250)px, 어둠 제거
  - 광원 색상: RGB(255,160,40) 앰버/오렌지 (횃불색 통일)
- **렌더링**: additive 블렌딩 (lighter), 고정 크기 불꽃 (반경 8px)
  - 외곽(#662200) → 글로우(#cc5500) → 코어(#ffbb44) → 핵(#fff8e0) + 불꽃혀 4개 고정
  - 위로 올라가는 불꽃 혀 파티클 (3~11개, 크기 비례)
- **궤도**: cos(t×0.015+π/2) × 32px, 정현파 Y오프셋 (느린 타원)
- **세이브**: `game.irisSz` (G._irisSz)에 저장/로드
- **토글**: Y키 (BINDS.irisToggle = KeyY) — ON/OFF 전환, G._irisOff 플래그
  - ON 시: 토치 반경에 sz/100×300 추가 (기본 +60, 최대 +300). 불꽃 애니는 고정 크기, 빛만 커짐
  - OFF 시: 업데이트/렌더링/광원/토치오버레이 모두 중단, 악의 소모 없음
