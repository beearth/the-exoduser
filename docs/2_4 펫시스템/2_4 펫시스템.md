## 요정불꽃 시스템
- **크기**: 1~100 조절 가능 (ESC 설정 → 슬라이더, 기본값 20)
- **악의 소모**: 없음 (무료)
- **조명 효과**:
  - 메인 광원: 반경 200+(sz/100×600)px, 밝기 0.6~1.0, RGB(255,170,50)
  - 코어 광원: 메인의 40%, RGB(255,230,150)
  - 주변 광원: 메인의 130%, RGB(200,120,30)
  - 동적 광원 방식 (_pushLight) — 캐릭터 글로우와 동일 시스템, 자동으로 따라감
  - **토치 오버레이 조명** (destination-out 방식, 횃불과 동일):
    - 외곽 소프트 글로우: 반경 450px, alpha=flickr×0.45
    - 내곽 코어: 반경 220px, alpha=flickr×0.7
    - flickr 범위: 0.49~0.81 (sin 2파장 합성)
- **렌더링**: 고정 크기 불꽃
  - 번짐 글로우 헤일로: 반경 55px (#ffa028, alpha=0.18×flickr) + 반경 36px (#ffcc44, alpha=0.12×flickr)
  - 불꽃 코어: 반경 14px (#ffcc33) + 핵 반경 7px (#ffffee)
  - 날개: 좌우 ellipse, 펄럭임 sin(t×0.12)
  - 위로 올라가는 꼬리 파티클 8개 (#ffee55→#ff8811)
- **궤도**: cos(t×0.015+π/2) × 32px, 정현파 Y오프셋 (느린 타원)
- **세이브**: `game.irisSz` (G._irisSz)에 저장/로드
- **토글**: Y키 (BINDS.irisToggle = KeyY) — ON/OFF 전환, G._irisOff 플래그
  - ON 시: 플레이어 본체 광원 최소화 (800→60, 1.0→0.08) + 불꽃 _pushLight 3겹이 주 광원
  - OFF 시: 업데이트/렌더링/광원/토치오버레이 모두 중단, 악의 소모 없음
