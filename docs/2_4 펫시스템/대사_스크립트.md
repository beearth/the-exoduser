# 펫 대사 스크립트 (전체)

> **코드 위치**: `game.html` [S03e] PET_DIALOGUE (~5717줄~)
> **시스템 함수 2종**:
> - `_petSayCD(id, who, txt, durSec, cdSec, pairWho, pairTxt, pairDur)` — 일반 대사. 글로벌 쿨 15초, 말풍선 중이면 차단
> - `_petSayUrgent(id, who, txt, durSec, cdSec, pairWho, pairTxt, pairDur)` — **긴급 대사**. 모든 쿨 무시, 조건 충족 시 무조건 출력. 같은 id 재생 중이면 스킵(`_uid` 가드). 긴급 후 글로벌 쿨 10초 (일반 대사만 억제)
> - `id`: 대사 고유 키 (중복 방지)
> - `who`: 선발 화자 (`crow`=핵터, `cat`=디로이)
> - `cdSec`: 같은 id 재출력 쿨다운 (초). 9999=1회성
> - `pairWho/pairTxt`: 티키타카 (1초 뒤 후발 대사)
> - **글로벌 쿨다운**: 일반 대사 후 15초 / 긴급 대사 후 10초 (연속 대사 방지)
