# 3.3 키바인딩 + 설정

## 키바인딩 (BINDS / BINDS2)
- BINDS: 주요 키 (이동, 공격, 스킬, 인벤, 설정 등)
- BINDS2: 보조 키 (같은 액션의 대체 키)
- 리바인딩 시스템: 설정 화면에서 변경 가능
- **키 설정 초기화**: 키바인딩 목록 하단 "⟲ 키 설정 초기화" 버튼 클릭, 또는 설정 패널(ESC) 열린 상태에서 **F5 키** → BINDS/BINDS2 전부 기본값 복원 (마우스 클릭 불가 상황 대비)

## 주요 바인딩
| 액션 | 기본 키 |
|---|---|
| 이동 | WASD |
| 공격 | 좌클릭 |
| 패링/방패 | 우클릭 |
| 돌진 | Shift |
| 마법 | E |
| 스킬 1~4 | 1~4 |
| 궁극기 | Z |
| 인벤토리 | TAB |
| 스킬패널 | K |
| 대장간 | G |
| 상호작용 | R |
| 처형 | X (그로기 보스 전용, 스킬 습득 필요) |

## 게임 설정 (OPT)
| 키 | 용도 |
|---|---|
| fpsCap | FPS 상한 (_FPS_STEPS=[30,60,120,144,244]) |
| bloom | 블룸 효과 |
| postfx | 포스트프로세싱 |
| trail | 잔상 |
| parts | 파티클 |
| torch | 횃불 조명 |
| ambPart | 앰비언트 파티클 |
| fog | 안개 |
| grain | 필름 그레인 |
| shake | 화면 흔들림 |
| hitStop | 히트스톱 |

## 하드웨어 Tier 프리셋
| Tier | 조건 | 설정 |
|---|---|---|
| S | 120fps+ | 전부 ON |
| A | 60~120fps | 블룸/트레일 ON |
| B | 40~60fps | 트레일만 |
| C | 40fps 미만 | 전부 OFF |

## Electron 전용
- IS_ELECTRON 플래그
- Tier A 강제
- 로그 억제
- 기본 fpsCap: 244

## 수정 금지
- BINDS, BINDS2 구조
- 리바인딩 시스템
- _FPS_STEPS 배열
