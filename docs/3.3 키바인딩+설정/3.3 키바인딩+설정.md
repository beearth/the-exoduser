# 3.3 키바인딩 + 설정

## 키바인딩 (BINDS / BINDS2)
- BINDS: 주요 키 (이동, 공격, 스킬, 인벤, 설정 등)
- BINDS2: 보조 키 (같은 액션의 대체 키)
- 리바인딩 시스템: 설정 화면에서 변경 가능
- **키 설정 초기화**: 키바인딩 목록 하단 "⟲ 키 설정 초기화" 버튼 클릭, 또는 설정 패널(ESC) 열린 상태에서 **F5 키** → BINDS/BINDS2 전부 기본값 복원 (마우스 클릭 불가 상황 대비)

## 키바인딩 표시명 (BIND_NAMES / BIND_NAMES_EN)
- `BIND_NAMES`: 한국어 표시명 (위로 이동, 무기 공격, 칼등 처내기 등)
- `BIND_NAMES_EN`: 영문 표시명 (Move Up, Weapon Attack, Blade Parry 등)
- `_bindName(key)`: OPT.lang에 따라 한/영 자동 선택 헬퍼 함수
- 설정 화면 키바인딩 목록에서 사용

### 키 토글 설정창 미러 (자동저장)
- 설정창(ESC) → 게임 옵션 섹션의 기동:칼날개 위쪽에 키 토글 체크박스 2종 미러:
  - **석궁 자동발사 (T)** (`#optXbowAuto`, 색 #ffe066): T키와 양방향 동기화
  - **버스트루프 당기기 (H)** (`#optBlPullMode`, 색 #44ccff): H키와 양방향 동기화
- 체크박스 클릭 = 키 누른 것과 동일 동작 (showPH/addTxt 안내 포함)
- 키 입력 시 체크박스 상태 자동 갱신 (T→optXbowAuto, H→optBlPullMode)
- **자동저장**: 토글 변경 시 `OPT.xbowAuto`/`OPT.blPullMode` → `saveSettings()` 즉시 호출
  - 키 누름/체크박스 클릭 둘 다 OPT 갱신 + localStorage 저장
  - 게임 재시작 시 `_xbowEquipped=OPT.xbowAuto` 로 복원
  - `P._blPullMode`는 P가 있으면 P 값 우선, 없으면 `OPT.blPullMode` 폴백
- 설정창 열 때 현재 상태 반영 (체크박스 = P 우선 → OPT 폴백)

## 스킬패널 (K키) — 아이콘 모드 + 디아블로식 상세 툴팁 (2026-04-11)

### 아이콘 그리드
- **기본 모드**: 모든 스킬이 74×108 아이콘으로 표시 (`_skCompact=true` 기본값)
- **아이콘 이미지**: `img/skillskin_upscaled/{skill.id}.png` 사용 (48×48, pixelated)
  - 폴백: 이미지 없을 시 `onerror` 핸들러 → 이모지(`sk.emoji`) 표시
- **아이콘 표시**:
  - PNG (48×48)
  - **스킬 이름** (font 11px, ★/HTML 제거 후 ellipsis 잘림, 폭 70px, text-shadow로 가독성)
  - Lv.X (학습 시 #44ff66) 또는 ⭐SP 비용 (미학습 #ff8844, font 10px)
- **카테고리 컬럼**: compact 모드에선 아이콘들이 flex-wrap 그리드로 배치, 펼친 스킬만 컬럼 전체 폭 차지

### 펼친 상세보기 (디아블로식 D2/D3 스타일)
- **클릭**: 아이콘 클릭 → 해당 스킬만 카테고리 컬럼 안에서 펼쳐서 상세보기
- **헤더**: 48px PNG + 스킬명 + 숙련도 + ✕ 닫기 버튼
- **`_skDetailHTML(sk, slv)` 함수**가 동적 스탯 섹션 생성:
  1. **스킬 정보**: 타입(물리/활/마법) / 레벨 / SP비용 / 악의비용 / 쿨다운 / 요구 레벨
  2. **데미지** (학습 시): 기본 참조뎀(meleeRef/bowRef/magicRef) / 스탯 배율 / 패시브 배율 / 레벨 배율 / 예상 평균뎀
  3. **치명타** (학습 시): 치명타 확률 / 치명타 배율 / 유효 배율
  4. **스킬 고유** (`_skSpecificDetails(sk, slv)`): id별 분기로 컴보 배율/관통/범위/사거리 등
  5. **설명** (원본 desc): 카테고리 섹션 아래 italic
- **동적 계산**: `meleeRef()`, `statStr()`, `pAtkMul()`, `statCrit()`, `statCritDmg()` 등을 try/catch로 안전 호출 → P 없을 때 폴백
- **고유 수치 분기 (`_skSpecificDetails`)**: kiSlash/whirlwind/fireball/detonate/maliceHunt/giantSlam/chargeBoost/magicBlink/darkPillar/spikeTrap 등 주요 스킬은 각자 고유 표 반환

### + / - 버튼 (학습/강화/취소/레벨다운)
펼친 상세보기 하단에 두 개의 버튼이 나타남:

| 버튼 | 조건 | 동작 |
|---|---|---|
| **+ 습득** | 미습득 + canLearn | `_skClick()` — `P.skills[sk.id]=1`, SP/악의 차감, 슬롯 자동 배정 |
| **+ 레벨업** | 학습됨 + canUp | `_skClick()` — `P.skills[sk.id]++`, SP/악의 차감 |
| **− 레벨** | 학습 + slv>1 | `_skUnclick()` — `P.skills[sk.id]--`, 직전 강화 비용(`_skillUpSpCost(slv-1)`) **환불**, `sk.upMat` 환불 |
| **− 취소** | 학습 + slv===1 | `_skUnclick()` — `delete P.skills[sk.id]`, **학습 비용 전액 환불** (`sk.spCost` + `sk.matCost`), 모든 슬롯에서 자동 해제 (LMB/charge/bow/magic/Q/tech/ULT) |

- 환불 시 `SFX.pickup()` + `addTxt(...) '취소!' or 'Lv.X (-)'` 알림
- 패널 닫기 시 `_skExpandedId=null` 리셋 (`closeAllPanels`)
- **전역 상태**: `_skExpandedId` (한 번에 하나만 펼침)
- **코드 위치**: game.html `_skDetailHTML`/`_skSpecificDetails` (라인 ~28049), `_renderSkillRow` (라인 ~29097), `_skUnclick` (라인 ~29503 인라인), `_skClick` (라인 ~29720)
- **아이콘 폴더**: `img/skillskin_upscaled/` (50+ 스킬 PNG, kiSlash/whirlwind/fireball 등 skill.id 매칭)

### BGM 선택 시스템
- `OPT.bgmTrack='auto'`: 스테이지별 자동 BGM (기본)
- 카테고리 선택 시: 해당 카테고리 내 랜덤 셔플, death/victory/forge만 자동전환 허용
- `t:경로` 선택 시: 해당 트랙 루프 재생, 스테이지 자동전환 완전 차단 (`BGM.playTrack()`)
- 선택 가능: 12카테고리 랜덤 + 27곡 개별 트랙 (optgroup으로 장별 그룹화)
