# 게임패드 매핑표 — THE EXODUSER

> 2026-05-19 기준. Xbox 360/One/Series 레이아웃 기준. PS는 △○✕□/L1L2R1R2 대응.

---

## 1. 전투 (기본)

| 버튼 | 키보드 대응 | 기능 | 비고 |
|------|------------|------|------|
| **A** | 좌클릭 (mouse0) | 무기 공격 | 오토에임 |
| **B** | Space | 대왕치기 (광역 기절) | |
| **X** | 우클릭 (mouse2) | 패링 / 방패 | |
| **Y** | Q | 패링 (데빌포스) | |
| **LB** | Shift | 돌진 / 작살 발사 | 즉시 발동 |
| **RB** | Ctrl | Ctrl 스킬 (유령걸음/얼음보주) | |
| **LT** | — | **모디파이어** (아래 참조) | 단독 탭=회복(F) |
| **RT** | Z | 필살기 (궁극기) | |
| **L3** (좌스틱 클릭) | R | 아이템 줍기 | 홀드=연속줍기 (150ms 간격) |
| **R3** (우스틱 클릭) | E | 마법 E슬롯 (beam/magic) | |

---

## 2. 모디파이어 (LT 홀드 + 버튼)

| 입력 | 키보드 대응 | 기능 |
|------|------------|------|
| **LT + A** | 1 | 스킬슬롯 1 |
| **LT + B** | 2 | 스킬슬롯 2 |
| **LT + X** | 3 | 스킬슬롯 3 |
| **LT + Y** | 4 | 스킬슬롯 4 |
| **LT 단독 탭** | F | 회복 영역 |
| **LT + RT** | X | 처형 (execution) |

> LT 홀드 중 ABXY를 누르면 스킬슬롯 발동, KeyF는 안 나감.
> LT를 ABXY 없이 뗄 때만 KeyF 탭 발동 (100ms).

---

## 3. 스틱

| 스틱 | 기능 | 비고 |
|------|------|------|
| **좌스틱** | 이동 (WASD) + facing 방향 | 우스틱 마법 발동 중에는 facing에 관여 안 함 |
| **우스틱** | 기울이면 마법(E) 자동 발동 + 방향 지정 | 트윈스틱 방식, 데드존 0.2. 발동 중 좌스틱은 이동만 담당 |
| **우스틱** (UI) | 스크롤 | 모든 UI 패널. 가상커서 모드→커서 위치 영역, 선택 모드→현재 포커스 영역 |

---

## 4. D-pad (메뉴/UI 바로가기)

| 방향 | 키보드 대응 | 기능 |
|------|------------|------|
| **↑** | K | 스킬 창 열기 / **한 번 더 → 닫기** |
| **↓** | G | 대장간 열기 / **한 번 더 → 닫기** |
| **←** | C | 능력치 창 열기 / **한 번 더 → 닫기** |
| **→** | V | 창고 열기 / **한 번 더 → 닫기** |

> 토글 동작: 해당 창이 열려있는 상태에서 같은 버튼을 다시 누르면 닫힘.  
> 열린 직후 20프레임(≈0.33초) 쿨다운으로 즉시 닫힘 방지.

| 버튼 | 동작 | 적용 화면 |
|------|------|-----------|

| **LB / RB** | 최상위 탭(계층) 전환 | 스킬창(추천/공격/방어/특수), 대장간 탭 등 |
| **A** | 선택 / 확인 / 장착 | 모든 UI |
| **B** | 뒤로 / 닫기 | 모든 UI |
| **X** | 쓰레기 마킹 (분해) | 인벤토리 |
| **Y** | 상세 보기 / 장비 해제 | 인벤토리 |
| **LB / RB** | 영역 전환 (장비↔가방↔창고) | 인벤토리 |
| **LB / RB** | 탭 전환 | 대장간 등 |

## 7-10. 설정 패널
| 버튼 | 동작 |
|------|------|
| D-pad ↑↓ | 항목 이동 (키바인딩, 슬라이더, select 등) |
| D-pad ←→ | range 슬라이더 값 조절 / select 옵션 순환 |
| 좌스틱 | 가상 커서 이동 |
| A (가상커서 모드) | 커서 위치의 슬라이더/select → D-pad 선택 모드로 전환하여 포커스 |
| A (선택 모드) | 키바인딩 입력 대기 진입 |
| LB / RB | 섹션 점프 |

> 가상 커서로 슬라이더 텍스트/바/값 아무 데나 A → 해당 슬라이더 포커스 → D-pad 좌우로 조절

## 8. 자동 입력 모드 전환

| 이벤트 | 동작 |
|--------|------|
| 게임패드 연결 | 즉시 패드 모드, 커서 숨김 |
| 패드 버튼/스틱 입력 | 패드 모드 활성화 |
| 마우스 이동 3px+ | KBM 복귀, 커서 복원 |
| 키보드 입력 | KBM 복귀, 커서 복원 |
| 게임패드 해제 | KBM 복귀 |

> 선택 화면 없이 자동 감지 + 실시간 전환. localStorage에 마지막 상태 저장.
