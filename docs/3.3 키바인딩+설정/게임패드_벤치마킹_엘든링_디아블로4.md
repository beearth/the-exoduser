## 1. 공통 원칙 (두 게임 모두)

| 원칙 | 설명 |
|------|------|
| **자동 감지** | 패드 입력 → 즉시 패드 모드, 마우스/키보드 → 즉시 KBM 모드. 선택 화면 없음 |
| **버튼 프롬프트 동적 전환** | 화면 내 "E 상호작용" → 패드 모드에서 "Ⓐ 상호작용"으로 자동 변경 |
| **B = 항상 뒤로/닫기** | 어떤 UI 깊이에서든 B는 한 단계 뒤로. 최상위면 UI 닫기 |
| **커서 숨김** | 패드 모드에서 마우스 커서 완전 숨김 |
| **모디파이어 조합** | 버튼 부족을 조합키(홀드+버튼)로 해결 |
| **래디얼/휠 메뉴 없음(엘든링) / 있음(디아4)** | 장르에 따라 다름 |

## 2. 엘든링 (Elden Ring)

### 2-1. 전투 매핑

| 버튼 | 기능 | 비고 |
|------|------|------|
| A | 구르기/백스텝/달리기 | 홀드=달리기, 탭=구르기 |
| B | 뒤로/닫기 (메뉴) | 전투 중 미사용 |
| X | 아이템 사용 | 하단 퀵슬롯 |
| Y | 두손잡기 전환 | Y+LB=왼손 두손, Y+RB=오른손 두손 |
| LB | 왼손 약공격/방패 | |
| RB | 오른손 약공격 | |
| LT | 왼손 강공격/전회 | |
| RT | 오른손 강공격 | |
| L3 | 은신 | |
| R3 | 타겟 고정/해제 | |
| D-pad ↑↓←→ | 퀵슬롯 전환 | 상:마법, 하:아이템, 좌우:무기 |
| Start | 메뉴 | |
| Select | 맵 | |

### 2-2. 핵심 전투 UX

- **타겟 고정 (R3)**: 가장 가까운 적에 카메라 + 에임 고정. 우리 게임의 오토에임과 유사
- **홀드 분기**: A를 탭하면 구르기, 홀드하면 달리기. 하나의 버튼에 2기능
- **모디파이어 조합**: Y 홀드 + LB/RB = 두손잡기 전환. 단독 Y는 사용 안 함 (메뉴에서만)
- **트리거 아날로그**: 없음. 디지털처럼 사용

### 2-3. 메뉴/인벤토리

| 조작 | 동작 |
|------|------|
| LB/RB | 메인 탭 전환 (장비/소지품/마법 등) |
| D-pad/좌스틱 | 그리드 2D 이동 (셀 단위) |
| A | 선택/장착/사용 |
| Y | 상세 정보 |
| X | 정렬/보조 기능 |
| B | 뒤로/닫기 |
| 우스틱 | 카테고리 간 빠른 이동 |

**엘든링 인벤토리 특징:**
1. 그리드가 아닌 **세로 리스트** (가로 카테고리 탭 + 세로 아이템 리스트)
2. 아이템 위에 커서가 올라가면 **즉시 우측에 상세 표시** (호버 = 선택)
3. 장비 교체: A → 장착 가능한 아이템 리스트 필터 → A로 교체
4. LB/RB로 탭 전환 시 **이전 탭 위치 기억**
5. 서브메뉴(컨텍스트): A 누르면 "사용/장착/버리기" 세로 목록 → 다시 A/B

### 2-4. 엘든링에서 배울 점

1. ✅ **셀 이동 = 즉시 상세 표시** (호버 불필요)
2. ✅ **B = 항상 뒤로** (컨텍스트 메뉴 → 패널 → 게임, 계층적)
3. ✅ **LB/RB = 탭 전환** (인벤 내 카테고리)
4. ✅ **A 하나로 "선택 → 컨텍스트 메뉴 → 확인"** 3단계 통일
5. ✅ **Y 홀드 + 다른 버튼 = 모디파이어** (우리의 LB 모디파이어와 동일 패턴)

## 3. 디아블로 4 (Diablo IV)

### 3-1. 전투 매핑

| 버튼 | 기능 | 비고 |
|------|------|------|
| A | 구르기/회피 | 쿨타임 있음 |
| B | 스킬 2 | |
| X | 스킬 1 | |
| Y | 스킬 3 | |
| LB | 스킬 4 | |
| RB | 기본 공격 | |
| LT | 스킬 5 | |
| RT | 스킬 6 (궁극기) | |
| L3 | 이동 (스틱) | |
| R3 | 힘의 행동 | |
| D-pad ↑ | 물약 | |
| D-pad ←→ | 퀵슬롯 순환 | |
| D-pad ↓ | 감정표현 휠 | |
| Start | 메뉴 | |
| Select | 맵 | |

### 3-2. 핵심 전투 UX

- **6스킬 직접 매핑**: B/X/Y/LB/LT/RT에 각각 스킬 1~6 배정. 모디파이어 없이 즉시 발동
- **RB = 기본 공격**: 우리의 A(mouse0)와 동일
- **A = 회피 전용**: 구르기/회피에만 사용. 우리의 LT(Shift/돌진)과 유사
- **좌스틱 = 이동 + 에임**: 이동 방향이 곧 공격 방향 (오토에임)
- **우스틱 = 안 씀** (전투 중). 메뉴에서만 사용

### 3-3. 메뉴/인벤토리

| 조작 | 동작 |
|------|------|
| LB/RB | 메인 탭 전환 (캐릭터/인벤토리/스킬/맵/일지) |
| D-pad/좌스틱 | 그리드 2D 이동 |
| A | 장착/사용 |
| X | 분해 (마킹) |
| Y | 아이템 드롭/버리기 |
| B | 뒤로/닫기 |
| R3 | 비교 토글 |
| RT | 전설 추출 |

**디아블로4 인벤토리 특징:**
1. **2D 그리드** (가방 전체가 격자, 아이템 크기 1×1/1×2/2×2)
2. 셀 이동 시 **즉시 좌측에 아이템 스탯 표시 + 장착 아이템과 비교**
3. X를 누르면 **분해 마킹** (노란 망치 표시) → 대장장이에서 한꺼번에 분해
4. **필터 없음** — 그리드를 직접 탐색
5. 장착: 아이템 위에서 A → 즉시 장착 (확인 없음)
6. **우스틱으로 빠른 스크롤** (대량 아이템)

### 3-4. 디아블로4에서 배울 점

1. ✅ **6스킬 직접 매핑** (모디파이어 없이 B/X/Y/LB/LT/RT)
2. ✅ **X = 분해 마킹** (쓰레기 토글과 동일)
3. ✅ **A = 즉시 장착** (컨텍스트 메뉴 없이)
4. ✅ **그리드 2D 탐색** — 우리 인벤토리와 동일 구조
5. ✅ **셀 이동 = 즉시 비교 표시** (장착 중 vs 선택 아이템)
6. ✅ **RT = 특수 동작** (추출/강화 등 컨텍스트에 따라 변경)

## 5. 주의사항

- **엘든링은 세로 리스트**, **디아4는 2D 그리드** — 우리는 디아4에 가까움 (12열 그리드)
- 엘든링의 장비 교체 플로우(A→필터 리스트→A)는 복잡함. 디아4처럼 **A 즉시 장착**이 나음
- 두 게임 모두 **패드 모드에서 마우스 커서 완전 숨김** — 우리도 동일
- 두 게임 모두 **UI 열려있을 때 게임 입력 완전 차단** — 우리도 동일 (3.5단계)
