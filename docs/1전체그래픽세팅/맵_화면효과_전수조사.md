## 1. 렌더 파이프라인 순서

아래는 매 프레임 최종 합성 순서 (위→아래 = 먼저→나중):

```
 1. 배경 레이어 (패럴랙스)
 2. 맵 타일 (타일맵 스트리밍)
 3. 맵 장식 오브젝트 (MAP_OBJS, light:1 포함)
 ~~3.5. Ori 스타일 포인트라이트 (플레이어 위치 640×640 앰버 글로우, lighter) — 제거됨~~
 4. 보스 게이트 비디오
 5. 적/NPC 렌더
 6. 플레이어 렌더
 7. 탄막(프로젝타일)
 8. 파티클/임팩트 VFX
 9. UI (HUD 레이어)
───── 포스트프로세싱 ─────
10. 라이팅 캔버스 합성 (_renderLighting)
11. 토치 캔버스 합성 (_torchCache)
12. 블룸 합성 (lighter, 2프레임 주기)
13. 탄막 글로우 패스 (lighter)
14. 유령불꽃 스프라이트
15. 화톳불 배리어 비주얼
16. 안개 입자
17. 환경광 오버레이 (2프레임 주기)
18. 동적 비네팅 (2프레임 주기)
19. 저체력 크랙/틴트/비네팅
20. 보스 시네마틱 (검은 바)
21. 미니맵
22. CSS 필터 (brightness, contrast, grain)
```

## 2. 토치(횃불) 어둠 시스템

맵 전체를 어둠으로 덮고, 광원 위치만 `destination-out`으로 뚫는 방식.

### 상수 (`_TORCH`, ~라인 2511)

| 상수 | 값 | 설명 |
|------|------|------|
| `baseRadius` | 550 | 기본 조명 반경(px) |
| `flickerSpeed` | 0.003 | 깜빡임 속도 |
| `flickerAmp` | 10 | 깜빡임 진폭 |
| `combatBoost` | 100 | 공격/차징 중 반경 추가 |
| `lowHpShrink` | 0.6 | 저체력 시 반경 축소 비율 |
| `bossBoost` | 150 | 보스 생존 시 반경 추가 |

### 토치 반경 계산 (~라인 120442-120448)

```
기본 = baseRadius (550)
+ 공격/차징 중: +100
+ 화염 무기 장착: +40
+ 보스 생존: +150
× 저체력 (HP ≤ 30%): × (0.6 + 0.6 × (HP% / 30%))
+ 깜빡임: sin(now*0.003)*10 + sin(now*0.0071)*6
```

### 어둠 색상 (지옥별, ~라인 133962)

| 지옥 | 색상 | 분위기 |
|------|------|--------|
| 0 림보 | `#061018` | 차가운 남색 |
| 1 색욕 | `#061008` | 어두운 녹색 |
| 2 탐식 | `#120608` | 짙은 적갈색 |
| 3 탐욕 | `#0a0614` | 깊은 보라 |
| 4 분노 | `#0e0c06` | 어두운 황색 |
| 5 이단 | `#0c0a08` | 회갈색 |
| 6 폭력 | `#04020c` | 짙은 자주 |

### 토치 캐시에 그려지는 광원 (~라인 133957-133997)

| 광원 | 위치 | globalAlpha | 반경 | 조건 |
|------|------|------------|------|------|
| 메인 플레이어 | (P.x, P.y) | 1.0 | 동적 계산 | 항상 |
| 보스 | (boss.x, y) | 0.7 | 80px | 보스 생존 |
| 출구 | (exit.x, y) | 0.8 | 50px | 보스 사망 후 |
| 단조 대장간 | (forge.x, y) | 0.6 | 70px | 대장간 존재 |
| 시작방 횃불 | (room[0]) | 0.5 | 120px | 시작방 존재 |
| **유령불꽃** | (iris.x, y) | flk×0.45 + flk×0.7 | OPT.irisGlow (기본700) + 32% | 유령불꽃 ON |
| 보스문 횃불 | (gate.x, y) | 0.6 | 100px | 게이트 존재 |

### 합성 방식
- `destination-out`: 어둠(fillRect)에서 빛을 뚫는 방식
- `_mkTorchStamp(r)`: 방사형 그라디언트 (중심=검정 → 끝=투명)
- `_getTorchSmall(r)`: 작은 광원용 스탬프 (반경별 캐시)

### 캐시 갱신 (~라인 133957)
- `_torchStale` (캔버스 크기 변경)
- 유령불꽃 ON일 때 매프레임
- 그 외 6프레임마다 1회 (`_gameFrame%6===2`)

## 3. 포인트라이트 시스템

별도 캔버스(`_lightCvs`)에 라이팅 합성. 토치와는 독립된 레이어.

### 핵심 함수 (~라인 1776-1839)
- `_pushLight(wx, wy, r, intensity, cr, cg, cb)` — 광원 등록
- `_collectLights()` — 매 프레임 광원 수집
- `_renderLighting()` — 라이팅 캔버스 렌더

### 광원 제한: 최대 120개

### 현재 등록되는 광원 (유령불꽃만)

| 광원 | 반경 | 강도 | RGB |
|------|------|------|-----|
| iris 메인 | OPT.irisGlow (700) | 0.45~0.61 (진동) | (255,180,60) 앰버 |
| iris 서브 | irisGlow×0.35 | 메인×0.4 | (255,210,120) 밝은 앰버 |

### 지옥별 환경광 (~라인 1697-1706)

```
0 림보: rgba(8,25,8, .06)    — 은은한 녹색
1 색욕: rgba(8,20,3, .06)    — 은은한 녹색
2 탐식: rgba(30,3,0, .07)    — 은은한 적색
3 탐욕: rgba(18,0,22, .07)   — 은은한 보라
4 분노: rgba(24,18,0, .06)   — 은은한 황색
5 이단: rgba(18,12,3, .07)   — 은은한 갈색
6 폭력: rgba(12,0,20, .08)   — 약간 짙은 보라
```

### 라이트 스탬프 (~라인 1713)
- `_LIGHT_RADII`: [10, 15, 20, 25, 30, 40, 50, 63, 88, 125, 175, 250, 400]
- `_lightStamps`: 흰색 방사형 그라디언트 (사전 렌더)
- `_amberStamps`: 앰버색 (0지옥 전용)

## 4. 유령불꽃(Iris) 조명

유령불꽃은 **토치 레이어 + 라이팅 레이어 양쪽**에 동시 기여하는 유일한 광원.

### 펫 정의 (~라인 2644)

| 속성 | 값 |
|------|------|
| 색상 | `#44aaff` |
| 궤도 속도 | 14 |
| 궤도 반경 | 55px |
| 기본 크기 | 20 (옵션 1~100) |

### 옵션 UI

| 슬라이더 | ID | 변수 | 범위 | 기본 | 저장 |
|----------|------|------|------|------|------|
| 유령불꽃 크기 | `optIrisSz` | `G._irisSz` → `pet.sz` | 1~100 | 20 | 게임세이브 + settings |
| 유령 글로우 | `optIrisGlow` | `OPT.irisGlow` | 100~1500 | 700 | settings |

### 토치 레이어 기여 (~라인 133979-133991)
- 메인 스탬프: 반경 `OPT.irisGlow`, alpha `flk×0.45`
- 서브 스탬프: 반경 `irisGlow×0.32`, alpha `flk×0.7`
- 깜빡임: `0.65 + sin(frame×0.1)×0.1 + sin(frame×0.17)×0.06`

### 라이팅 레이어 기여 (~라인 1793-1799)
- 광원 1: 반경 irisGlow, RGB(255,180,60)
- 광원 2: 반경 irisGlow×0.35, RGB(255,210,120)

### 스프라이트 렌더 (~라인 134013-134042)
- 헤일로 글로우: `#2288ff` (alpha 0.18) + `#44bbff` (alpha 0.12)
- 헤일로 크기: `20 + pet.sz × 0.8` (옵션 연동)
- 불꽃 이미지: flame01~04 (4프레임 애니, 속도 0.08)
- 불꽃 크기: `20 + pet.sz × 0.8` (옵션 연동)
- 폴백(이미지 미로드): `#44ccff` 원 14px + `#eeffff` 코어 7px

### 켜기/끄기 (~라인 118037-118040)
- 키바인딩: `irisToggle`
- 변수: `G._irisOff`

## 5. 블룸(Bloom) 포스트프로세싱

### 조건
- `OPT.bloom` && 품질 ≥ mid

### 동작 (~라인 1659-1690)
- 해상도: 1/6 (메인 캔버스 → 다운샘플)
- **짝수 프레임**: 다운샘플 → multiply(rgb 80,80,80) → 4방향 블러(±3px, alpha 0.25) → lighter 합성
- **홀수 프레임**: 이전 블룸 캐시 재사용 (합성만)

### 합성
- `globalCompositeOperation`: **lighter**
- `globalAlpha`: 일반 0.25 / 보스 생존 0.35

## 9. 동적 비네팅

### 기본 비네팅 (~라인 1604-1610)
- 방사형 그라디언트: 중심 h×0.3 → 외곽 h×0.85
- 색상: `rgba(0,0,0,0)` → `rgba(0,0,0,0.55)`

### 동적 비네팅 (~라인 134074-134084)
- 그라디언트: 중심 w×0.35 → 외곽 w×0.7
- 지옥별 색상 (토치 어둠과 유사한 색조)

| 지옥 | 비네팅 색상 |
|------|------------|
| 0 림보 | `rgba(20,60,120)` |
| 1 색욕 | `rgba(20,60,15)` |
| 2 탐식 | `rgba(100,20,10)` |
| 3 탐욕 | `rgba(50,15,80)` |
| 4 분노 | `rgba(80,60,10)` |
| 5 이단 | `rgba(50,40,25)` |
| 6 폭력 | `rgba(30,10,60)` |

- 스톱: 0(투명) → 0.7(alpha 0.05) → 1.0(alpha 0.15)

### alpha 값

| 상황 | alpha |
|------|-------|
| 일반 맵 | 0.2 |
| 보스 아레나 | 0.35 |
| 보스 HP < 50% | 0.5 + 0.15 × (1 - HP/maxHP) |
| 보스 HP ≈ 0% | ~0.65 |

### 캐시: `G._dvgCache` — 스테이지/캔버스 크기 변경 시 재생성

## 11. 보스 아레나 바닥 VFX

지옥별로 다른 보스방 전용 바닥 연출 (~라인 131493-131641):

| 지옥 | 효과 | 상세 |
|------|------|------|
| 0 얼음 왕좌 | 서리 결정 + 아이스 링 | X자 12개 + 원형 링 파동 |
| 1 독충 둥지 | 슬라임 웅덩이 + 점액 줄기 | 원 10개 + 방사선 4개 |
| 2 화염 제단 | 용암 균열 + 불씨 + 엠버 | 불꽃 8개 + 회전 입자 8개 |
| 3 심연 제단 | 어둠 웅덩이 + 보라 림 + 암흑 호 | 원 6개 + 호 2개 |
| 4 암전 타워 | 주기적 번개 + 전하 링 | 동적 번개 + 아크 링 |
| 5 백골 왕좌 | 외곽 해골 장식 + 연한 링 | 십자 16개 + 링 |
| 6 혼돈 균열 | 회전 보라 호 4겹 + 픽셀 산란 + 특이점 | 복합 |

- 업데이트: `G._bfxT`에 누적
- 보스 시네마틱 진입 시 활성화

## 12. 저체력 위기 시각화

HP ≤ 30%일 때 발동 (~라인 133782-133867):

### 심각도 계산
```
severity = 1 - (현재HP / (maxHP × 0.3))
```

### 효과 목록

| 효과 | 상세 |
|------|------|
| 심장박동 펄스 | 속도 300ms→120ms (심각도↑), 2단계 sin 피크 |
| 빨강 틴트 | `#880000`, alpha = `(severity×0.3 + 0.08) × pulse` |
| 빨강 가장자리 비네팅 | 방사형 그라디언트, alpha = `(0.3 + severity×0.5) × (0.6 + heartbeat×0.4)` |
| 화면 테두리 글로우 | `#aa0000`, 선폭 `4 + severity×6`, alpha = `(0.2 + severity×0.4) × pulse` |
| 균열 오버레이 | HP 변화 시 재생성, 개수 `2 + severity×8`, 화면 가장자리→중앙, 40% 분기 확률 |
| 심장박동 사운드 | 간격 800ms→300ms, 50Hz + 55Hz 2단계 |

### 토치 반경 축소 (~라인 120444)
```
반경 × (0.6 + 0.6 × (HP% / 30%))
```

## 14. 피격 플래시/화면 흔들림

### 피격 플래시 (Hit Flash)
- 함수: `doHitFlash(color, intensity)` (~라인 6350-6351)
- 최대 alpha: 0.5
- 감쇠: 프레임당 0.08

| 상황 | 색상 | 강도 |
|------|------|------|
| 일반 피격 | `#ffffff` | 0.18 |
| 보스 피격 | `#ffffff` | 0.2 |
| 패링 성공 | `#00ccff` | 0.25~0.3 |

### 화면 흔들림 (Screen Shake)
- 변수: `G.shake`
- 공식: `(Math.random() - 0.5) × G.shake × 2`
- 감쇠: `G.shake *= 0.86`
- 최소: < 0.3 → 0 강제
- 옵션: `OPT.shake` (0~100%, 기본 80)

| 상황 | 강도 |
|------|------|
| 기본 공격 | 4~8 |
| 패링/블록 | 6~18 |
| 보스 슬램 | 14 + hitStop |
| 차징 완료 | 6~12 |
| 스킬 기폭 | 4~45 |
| 체간 파괴 | 25 + slowMo=40 |

## 17. 효과 간 상호작용/충돌

| 효과 A | 효과 B | 관계 |
|--------|--------|------|
| 토치 (dest-out) | 라이팅 (dest-out) | **독립 레이어** — 별도 캔버스에서 각자 처리, 충돌 없음 |
| 유령불꽃 | 토치+라이팅 | **양쪽 동시 기여** — 토치 어둠 뚫기 + 라이팅 앰버광 |
| 블룸 (lighter) | 탄막 글로우 (lighter) | **누적 가산** — 순차 합성으로 밝기 누적 |
| 환경광 | 비네팅 | **프레임 교대** — 짝수프레임=비네팅, 홀수프레임=환경광 |
| 안개 | 유령불꽃 | 위치 추적 변수 있으나 **현재 미사용** (흡수/밀어내기 없음) |
| 저체력 비네팅 | 보스 비네팅 | **동적 영향** — 보스 HP에 따라 비네팅 강화 |
| 토치 반경 | 저체력 | **동적 축소** — HP 30% 이하에서 조명 줄어듦 |
| bloom | 보스 생존 | **강도 변화** — 일반 0.25 → 보스전 0.35 |
| CSS grain | bloom | **독립** — CSS 필터 vs Canvas 합성 |
